Notes:
Scenes are stored in showfiles, so all of the parameters listed below will also be stored within the showfile, including any changes not stored to a scene (e.g. the last EQ setting of a channel will still be stored in the showfile even if it was not stored in the scene before the showfile was saved).

The scope buttons in the Copy menu (EQ, Gate, Aux etc.) follow the same scope as listed below.

Console Storage

The following settings are stored in Console Storage. This means they are not stored in scenes or showfiles, and are specific to that individual console.

  • Sample Rate
  • Clock Source
  • Dante Expander Device Name (console end, see Setting Up Dante for more details)
  • BLII/X-Light Bridge Device Name (console end)
  • All settings for Connectivity and Dante Control networks (DHCP/Fixed, IP address etc.)
  • Remote surface Manual Discovery settings

Not Stored

The following settings are not stored in scene, showfile or console storage. These settings will be reset to their default OFF setting when the console is rebooted or when a new showfile is loaded.

  • Path Solo On/Off
  • Solo In Place On/Off
  • Mute Safe On/Off
  • Path Global Recall Safe On/Off (option to store in showfiles)
  • Main/Remote Surface mode (defaults to Main)
