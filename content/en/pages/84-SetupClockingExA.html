System Example A: Connecting stageboxes to a Live console via MADI

  1. Set the Live console's clock source to Internal (Int), as described in the main Clocking section.
  2. Connect each stagebox MADI port 1 Out & In to the console using a pair of 75 Ω BNC coax cables (max 75 m, conforming to spec defined in the Installation Guide). The stagebox will be automatically detected by the console and this console will be the Master console with full control.
  3. If a redundant MADI connection is required, ensure each stagebox MADI port 1 is connected to an odd numbered port on the console and connect each stagebox MADI port 2 Out & In to the next console MADI port (e.g. stagebox ports 1 & 2 to console ports A1 & A2 respectively). In the console's I/O page (Menu > Setup > I/O) turn on Redundancy Status for the relevant pair of MADI ports.
  4. Ensure each stagebox is set to clock from MADI, as described in the main Clocking section.
