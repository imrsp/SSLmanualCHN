V6.2

V5.2
  • Added integration of d&b Soundscape Control

  • Multiple Fade Time Edit
  • Onscreen Meter scale -60dB option

V5.1
  • Added MIDI Control Change (CC) message support to Automation Input & Output Actions and Event Manager

  • Added routing display indicators next to Detail View Route Input and Output buttons
  • Added Zoom and View Select functionality to Overview
  • L100 and L200 can now connect to up to two remote surfaces
  • Added ability to load showfiles with dry paths onto full path consoles
  • Added ability to enable/disable echoing of OSC messages to Generic OSC devices
  • Reordered L-ISA Quick Control assignments to more closely match L-ISA Controller layout
  • Improved gearing of L-ISA Pan and Pan Spread controls for greater resolution
  • Improved labelling of drive buttons in all file browser interfaces
