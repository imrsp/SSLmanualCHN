Main Screen Menu

The Menu buttons in the rightmost column of the Main Screen allow for configuration. There are two menu trees, operational and setup:

Expands the menu tree:
Channel View displays the Fader Tile that is selected with the Fader Tile's Screen button. This is where you'll spend most of your time. The Screen button will also call this view from any other menu.
Automation is where scenes within the showfile are managed.
Overview shows all paths in the showfile at once. It's great for getting an idea how the showfile is configured at a glance.
Effects displays the rack of on-board effects.
Matrix accesses the output matrices.
Screen Query provides ability to Query from Main Screen.
Setup opens the console setup configuration menus:
Options accesses a range of console and user options, arranged into tabs of different sections.
Console Configuration is where the DSP paths are allocated. This is where you will begin.
Layer Manager is where your physical Fader Tile layout is defined. You would use this after Console Configuration.
I/O is where the physical audio connections are defined and managed.
XY Routing allows for bulk routing in an XY interface.
File Manager allows access to Showfile management, processing Preset file management and import/export of input patch files.
System / Power* (press & hold) allows you to shutdown the console properly and provides diagnostic tools and system level settings. (Labelled System on SOLSA.)
Tools opens the console tools menus:
Talkback Groups enables Talkback Groups to be configured and engaged.
Signal Switcher can be used for bulk switching of signals; refer to the Signal Switcher section.
Event Manager is used to set up Events, consisting of one of more state-driven sources, which when met will cause one or more actions to be triggered.
Help accesses the SSL Live Help System.

Note: Once in the Setup or Tools menu you can quickly toggle to the other by pressing the Setup / Tools button.
