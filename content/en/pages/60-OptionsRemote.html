Remote Options

These settings relate to remote control connections for the console.

Remote Options are found in MENU > Setup > Options > REMOTE tab.

The console can be remotely controlled by the following devices:

  • TaCo Tablet Control app for iOS and Android devices
  • SOLSA (SSL On/Offline Setup Application) PC app
  • Another SSL Live console in Remote Mode
  • A Remote Expander

When viewing the REMOTE tab on the console, two lists of connections are displayed.

SSL LIVE CONNECTIONS is a list of SOLSA, other Live consoles in Remote Mode and Remote Expander connections. For more information on connecting and remote controlling the console via SOLSA, other Live consoles or the Remote Expander, please see Remote Surfaces.

TABLET CONTROLLERS is a list of the tablets running TaCo connected to the console. For more information on connecting and remote controlling the console via TaCo, please see TaCo Tablet Control.

When SOLSA is in Main Surface Mode the SSL LIVE CONNECTIONS list will be empty as SOLSA cannot act as the Main surface to a remote connection. When SOLSA is in Remote Surface mode, this section lists the name and status of the console that SOLSA is currently controlling.

To switch between Main Surface and Remote Surface Mode, press and hold the appropriate button.
