External Control Options

These options are for configuring control of external devices from the console.

External Control Options are found in the MENU > Setup > Options > EXTERNAL CONTROL tab.

Tap a device in the OSC SETTINGS section to view and edit details in the lower DEVICE CONFIGURATION area.

Populate the device configuration fields for the selected device and enable control from the console using the Enable Control button.

Refer to the External Control section for further information.

The console's Connectivity/Control network IP address is also displayed in this page; this can be changed in the NETWORK Options tab.
