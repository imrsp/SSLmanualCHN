数字键盘

左上角的数字键盘可用于输入数值；回车键的功能与屏幕数字键盘中的 OK 按钮相同。 这些按键也可分配为 User Keys；配置详情请参阅 User Key Options。

主推子

Master Tile 右侧的推子可分配给控制台上的任意输入或输出通道。无论该通道当前是否在 Fader Tile 上启用，都可以持续控制其电平。通常可将其用于某条 Master Bus、主唱电平或 Solo 返送音箱电平；也可以分配给一个 VCA，由该 VCA 控制一组母线。

要将通道分配给 Main Fader：

将包含待分配通道的 Bank 调到某个 Fader Tile 上；

按住 Main Fader 上方的 Call 按钮，直到该按钮闪烁青色。当前所分配通道的 Select 按钮也会闪烁；

按下要分配通道的 Select 按钮 – 该按钮会闪烁青色，而原分配通道的按钮会停止闪烁；

再次按 Call，退出分配功能。

Focus Fader

Focus Fader 及其对应的 Mute、Q 和 Solo 按钮，以及 Control Tile 左下角的 Quick Controls（如配备），始终作用于“Focused”（聚焦）信号路径。

Focus Fader 通常跟随当前所选信号路径，但也可“分配”给某条特定路径，例如需要快速访问的重要输入通道、VCA、Solo 通道或附加 Master。

Focus Fader 的分配方法与 Main Fader 相同（按住 Focus Fader 的 Call 按钮，然后选择信号路径，再按 Call 确认选择）。

要快速访问所分配的信号路径，按一次 Focus Fader 的 CALL 按钮。

还可将 Focus Fader 锁定到所分配的信号路径。双击 CALL 按钮，按钮变为绿色即表示已锁定。再次双击 CALL 可恢复正常操作。

Mute 与 User Keys

Mute/User 按键可在 Mute Groups（静音组）或 User 按键之间切换，方法是使用 MODE 按钮。有关 Mute Groups 的更多信息，请参阅 静音与静音组。

User Keys 可由用户自定义，并可在 用户按键选项 中配置。
