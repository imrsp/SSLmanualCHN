操作面选项

这些设置涉及 Query 配置、Follow 模式等操作面专用选项。

操作面选项位于 MENU > Setup > Options > SURFACE 选项卡。


区块设置

推子区块

启用 Screen Follow Select 后，最后一次按下 Select 按钮所在的推子区块会自动调出到主屏幕。

Focus 推子锁定

主控区块中的 Focus 推子可锁定到指定通道信号路径，具体方法见 Control Tile 章节。如果 User Options 页面的 Link Channel Control Tile 按钮已启用，则控制区块也可锁定到该通道；该按钮位于 Focus Fader Lock 区域。

数字键盘

该 Lock To User Keys 按钮可防止启用屏幕键盘时主控区块上的数字键区变为数值键盘。这样，分配给该键区的用户自定义功能始终可用。

路径标签 LCD

路径标签 LCD 的底行文字可显示分配给该通道条的路径信息（路径类型、编号和格式），也可显示 Solo 母线路由信息。使用 Show Info 和 Show Solo 按钮可在两种显示之间切换。

Query 设置

本区域包含 Query 的相关设置。Query 的使用和设置方法请参阅 Query 页面。


Select/Solo/Query 跟随选项

该 Follow Setup 区域可对推子区块上的多种按钮组合建立链接（Follow）。例如，按下某信号路径的 Select 按钮时，可自动将该路径置于 Solo 状态，反之亦然。

如需配置 Follow 模式，请先启用 Edit 按钮。

三行分别对应推子区块上的硬件 Select、 Solo 和 Q （Query）按钮。各列表示按下硬件按钮时应由哪个按钮跟随。各列按信号路径类型（Channels、Stems、Auxes 等）排列，因此可为不同路径类型设置不同的跟随模式。

在网格中启用相应按钮，以指定按下某硬件按钮（行）时需要跟随的功能（列）。例如，如需为 Auxes 启用 Solo 跟随选中，请找到 Solo 列，该列位于 Auxes 区域，然后按下该列与 Select 行交叉处的按钮。

使用网格 Edit 行和列中的按钮选择整行或整列，然后使用 Set All、 Clear All 或 Toggle 按钮更改所选项目的状态。
