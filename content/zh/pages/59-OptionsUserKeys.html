用户按键选项

这些设置与控制台操作面上可由用户分配的各种按键有关。

用户按键选项位于 MENU > 设置 > Options > USER KEYS 选项卡。

有多组用户按键可分配给各种功能，以便快速访问。这些按键包括：

  • Keypad 按键：主控区块上的十个数字键盘按钮（如装有控制区块，也可从其屏幕访问）
  • Mute/User 按键：主控区块上的十个静音组主控键可使用 MODE 键切换为用户按键（如装有控制区块，也可从其屏幕访问）
  • Channel View Popout ：十六个虚拟用户按键，位于可从主屏幕通道视图侧栏打开的弹出窗口中
所有用户按键的分配方法相同：在 User Keys 选项卡中找到相应部分，例如 MUTE/USER 按钮部分：

使用 按钮，从每个用户按键字段右侧列出的可用功能中进行选择（完整列表见下文）。
