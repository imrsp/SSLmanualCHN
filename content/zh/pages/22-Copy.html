复制

可使用 Channel View 屏幕右侧的 Copy 按钮，在信号路径之间复制信号处理、母线及声像设置。

复制功能既可复制完整通道，也可复制单独的通道组件（包括信号处理和声像的各个组成部分）。

首先在屏幕上点按要复制来源的通道，或使用推子区块上的 SELECT 按钮。 然后点按 Copy 打开 Copy 窗口。

作为复制来源的信号路径以绿色标示。

从列表中选择要复制的参数。也可以使用 Set All 设置全部参数，并使用 Toggle 切换其开启或关闭状态。

现在选择要粘贴到的信号路径，可在屏幕上选择，也可使用推子区块上的 SELECT 按钮。 这些粘贴目标信号路径以紫色标示。

然后按住 Apply。

完成后，再次按下 Copy 关闭窗口并清除选择。

提示：使用范围选择可快速复制到多个信号路径。按住范围内第一个信号路径的 SELECT 按钮，再点按最后一个信号路径的 SELECT 按钮，然后松开两个按钮。也可以跨相邻的推子区块执行此操作。

以下组件可单独复制：

  • Full 和 Dry 处理的 Master 发送
  • Full 和 Dry 处理的 Aux 发送
  • Full 和 Dry 处理的 Stem 发送
  • 通道 Input 设置
  • Pan
  • Fader 电平
  • Mute 状态
  • EQ（仅限完整处理和矩阵输出信号路径）
  • 高通和低通 Filters（仅限完整处理和矩阵输出信号路径）
  • Compressor（仅限完整处理信号路径）
  • Gate（仅限完整处理信号路径）
  • Delay（仅限完整处理和矩阵输出信号路径）
  • All Pass 滤波器（仅限完整处理和矩阵输出信号路径）
  • VCA 分配

复制发送混音

可以使用 Channel View 中的 Bus Copy，将某个 Stem、Aux 或 Master 混音复制到另一个 Stem、Aux 或 Master 母线。

  1. 选择要作为复制来源的母线信号路径条。
  2. 点按 Copy 打开复制窗口。作为复制来源的信号路径将以绿色标示。
  3. 点按 Bus Copy。
  4. 选择要复制的信号路径类型。使用 Set All 和 Toggle 可更快地调整选择。
  5. 现在选择要粘贴到的信号路径，它们将以紫色标示。
  6. 然后按住 Apply。
  7. 完成后，再次按下 Copy 关闭窗口并清除选择。
注意：母线复制中只会提供能够向源信号路径发送信号的信号路径（例如 Aux 不能发送到 Stem）。

Bus Copy 可在不同母线类型之间复制，例如将 Aux 混音复制到 Stem。

警告：复制过程中被覆盖的设置无法直接恢复，只能从先前保存的场景或演出文件中调用。

Copy 包含哪些参数？

Copy 所包含的参数和设置与场景存储所包含的内容相同。要了解场景包含哪些参数，请参阅 存储。
