本地 I/O

所有 SSL Live 控制台型号都配有相同类型的本地 I/O，但各型号每种 I/O 的数量不同。本页底部列出了各控制台型号的完整 I/O 容量。


模拟

Live 本地 I/O 和舞台接口箱采用 SSL 专利话放技术，可实现 SSL SuperAnalogueTM 性能。 超越行业标准的录音室级话筒前置放大器结合 24bit/96 kHz DAC，可在 20 Hz 至 20 kHz 范围内实现 ±0.05 dB 以内的频率响应（低至 10 Hz 时为 1.3 dB 以内）以及 0.005% 的 THD。 电路采用直流耦合（信号路径中没有电解电容）和高输入阻抗设计。 话放增益可通过 16,000 多个步进进行极高精度控制，从而保证完全平滑的调节、出色的共模抑制和极低失真。


MADI

SSL Live 控制台配有 SC 光纤和 BNC 同轴 MADI 接口，可连接 SSL Live MADI 舞台接口箱或其他第三方 MADI 设备。


Dante

Dante Brooklyn II 模块通过冗余连接提供 96 kHz 下 32x32 通道的 I/O（或 48 kHz 下 64x64 通道）。 这些 Dante 连接还支持控制 SSL Network I/O 舞台接口箱的话放。

利用 SSL 的 Dante 双域路由，可以直接在控制台屏幕上同时完成控制台侧和 Dante 侧的路由，因此无需使用 Dante Controller 建立任何路由。


Blacklight II

对于通道数较多的系统，可使用 SSL 专有的 Blacklight II 高带宽传输连接， 以减少互连线缆的数量。 Blacklight II 可通过单根光纤双向传输 256 路 96 kHz 音频信号。

使用 BLII-MADI Concentrator 或 BLII-Dante Bridge，可分别将 Blacklight II 转换为 MADI 或 Dante。

Blacklight II 使用 Neutrik OpticalCON DUO 连接器。

Blacklight II 通常缩写为 BLII。


AES

SSL Live 控制台还配有支持完全可变采样率转换的 AES I/O。
