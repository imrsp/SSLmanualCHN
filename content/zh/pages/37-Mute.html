静音

每个通道都有一个 Mute 按钮，位于通道推子上方。还可通过以下方式将通道静音：

  • 自动化静音：通过通道已静音的自动化“场景”启用；
  • 组静音：通过通道所属的静音组启用；
  • VCA 静音：使用通道所属 VCA 控制通道上的 Mute 按钮启用。（下一篇设置帮助页面介绍 VCA。）

这些 Mute 按钮的颜色表示以下状态，详见本页后文：

熄灭：未静音（可手动静音，也可由自动化或组静音）；

橙色：组静音（由静音组或 VCA 静音）；

红色：手动或自动化静音（组无法解除该通道的静音）；

红色闪烁：安全静音（防止自动化或组解除静音）；

绿色：安全取消静音（防止自动化或组将其静音）。

注意： 静音不会关闭录音机输出或任何推子前 Insert 发送。

静音组

定义： 静音组允许按一个按钮便将多个通道静音，例如整个管乐声部，或某一幕中的全部表演者。

SSL Live 控制台有十个静音组，通过主控区块上的十个静音组按钮控制。

任何类型的通道均可加入静音组，而且每个通道可同时属于任意数量的静音组。

要将静音组中的所有信号静音，只需按相应的静音组按钮。该静音组按钮以及组内所有通道的 MUTE 按钮都会变为橙色。

注意： 如果在通道本地使用其自身的 Mute 按钮将其静音，则无论所属静音组处于何种状态，该通道都会保持静音，按钮变红即表示此状态。

也可从控制区块访问静音组主按钮：

按控制区块上的 Home 按钮，然后在 Home 屏幕中选择 Mute Groups 选项。

使用屏幕按钮或按下屏幕按钮旁的旋转编码器，即可开启或关闭静音组。

将通道分配给静音组

可使用主控区块的静音组按钮，或主屏幕、控制区块中的静音组功能，将通道分配给静音组：

注意： 如果在静音组处于启用状态时将通道加入该组，通道会立即静音。

主控区块静音组按钮：

按住所需修改组对应的主控区块静音组按钮；按钮会闪烁橙色，表示已进入配置模式。

要将通道加入组，请按其 SELECT 按钮；按钮闪烁橙色表示该通道已加入。

要从组中移除通道，请按其 SELECT 按钮；按钮会停止闪烁。

也可按住范围一端通道的 SELECT 按钮，然后按范围另一端通道的 SELECT 按钮，以加入一段连续通道。

再次按主控区块静音组按钮即可恢复正常操作。


详细视图

静音组快速控制的扩展版本位于 MG + VCA 详细信息页面中；在 MG 或 VCA 分配按钮处于启用状态时，双击快速显示即可打开：

如果页面右侧的 Lock 按钮处于启用状态，请触摸所需静音组的选择器，然后使用屏幕底部 MG On 标签下方的上快速键，将通道加入或移出该组。

If Lock 未启用时，只需按静音组选择器即可添加或移除通道。


快速控制：

在通道条中选择 MG 分配按钮：

通道条的选择器区域会显示十个静音组选择器以及一个 MUTE SAFE 按钮。

触摸要添加或移除当前通道的静音组选择器。

如果屏幕右边缘的快速控制 Lock 已启用，选择器会变为蓝色，表示该组已分配给快速控制。随后可使用上快速键将通道加入或移出该静音组；MG 选择器的数字区域变绿，表示该通道已加入该组。

如果快速控制 Lock 未启用，则通道在分配给快速控制的同时也会加入该静音组。

左快速键用于通道安全功能，详见下文。

通道 MG 分配按钮中的色条会变为橙色，表示该通道已分配给至少一个静音组。

MG 选择器显示状态如下：

未分配且未启用；

已分配给快速控制但未启用；

已分配给快速控制且已启用；

未分配但已启用。


控制区块

也可使用控制区块编辑通道的静音组分配：

按控制区块上的 VCA / MG 按钮，将该功能分配给控件：

屏幕顶部显示“Selected”通道；两侧的 Prev 和 Next 箭头按钮可用于滚动浏览通道。

静音组选择器显示在屏幕左侧。

要将当前通道分配给静音组，请确保上方的 Lock 按钮未启用，然后触摸所需静音组的选择器。

如果 Lock 已启用，将无法编辑屏幕内容。

静音安全

MUTE SAFE 可覆盖可能作用于通道的任何间接静音（静音组、VCA 或自动化静音）。

当 MUTE SAFE 在当前已静音的路径上启用时，该路径会自动进入 Safe Mute 状态。

当 MUTE SAFE 在当前未静音的路径上启用时，该路径会自动进入 Safe Unmute 状态。

红色闪烁： Safe Mute （防止自动化或组解除静音）；

绿色： Safe Unmute （防止自动化或组将其静音）。

触摸 MUTE 按钮，会使该路径在 Safe Mute 和 Safe Unmute之间切换。

The MUTE SAFE 功能可通过上述快速控制 MG 页面启用，也可在通道详细信息对话框或控制区块中以类似方式启用。触摸 MUTE SAFE 可启用或停用该功能。

另请参阅下文的 自动静音安全 和 临时自动静音安全 模式。

注意：
MUTE SAFE 的开关状态不会存储在场景或演出文件中。

自动静音安全

自动静音安全 可用于在任一路径上快速启用 静音安全 模式。例如演出期间乐器损坏并产生噪声时，需要将通道设为安全静音；或有人拿错麦克风时，需要设为安全取消静音。Auto Mute Safe 模式通过用户按键启用。启用后，按任意 MUTE 按键会在该路径上启用 静音安全 模式并切换其静音状态。该路径会保持安全静音/安全取消静音，直到问题解决并手动停用 Mute Safe。

要使用 自动静音安全 ，请先在 User Key 中将其添加为 MENU > Setup > Options > USER KEYS 选项卡中的用户按键。 自动静音安全 位于用户按键选项的 Console Functions 部分。

分配给用户按键后，按该用户按键即可启用 自动静音安全 模式。现在按任一路径上的 MUTE 按钮，该路径的 MUTE SAFE 即会开启。 自动静音安全 会保持启用，因此可通过其他路径的 静音安全 按键将这些路径置于 MUTE 模式。根据之前的 Mute 状态（分别为未静音或已静音），路径现在会处于 Safe Mute 或 Safe Unmute 状态。

随后触摸路径的 MUTE 按钮，会使该路径在 Safe Mute 和 Safe Unmute 状态之间切换，直到手动停用 静音安全 模式。

要关闭 自动静音安全 ，请再次按用户按键。已经进入 MUTE SAFE 的路径在 自动静音安全 停用后仍会保持该模式。请前往快速控制 MG 页面、通道视图或详细视图，关闭该路径的 MUTE SAFE 。其他路径的 MUTE 按钮现在会恢复正常操作。

临时自动静音安全

临时自动静音安全 扩展了上述 自动静音安全 模式。它主要用于线路检查或声音检查，只需临时覆盖静音状态时（例如从乐队静音组中单独取消某些通道的静音以确认信号）， 临时自动静音安全 也可分配给用户按键。

要将 临时自动静音安全 分配给 User Key ，请前往 MENU > Setup > Options > USER KEYS 选项卡中的用户按键。 临时自动静音安全 位于用户按键选项的 Console Functions 用户按键选项中。分配后，按该用户按键即可启用 临时自动静音安全 模式。

启用后，按任意 MUTE 按键会使该路径进入 静音安全 模式并切换 Mute 状态（与上述 Auto Mute Safe 完全相同）。但在临时模式下，第二次按 Mute 键会直接退出 Mute Safe 模式，使 Mute 恢复到其“正确”的底层状态，而不是在 Mute Safe 模式内切换状态。停用 临时自动静音安全 模式会取消由该模式触发的所有 Mute Safe。手动启用的 Mute Safe（无论是在 Temporary Auto Mute Safe 启用之前还是之后）在关闭临时模式时均保持不变。
