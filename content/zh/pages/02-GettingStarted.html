快速入门指南

本页旨在帮助您快速上手控制台。

如需进一步了解其中任何主题，请返回主页；主页对控制台的各个部分均有详细说明。

请注意：如果使用 Dante 舞台接口箱，本快速入门指南假定读者已完成 Dante 认证第 1、2 级，或具备同等 Dante 使用经验。 如果不具备这些条件，请参阅设置 Dante了解更多信息。

1. 开启控制台

将控制台背面的 PSU 开关拨到“on”位置（配备双冗余电源的 L100 控制台没有这些开关）。

按下控制台 Control Tile 或 Tablet Tile 上的 PSU 开关。

控制台随即启动。


2. 设置采样率

在控制台上进入 System 页面（MENU > Setup > System / Power > System 选项卡）。

按住 96 kHz 或 48 kHz 按钮选择采样率。按钮开始闪烁后即可松开。

如果更改了采样率，请重启控制台。


3. 选择主时钟

在控制台上进入 MENU > Setup > Options > SYSTEM 选项卡。

在 System Clock 区域，从可用选项中选择一个 Main 作为主时钟源；也可选择一个 Backup 备用时钟源。

按住 APPLY。

对于基于 MADI 的系统， SSL 建议选择控制台作为主时钟（在屏幕上选择 Int，即内部时钟）。舞台接口箱及其他所有 MADI 设备应该作为从时钟锁定至控制台。

对于基于 Dante 的系统，SSL 建议让控制台从属于 Dante 网络（在屏幕上选择 Dante Expander）， 以利用 Dante 的时钟选举机制。

更多信息请参阅时钟设置。


4. 加载模板演出文件

进入 MENU > Setup > File Manager > Showfiles 选项卡。

点击左侧的 System Templates。

选择最适合系统的模板：

  • ... over BLII-MADI Concentrator 模板适用于使用 MADI 舞台接口箱和 BLII-MADI Concentrator 的系统
  • ... over MADI 模板适用于 MADI 舞台接口箱直接连接控制台的系统（不使用 BLII-MADI Concentrator）
  • ... over Dante - Local Dante Port 模板适用于使用 Dante 舞台接口箱但不使用 BLII Bridge 的系统
  • ... over Dante - BLII Bridge 模板适用于使用 Dante 舞台接口箱和 BLII Bridge 的系统
  • ... over Dante - X-Light Bridge 模板适用于使用 Dante 舞台接口箱和 X-Light Bridge 的系统

点击一个模板，然后按住 Load。

SSL 建议：在本快速入门指南中，请不要加载 Blank 演出文件。
其他模板已预先完成大部分配置。使用 Blank 演出文件会使设置过程耗时更长。

选中模板后，Showfile Notes 区域会说明该演出文件的设置。你可以随时返回查看。双击 Showfile Notes 区域可编辑备注。

点击演出文件列表顶部的 HD 按钮。按 Save As，输入演出文件名称，然后按 OK。

SSL 建议：建议从控制台内置硬盘运行演出文件，而不是从 USB 存储设备运行。控制台每分钟保存一次演出文件备份。 如果发生断电，控制台再次开机时会加载最近一次备份的演出文件。备份演出文件无法保存到 USB 存储设备，因此从 USB 运行演出文件时不具备此项保护。

5. 连接舞台接口箱

MADI 舞台接口箱

MADI 舞台接口箱

以下说明适用于带模拟输入的 MADI 舞台接口箱：ML 32.32 和 ML i32。

舞台接口箱采样率与时钟设置

在舞台接口箱背面找到 SETUP 区域。检查所示采样率是否与控制台设置一致。MADI 连接不支持采样率转换。

如果采样率不一致，请同时按住 RATE 和 CLOCK 按钮，直到挂锁 LED 开始闪烁。

反复按 RATE 按钮，直到所示采样率与控制台采样率一致。请注意，96 kHz 通过 48 和 x2 LED 同时亮起来表示。

反复按 CLOCK 按钮，直到 MADI LED 亮绿灯。此时舞台接口箱会从控制台获取时钟。如果 LED 亮红灯，表示时钟方案无效。 此时请重新检查第 3 步，确认是否有遗漏。

确保 ID 区域的 1-32 LED 亮起（在 48 kHz 下以菊花链方式连接舞台接口箱时除外）。反复按 ID 按钮， 直到 1-32 LED 亮起。（如需在 48 kHz 下以菊花链方式连接舞台接口箱，请参阅时钟设置章节。）

舞台接口箱连接

Blacklight II（BLII）Concentrator 可通过一对光纤线缆，以冗余方式将最多 8 台 MADI 设备连接到控制台。

使用 BNC，将舞台接口箱背面的 MADI Port 1 连接到 Blacklight II Concentrator 背面的 Port 1A。

使用 BNC，将舞台接口箱的 MADI Port 2 连接到 BLII Concentrator 的 Port 1B。

使用 Neutrik OpticalCON DUO 连接，将 BLII Concentrator 前面的 BLII A Main 连接到控制台背面的 BLII Port 1a。

使用 Neutrik OpticalCON DUO 连接，将 BLII Concentrator 上的 BLII A Backup 连接到控制台上的 BLII Port 1b。

请注意：如果没有 BLII-MADI Concentrator，也可以将舞台接口箱直接连接到控制台。 只需使用 BNC，将舞台接口箱的 MADI Ports 1-2 直接连接到控制台上一对奇偶 MADI 端口。

在控制台上进入 MENU > Setup > I/O > Local/MADI Configuration。

在屏幕右侧的 BLII 标题下，第一个端口应显示为绿色，并带有舞台接口箱图像。 如果背景为红色或灰色，或者没有舞台接口箱图像，请参阅本地/MADI I/O 配置。


对其他 MADI 舞台接口箱重复上述步骤。

此时已通过 1:1 跳线建立从舞台接口箱到通道输入的路由。对于 Monitor 模板，也已路由 Aux Mix 输出。如需调整这些路由，请参阅路由。

Dante 舞台接口箱

Dante 舞台接口箱

以下说明仅适用于 Net I/O SB 32.24 舞台接口箱。如需连接其他 Net I/O 舞台接口箱，请参阅Network I/O 舞台接口箱。

请注意：SSL Live 控制台包含 4 个用于 Dante 的网络设备：
· Dante Control（Primary 和 Secondary）：这些设备处理 Dante Control数据（即 Net I/O 舞台接口箱增益和路由）
· Dante Audio（Primary 和 Secondary）：这些设备处理 Dante 音频。在 SSL Live 控制台上，该设备也称为 Dante Expander，并包含 Dante Brooklyn II 模块。
所有这些设备必须在同一子网内使用不同的 IP 地址。

设置控制台 Dante Control 的 IP 地址

在控制台上进入 MENU > Setup > Options > NETWORK 选项卡。

在 Dante Control Settings 下，按 Fixed 按钮，将其用于 Primary Control Network。

  • 双击 IP Address 字段，输入 10.101.x.x*，然后按 OK。
  • 双击 Netmask 字段，输入 255.255.0.0，然后按 OK。
  • 将 Gateway 保持为 0.0.0.0。

按住对应的 Apply 按钮。

按 Fixed 按钮，将其用于 Secondary Control Network。

  • 双击 IP Address 字段，输入 10.102.x.x*，然后按 OK。
  • 双击 Netmask 字段，输入 255.255.0.0，然后按 OK。
  • 将 Gateway 保持为 0.0.0.0。

按住对应的 Apply 按钮。

* 出现“x”的位置，请选择 1 至 255 之间的任意数字，但不能使用此网络中其他设备已占用的数字。 同一网络中的多台设备不能使用相同的 IP 地址。

设置控制台 Dante Audio 的 IP 地址

将计算机连接到控制台背面的 Primary Dante 端口。

确保计算机与舞台接口箱的 Primary Dante Audio 位于同一子网。如果不确定如何操作，请参阅设置 Dante。

打开 Dante Controller，在网络中找到控制台的 Dante Expander/Brooklyn Module。如果 Dante Controller 中看不到控制台，请参阅查找设置为不同子网固定 IP 地址的 Dante 设备。

在 Dante Controller 上双击控制台，进入 Network Config 选项卡。

在 Primary 下，单击 Manually configure an IP Address，然后输入：

  • IP Address：10.101.x.x*
  • Netmask：25.255.0.0
  • 将“DNS Server”和“Gateway”保持为 0.0.0.0

在 Secondary 下，单击Manually configure an IP Address，然后输入：

  • IP Address：10.102.x.x*
  • Netmask：25.255.0.0
  • 将“DNS Server”和“Gateway”保持为 0.0.0.0

* 出现“x”的位置，请选择 1 至 255 之间的任意数字，但不能使用此网络中其他设备已占用的数字。 同一网络中的多台设备不能使用相同的 IP 地址。

按 Apply。系统会要求确认 IP 地址更改，请单击 Yes。 更改生效前必须重启 Brooklyn II 模块。单击 Reboot。系统会要求确认重启，请单击 Yes。无需重启控制台。

设置控制台名称

进入 MENU > Setup > I/O > Local/MADI Configuration，然后点击屏幕左侧的 Dante Expander。

在 Dante Controller 中找到控制台的 Dante Expander。Dante Controller 中列出的名称必须与控制台 Local/MADI Configuration 的 Dante Expander 区域中列出的名称相同。 名称不会自动同步，因此如有需要，必须手动更改。

如果名称不同，请双击控制台屏幕上的名称，然后输入 Dante Controller 中显示的相同名称。

舞台接口箱连接

将舞台接口箱的 Dante Primary 接口连接到一台网络交换机。将控制台的 Dante Primary 接口连接到同一台交换机。

将舞台接口箱的 Dante Secondary 接口连接到另一台网络交换机。将控制台的 Dante Secondary 接口连接到第二台交换机。

注意：也可以将舞台接口箱与控制台的 Dante Primary 和 Secondary 端口直接互连，但这样其他 Dante 设备便无法连接到该网络。在本快速入门指南中，SSL 建议使用交换机。

设置舞台接口箱的 IP 地址

将计算机连接到舞台接口箱 Primary 接口所在的同一台网络交换机（或直接连接到舞台接口箱的 Primary 接口）。

确保计算机与舞台接口箱的 Primary 接口位于同一子网。如果不确定如何操作，请参阅设置 Dante。

打开 Dante Controller。每台实体舞台接口箱应显示两个舞台接口箱设备。如果看不到这些设备，请参阅查找设置为不同子网固定 IP 地址的 Dante 设备。

其中一个设备名称包含“Main”或“Mn”，另一个包含“Comp”。“Comp”设备的各个发送通道名称中也会包含“Compensated”。

“Comp”设备用于两台控制台共享一台舞台接口箱时的增益共享，此处暂不设置。有关 Dante 舞台接口箱增益共享的设置说明， 请参阅Net I/O 舞台接口箱：增益共享。

请注意：在本快速入门指南中，请忽略“Comp”设备。以下所有说明仅适用于“Main”设备。

双击舞台接口箱的“Main”设备，然后进入 Network Config 选项卡。

在 Primary 下，单击 Manually configure an IP Address，然后输入：

  • IP Address：10.101.x.x*
  • Netmask：25.255.0.0
  • 将“DNS Server”和“Gateway”保持为 0.0.0.0

在 Secondary 下，单击 Manually configure an IP Address，然后输入：

  • IP Address：10.102.x.x*
  • Netmask：25.255.0.0
  • 将“DNS Server”和“Gateway”保持为 0.0.0.0

* 出现“x”的位置，请选择 1 至 255 之间的任意数字，但不能使用此网络中其他设备已占用的数字。 同一网络中的多台设备不能使用相同的 IP 地址。

按 Apply。系统会要求确认 IP 地址更改，请单击 Yes。 更改生效前必须重启设备。单击 Reboot。系统会要求确认重启，请单击 Yes。

然后手动重启舞台接口箱：拔下舞台接口箱背面的电源线，再重新连接。

设置舞台接口箱的采样率

在 Dante Controller 中双击舞台接口箱，然后进入 Device Config 选项卡。

在 Sample Rate 区域，从下拉菜单中选择采样率。系统会要求确认采样率更改，请单击 Yes。

注意：控制台与 Dante 网络之间可以进行采样率转换。在本快速入门指南中，请将舞台接口箱设置为与控制台相同的采样率。

如需使用采样率转换，必须使用 Virtual Tie Lines。更多信息请参阅使用 Dante Virtual Tie Lines。

设置 Dante 网络的首选主时钟

无需在 Dante 网络上手动设置首选主时钟。Dante 网络具有自己的时钟选举机制，会自动选择最合适的首选主时钟。

更多信息请参阅时钟设置。

设置舞台接口箱名称

在 Dante Controller 中双击舞台接口箱，然后进入 Device Config 页面。

在 Rename Device 区域输入：

  • 第一台舞台接口箱：IO1-32Main
  • 第二台舞台接口箱（如适用）：IO33-64Main
  • 第三台舞台接口箱（如适用）：IO65-96Main

按 Apply。

如果所有配置均正确……

进入 MENU > Setup > I/O > Dante Configuration。

如果所有配置均正确，舞台接口箱“Main”设备应出现在 Configured Devices 列表中；设备左侧显示一个绿色图标，右侧显示四个绿色图标（用于显示两个 PSU 的状态，以及舞台接口箱是否已正确连接到 Primary 和 Secondary Dante 网络，即“NET”图标）。

屏幕顶部的资源条应显示已使用的 Dante 路由数量及总数。“Inputs”和“Outputs”区域均不应显示“0 of 0”。

如果实际显示与上述任何一项不符，请重新检查第 5 步，确认是否有遗漏，或参阅设置 Dante了解更多信息。


此时已通过 1:1 跳线建立从舞台接口箱到通道输入的路由。对于 Monitor 模板，也已路由 Aux Mix 输出。如需调整这些路由，请参阅路由。


6. Solo 操作

Master Tile（主控区块）（带有两个推子和 Solo 控制的区块）上有两个 Solo Bus 旋钮：Solo A 和 Solo B

旋转此旋钮可提高/降低音量。按下旋钮顶部可将 Solo Bus 静音。

FOH 模板

Solo Bus A 接收 Post All 信号

Solo Bus B 接收 Pre Fader 信号

所有信号路径均发送到两条 Solo 母线。选择使用 Post All 或 Pre Fader Solo Bus，并将另一条 Solo Bus 保持静音。

使用前挡板下方的任一耳机接口监听 Solo Bus。

Master Tile 上的 Main Fader（右侧）控制主 Master Bus 输出。

Monitor 模板

所有信号路径均发送到 Solo Bus A Pre Fader。

除 IEM Aux Mixes 外，所有信号路径均发送到 Solo Bus B Post All。

使用右侧耳机接口连接 IEM。使用本地模拟输出 A12 连接返送音箱。

在 Master Tile 上，Focus Fader（左侧）控制监听 IEM Master，Main Fader（右侧）控制监听 Wedge Master。


更多信息请参阅 Solo（独奏）。


7. 通道视图

按任一推子区块右侧的 SCREEN 按钮。该 Bank 中的通道会显示在主屏幕上。此时屏幕显示 Channel View（通道视图）。

按推子区块上的 CALL 按钮可调出不同的推子 Bank。

SCREEN 和 SWAP 按钮正上方有一块带上下箭头按钮的标签显示屏。按这些按钮可循环浏览 Layer。 浏览时，Bank 标签显示屏（CALL 按钮旁的显示屏）会随之变化。可以将 Layer 理解为 Bank 文件夹；每个 Layer 包含 5 个 Bank。 有关 Layer、Bank 以及配置自定义 User Layer 的更多信息，请参阅 Layer Manager（推子层管理器）。

在通道视图中，点击屏幕上的任意元素，可将该功能发送到推子区块顶部的编码器旋钮（称为 Quick Controls（快捷控制））。

双击通道条底部的彩色方块，可打开该通道的展开视图（称为 Detail View（详细视图））。

可从 Detail View 访问通道处理、Bus/VCA/Mute Group 分配和 Insert。

默认情况下，打开详细视图后，屏幕正下方的各个快捷控制会分别控制详细视图中所显示功能的一个参数。 例如，当详细视图显示 EQ 时，可使用快捷控制控制各个 EQ 参数。如需让快捷控制恢复正常功能，请关闭屏幕右下角的 Follow Detail。

SSL 建议的工作流程：
- 始终保持详细视图打开。
- 保持 Follow Detail 开启。
- 按推子区块上某通道的 SELECT 按钮，将该通道调入详细视图。
- 使用详细视图的多点触控界面或相应快捷控制调节通道处理参数。 （对于 L300、L350、L500、L500 Plus、L550 或 L650 控制台，也可使用 Control Tile（控制区块）（带小屏幕的区块）实现“一项功能一个旋钮”的工作流程。）

8. 对讲

系统有两个对讲通道，其输入分别路由自控制台前挡板下方的两个 XLR 接口输入。

要将对讲发送到 Aux、Stem 或 Master，先打开相应母线的详细视图，然后按 Fader/TB。

启用 TB to Bus

然后在 Master Tile 上按下对讲 TB1 或 TB2 旋钮，取消 对讲通道 1 或 2 的静音。此时即可向该母线讲话。

讲话结束后，再次按下旋钮，将对讲通道静音。

更多信息请参阅 对讲。


9. FX

模板提供 8 条效果通道（使用 Stem 信号路径）。这些效果通道在同一通道中包含 FX 发送和返回；效果器插入在 Stem 上。

系统已创建以下效果：

  • Hall Reverb（大厅混响）
  • Plate Reverb（板式混响）
  • Ambient Reverb（环境混响）
  • Vocal Thickener（人声增厚）
  • Tape Echo（磁带回声）
  • Delay（延迟）
  • Amp Cap（吉他放大器模拟）
  • SubmoniX（次低频谐波合成器）

要将信号发送到效果器，请按推子区块上效果 Stem 通道的 Q 按钮（Q 代表 Query（查询））。

此时，其他信号路径到该 Stem 的所有发送都会显示在推子上（推子翻转会自动开启）。使用推子调节发送电平。

完成后，再次按同一效果 Stem 上的 Q 按钮，退出查询模式。

退出查询模式后，效果 Stem 的推子控制 FX 返回。

请注意：发送到 Stem 和 Aux 时，通常还需要将发送切换为 IN。在模板中，Channel 到 Effects Stem 的发送已预先切换为 IN。 如果创建任何新信号路径或新发送，必须将发送切换为 IN 并提高电平，信号才能通过。更多信息请参阅母线路由。

10. 保存演出文件

进入 MENU > Setup > File Manager > Showfiles 选项卡。

按住 Save，保存对演出文件所做的更改。


11. 关闭控制台

进入 System 页面（MENU > Setup > System / Power > System 选项卡）。

按住红色的 Power Off Console 按钮关闭控制台。等待 PSU 开关由青色变回白色后，再拔下电源线。
