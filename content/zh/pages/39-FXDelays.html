延迟效果

Tempo Link 与 Tap Tempo

下列所有延迟效果器都带有 Tempo Link 按钮，可将同一 FX Rack 中多个延迟效果器的 Tempo 按钮相互联动，并可同时进行节拍点按。

请确保要联动的效果器位于同一效果器机架（垂直列）中。按住每个效果器模块的 Tempo Link 按钮即可加入联动。

点按任一已启用 Tempo Link 模块的 Tempo 按钮，即可同时调整同一 FX Rack 内所有已联动模块的速度（不超过各模块的延迟上限）。

该 BPM 显示区位于 Tempo Link 按钮右侧，用于显示当前效果器模块的 BPM（每分钟节拍数）。双击该显示区可输入指定 BPM 值。还可使用下方的 Note 菜单对 BPM 值进行倍频或分频。

  • 音符时值设为 1/4 （四分音符）时，延迟与点按速度一致。
  • 音符时值设为 1/8 （八分音符）时，延迟速率为点按速度的两倍（即延迟时间减半）。
  • 音符时值设为 1/2 （二分音符）时，延迟速率为点按速度的一半（即延迟时间加倍）。
  • 附点音符值（例如 1/4.）等于相应音符时值的 1.5 倍（因此延迟速率比非附点音符更慢）。
  • 三连音值（例如 1/4t）表示在两个相应音符的时值内演奏三个音符（因此延迟速率比非三连音更快）。

启用 Tempo Link 的效果器模块会保持 BPM 同步（受各效果器最大延迟值限制），但各模块的 Note 值彼此独立，因此不同模块可使用基础点按速度的不同倍数。

也可使用 User Buttons 点按速度。可从 Control Tile（Home > User Buttons）或 Master Tile 硬件访问 User Buttons（请确保 Mode 按钮呈绿色，且相邻的 User 标签亮起）。默认情况下，前六个 User Button 分配给 Tap Tempo，但可在 Options 页面中更改：

  • User Button 1 = FX Rack 1 的 Tap Tempo
  • User Button 2 = FX Rack 2 的 Tap Tempo
  • User Button 3 = FX Rack 3 的 Tap Tempo
  • User Button 4 = FX Rack 4 的 Tap Tempo
  • User Button 5 = FX Rack 5 的 Tap Tempo
  • User Button 6 = FX Rack 6 的 Tap Tempo

多抽头延迟

Multi-Tap Delay 提供 2、4 和 6 抽头版本，最多可提供六个抽头延迟频段；触摸右上角的 T1 至 T6 按钮，选择窗口其余区域当前编辑的频段。

按住这些按钮，可为每个频段调用独立的 Reset 按钮。

乒乓延迟

Ping Pong Delay 模块提供在左右声道之间交替出现的延迟。

磁带回声

Tape Echo 模块模拟模拟式磁带回声设备，除 Drive 和 Wow & Flutter (W&F) 外，还提供 Bass 和 Treble 音色控制。

Reps 控件用于调整反馈回模块的信号量。它可以形成正反馈回路，使效果产生饱和，表现与模拟磁带设备相似。
