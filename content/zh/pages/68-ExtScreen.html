外接屏幕

可将外接屏幕连接到控制台背面的 VGA 输出。 任何至少支持 1280 × 1024 像素并具有 VGA 输入的显示器均可使用。

也可使用外接触摸屏。有关购买信息，请联系 SSL。

使用外接触摸屏时，通过 VGA 将控制台的 Monitor 输出连接到屏幕。 然后使用 USB 2.0 A 转 B 电缆，将屏幕的 USB 端口连接到控制台的一个 USB Expander 端口

说明：
外接屏幕的 VGA 连接应在控制台开机 之前 接到控制台。
用于外接触摸屏的 USB 连线应在控制台开机 之后 接入。

外接屏幕可显示三种不同页面。第一种是 总览 屏幕，用于总览每条路径的设置和信号电平。 第二个是自动化屏幕，可查看场景列表和其他自动化相关信息。 第三个是效果器机架，显示可插入并控制 FX 的槽位页面。 这些页面也都可显示在控制台主屏幕上。

可从选项菜单中选择发送到外接屏幕的页面。请参阅 Console Options 了解更多信息。

总览屏幕

总览以缩略形式显示所有路径。使用右侧按钮选择不同视图。

Zoom 按钮可增大显示器上的路径尺寸。

View Select 按钮按以下方式更改显示的路径：

  • Overview：所有路径类型，按系统层排列（参阅 推子层管理器）。排列顺序如下：
    • Channel 输入通道
    • Stem 编组母线
    • Aux 辅助母线
    • Master 主母线
    • VCA
    • 矩阵输出
    • 对讲路径
    • Solo 输出
  • Inputs：仅输入和对讲通道
  • Outputs：仅输出路径（Stems、Auxes、Masters、Matrix、Solo 通道和 VCA）
  • Tiles：按控制台推子区块当前活动层中的方式显示路径
  • Remote Tiles：按控制台远程推子区块当前活动层中的方式显示信号路径

也可从主菜单中选择总览页面，在主屏幕上查看（MENU > Overview）。

说明： 总览一次只能显示在一个屏幕上；在主屏幕选择总览后，它会从外接屏幕移除，反之亦然。
提示：可将 User Keys 分配为快速将总览指派到外接屏幕或主屏幕。

触摸总览屏幕上的路径，会将其设为控制台的选中路径，以便通过主控区块的 Focus 推子、控制区块（如有）等进行操作。

双击电平表区域可将其展开（与通道视图屏幕中的电平表相同）。这会使总览条带显示更大的电平表，以及路径名称、颜色、静音、独奏和选择状态。

再次双击可缩小电平表并返回先前视图。

说明：
从总览屏幕选择路径不会改变推子区块布局，但选中路径会显示在通道视图中，并且其 Select 按钮在该路径恰好显示于推子区块上时会亮起。 如果使用推子区块的 Select 按钮选择通道，总览屏幕中也会选中同一通道。
控制台连接鼠标后，可使用屏幕按钮控制 Mute、Solo 等基本通道功能，也可双击电平表将其展开或收起。

提示：
  • 输入通道削波时，会在总览页面闪烁红色。
  • 在外接触摸屏上显示总览页面，可让操作员快速识别削波通道并点按选择，无需离开主屏幕上的通道视图。
  • 使用总览屏幕选择多条信号路径，供路径联动或复制使用。 按住 Shift 并点击，或触摸首尾条带，可选择一段连续路径。

自动化

外接屏幕还可显示 Automation。

此页面上半部分列出演出文件中的所有场景。 下半部分可显示多种内容，通过屏幕左下角按钮选择。 如果使用非触摸式外接屏幕，应先决定下半屏显示的内容（例如 Scene Name 用于查看所选场景的名称和备注），然后再将其分配到外接屏幕。

也可随时使用 USB 鼠标更改选择并浏览自动化屏幕。

说明： 自动化一次只能显示在一个屏幕上；在主屏幕选择后，它会从外接屏幕移除，反之亦然。
提示：可将 User Keys 分配为快速将自动化页面指派到外接屏幕或主屏幕。

效果器机架

外接屏幕还可显示 Effects Rack。

此页面上半部分显示可用 FX 槽位及各槽位分配的效果。 可在此添加或移除效果，然后从通道视图进行路由。

页面下半部分显示当前所选槽位及该槽位效果的控制。 显示在外接屏幕时，操作员可快速访问关键效果，例如主唱处理。 也可在此显示 Spectra-lyser 等分析效果。

说明： 效果器机架一次只能显示在一个屏幕上；在主屏幕选择后，它会从外接屏幕移除，反之亦然。
提示：可将 User Keys 分配为快速将效果器机架指派到外接屏幕或主屏幕。
