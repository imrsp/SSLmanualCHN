自动化场景列表

进入 MENU > Automation 可访问场景列表。使用 Show Details 按钮可在列表中显示更多或更少的行。

场景列表各列为每个场景行提供以下功能：

  • Skip 按顺序触发时跳过该场景。详情参阅 场景时间功能 。点击 Skip 列标题按钮可禁用 Skip 列。
  • GO 将场景调用到控制台，使其成为活动场景。点击 Go 列标题按钮可禁用列表中的 Go 按钮。
  • Trigger Time 设置自动触发场景的时间码位置。详情参阅 场景时间功能 。点击 Trigger Time 列标题按钮可禁用时间码触发。
  • Input Actions 详情参阅 场景输入和输出动作 。点击 Input Actions 列标题按钮一次可禁用所选列；再次触摸可禁用所有 Input Action 触发。
  • Fade Time 详情参阅 场景时间功能 。点击 Fade Time 列标题按钮可禁用场景淡变。
  • ID 表示场景在场景列表中的位置。
  • Scene Name 显示用户可编辑的场景名称。
  • Output Actions 详情参阅 场景输入和输出动作 。点击 Output Actions 列标题按钮一次可禁用所选列；再次触摸可禁用所有 Output Actions。
  • Transition 设置场景持续时间；时间结束后，控制台自动过渡到列表中的下一场景。详情参阅 场景时间功能 。点击 Transition 列标题按钮可禁用 Auto Transition 模式。
  • Lock 用于防止编辑场景。详情参阅下方 场景锁定 。
  • Group 定义各场景所属的编辑组（若有）。点击 Group 列标题按钮一次可启用 Group All 模式；再次触摸可全局禁用 Group 模式。详情参阅 场景组 。

场景列表的最后一列用于编辑列表本身。删除、移动或复制粘贴场景的方法如下：

触摸 Edit 按钮（屏幕右上角附近）启用编辑功能；

触摸行末的圆形 Edit 指示灯选择场景。指示灯变红表示该行正在编辑。

可选择多个场景（行）。触摸 Edit 标签本身可选择所有场景。

此时 Edit 按钮下方的三项编辑功能可用： Del （删除）、 Move 和 Copy。

如需删除场景，请触摸其 Edit 指示灯使其变红，然后按住 Del 按钮。

注意：
除非重新加载演出文件，否则删除场景无法撤销。
不能删除活动场景。

如需移动场景，请触摸其 Edit 指示灯使其变红；

触摸 Move 按钮 – 按钮变蓝表示已启用；

然后触摸当前位于目标位置的场景。所移动的场景将放在被触摸场景的上方。

所选场景会移到新位置，其他场景会移动以填补空缺。

如需复制场景，请触摸其 Edit 指示灯使其变红；

触摸 Copy 按钮 – 它会变蓝并变为 Paste 按钮。

然后触摸当前位于新场景粘贴目标位置的场景。

所选场景会复制到新位置，名称后附加 Copy 字样。其他场景会移动以腾出空间。

可按常规方式重命名复制的场景（参阅 场景名称和备注）。

场景 ID

Scene ID 仅用于显示，在软件中不起操作作用。选择 ID 列后， Auto ID 按钮（位于 Detail Display 右侧）决定重新排列场景列表时 ID 编号如何变化：

  • 按钮启用时，ID 列会直接重新编号以保持顺序。
  • 按钮停用时，重新排序场景不会改变 Scene ID；若新场景放在旧场景之间，其 ID 会从前一个场景开始向上递增，必要时使用小数，避免与后一个场景的 ID 相同。
    此后按住 Auto ID 按钮即可按顺序重新编号场景。

通过点击重新编号场景

注意： 请注意，使用此功能前必须关闭 Auto ID 。

点击 ID 列标题可启用点击重新编号。随后可按正确顺序点击场景进行编号，即第一个点击的场景编号为 1，第二个为 2，依此类推。按住 Reorder List 会按点击顺序重新排列场景。

注意：
若两个场景被赋予相同编号，控制台会保留它们彼此之间的相对位置。例如，如果场景列表中“Interlude”位于“My Song”之前，之后两者都被手动编号为场景 1，则“Interlude”仍排在第一，“My Song”排在第二，但两个场景都保留场景编号 1。

手动设置场景编号

在 Scene Detail View 中双击场景编号，然后使用键盘输入所需编号，即可手动为场景编号。

注意： 请注意，使用此功能前必须关闭 Auto ID 。

场景锁定

可将场景设为只读以防止编辑，包括 Input/Output Actions 和逐场景过滤器。

启用 Edit 后，点击挂锁按钮锁定场景。

该场景此后为只读，无法编辑；触发该场景时， Store 按钮会被禁用。

点击 Lock 列标题，可依次在锁定全部、不锁定和单独锁定场景之间切换。
