动态效果

Blitzer™

Blitzer 是一款用途极广的压缩器，既能产生柔和的饱和压缩，也能实现近乎砖墙的硬限幅。它还提供可调瞬态塑形和复古风格谐波 Drive 选项，用于为声音增添色彩。

使用 INPUT 增益控件调整进入压缩器的信号电平，使用 OUTPUT 增益对压缩器输出电平做 trim 补偿。

该 Auto Output Gain 按钮会自动施加一定的输出补偿增益，以补偿所用的 Input 增益和压缩比带来的电平变化。由于压缩曲线取决于多项因素，因此用于抵消 Input 增益的补偿量只是近似值；启用 Auto 并设置其他压缩参数后，仍可手动进一步调整 Output 增益，使湿声（压缩后）与干声的电平匹配得更准确。使用下述 Mix 控件混合两路信号时，这一点尤其重要。

ATTACK 和 RELEASE 控件用于调整压缩器的包络特性。

使用 RATIO 按钮选择压缩比。 1:1 模式可在不改变动态范围的情况下为信号增加染色，而 BLITZ! 模式则提供极为强烈的压缩效果。

TRANSIENTS 和 DRIVE 提供多种染色选项，可塑造信号瞬态并增加温暖的谐波。对应的电平表会指示加入信号的染色量。

该 DETECTION SIDECHAIN 控件可改变触发压缩器的 Key 信号频率内容，例如滤除低频，避免底鼓瞬态触发压缩器而产生抽吸效应。

MIX 控制压缩后湿声与未压缩干声的比例。顺时针旋至 100% 时，压缩器输出为全湿声。

母线压缩器

Bus Compressor 插件基于 SSL 大型模拟调音台中央区域中久负盛名的母线压缩器。它提供高品质立体声压缩，可精确控制音频信号的动态范围。

可将 Bus Compressor 插入立体声混音，使各元素更好地“粘合”在一起，同时保持宏大的声音。它也能有效控制鼓组吊麦或整套鼓的动态。由于提供立体声和单声道版本，几乎所有需要高品质压缩的场景均可使用。

该 In Gain 和 Threshld 控件可改变进入压缩器的信号电平，从而调整压缩量。

该 Makeup 增益控件用于微调离开压缩电路的信号，补偿压缩器造成的电平衰减。

Attack, Release 和 Ratio 控件用于改变压缩器特性。

请注意，由 Threshold 控件设置的压缩拐点会有意随 Ratio 设置而变化。降低 Ratio 设置会降低实际阈值，从而维持压缩信号的主观响度。

效果器顶部区域的模拟式表头显示当前施加的增益衰减量。

Mix 控制压缩湿声与未压缩干声的比例。

侧链选项

正常工作时，Bus Compressor 检测部分（侧链）接收的信号与模块正在处理的音频相同。也可将其他信号路由到侧链输入，使通过效果器的信号根据外部 Key 信号的特性进行处理。使用 Key Route 按钮路由 Key 输入信号，并启用 Ext Key 按钮以开启 External Key 模式。

侧链还包含高通滤波器（HPF），可滤除低频，防止底鼓等低频瞬态使压缩器产生抽吸效应。使用 HPF Freq 推子调整滤波器截止频率；置于最低位置可完全旁通 HPF。

使用侧链 Solo 按钮监听压缩器使用的 Key 信号（位于 Ext Key 和 HPF 选项之后）。

监听话筒压缩器

Listen Mic Compressor 模拟 SSL E 系列调音台监听话筒的声音。它最初用于防止录音棚通话话筒返送过载，其固定 Attack 和 Release 曲线也非常适合鼓组环境话筒，因此成为经典制作工具。

Mix 控制压缩后湿声与未压缩干声的比例。请注意， MakeUp 仅作用于信号的湿声部分。

Comp 控制压缩量，范围为 0 至 100%。

MakeUp 控制对增益衰减的电平补偿。

要模拟原始窄带监听话筒特性，请启用 EQ 按钮；若要在全频段使用压缩器，则保持 EQ 关闭。

注意： Listen Mic Compressor 的时间常数非常快，因此处理低频素材时很容易产生失真。

多段压缩器

Multi-Band Compressor 提供三个压缩频段；触摸左上角的 Low, Mid 或 High 按钮，选择窗口其余区域当前编辑的频段。按钮右侧的滑块用于设置 Low 与 Mid 之间，以及 Mid 与 High 之间的分频点。

各频段的大多数控件含义直观，可能需要说明的是 Max GR ，它限制最大增益衰减量；即使峰值远高于阈值，也只会衰减到 Max GR 设定值，其余动态范围保持不变，并按其他控件设置正常工作。

要重置某个频段，请按住对应的 Low/Mid/High 按钮并选择 Reset;

要重置整个模块，请按住 Reset FX.

侧链

正常工作时，Multi-Band Compressor 检测部分（侧链）接收的信号与模块正在处理的音频相同。也可将其他信号路由到侧链输入，使效果器依据外部 Key 信号特性进行处理。使用 Key Route 按钮路由 Key 输入信号，并启用 Ext Key 按钮以开启 External Key 模式。

可在多个位置 Solo 侧链信号。使用 Solo: 按钮选择 Solo 源点：全频段 Pre-Filter 点，或 Low, Mid 或 High 频段之一的处理后位置。

齿音消除器

De-Esser 是一套压缩与滤波系统，可去除齿音或其他与频率相关的异常声音。

定义： 齿音是人声话筒中常见的共振现象，会使“S”等辅音过于突出。

首先选择 De-Esser 配置。右侧图示说明 Broad Band 和 Band Split 两种齿音消除配置之间的区别。

设置 Filter Type 和 Frequency（频率），确保 De-Esser 响应正确的频率范围。

调整 Threshold 控件，将检测电平设为仅对齿音峰值起作用。

调整 Ratio 旋钮控制齿音移除量。

若只监听被移除的齿音，请按 Solo 按钮。这对于确认 Threshold 是否设置正确非常有帮助。

动态 EQ

Dynamic EQ 提供 1、2 或 4 段版本，各频段使用颜色编码。每个频段对频率的提升或衰减最多达到 up to 最大 Range 值。该增益由压缩侧链电路按比例控制，侧链响应由 Threshold, Ratio 及其他动态控件调整。

每个频段提供以下控件：

  • Filter Type ：EQ 滤波器形状； Bell, Low Shelf 或 High Shelf
  • 触发模式 ：决定动态 EQ 以哪种方式工作： Above Threshold, Below Threshold 或 Auto （取决于 Range；参见 下文)
  • Frequency ：EQ 频段的中心频率
  • Q ：EQ 频段宽度
  • Gain ：动态 EQ 变化所基于的基准增益
  • Range ：相对于基准 Gain 可施加的最大提升或衰减
  • Threshold ：动态 EQ 频段开始工作的电平阈值（高于或低于取决于模式）
  • Ratio ：动态部分启用时的增益衰减比例
  • Attack & Release ：动态部分的起音与释放时间

注意： Dynamic EQ 使用自动拐点：压缩比为 1:1 时拐点为 12 dB，压缩比为 ∞:1 时变为 0 dB；即 Ratio 越高，拐点越硬。

该 View 显示屏顶部的 View 按钮决定图表下方编码器可控制的参数；如果启用了 Follow Detail ，也会决定 Quick Controls 的参数。具体如下：

  • Band View ：显示 Freq, Q, Gain, Range, Threshld, Ratio, Attack 和 Release ，用于所选频段
  • Thresh View ：显示 Freq 和 Threshld ，用于所有频段
  • Range View ：显示 Freq 和 Range ，用于所有频段
  • Gain View ：显示 Freq 和 Gain ，用于所有频段

触发模式

每个频段的 Threshold Trigger 模式可使阈值触发器按以下三种方式之一工作：

  1. Auto ：Range 为负且信号高于 Threshold 时执行衰减；Range 为正且信号低于 Threshold 时执行提升
  2. Above Threshold ：侧链信号高于 Threshold 时触发动态 EQ，与 Range 设置无关
  3. Below Threshold ：侧链信号低于 Threshold 时触发动态 EQ，与 Range 设置无关

使用图表

靶心形控制点表示各频段在未施加动态 EQ 时的基准 Gain 。圆形控制点调整各频段的 Range （最大提升/衰减）。触摸任一控制点时会出现水平虚线，表示可调范围；高亮控制点时捏合图表可调整该频段的 Q 。

白色短划线和点线分别表示 Above Threshold 与 Below Threshold 限值。白色实线在两条限值线之间实时绘制当前施加的 EQ。

例如，当 Trigger 模式设为 Above Threshold 时，在信号超过 Threshold 前，EQ 实线会位于由基准 Gain 参数设定的 Below Threshold 点线上；超过 Threshold 后，EQ 实线会根据当前信号电平以及 Threshold、Ratio 和其他侧链动态参数，移向由 Range 参数设定的 Above Threshold 短划线。

上图示例中设置了正 Gain，且频段处于 Above Threshold 模式，因此信号低于 Threshold 时该频段被提升；一旦信号超过 Threshold，负 Range 参数便使该频段开始衰减，最大衰减量由 Range 值决定。

提示： 使用 Reset 按钮重置单个频段，或按住 FX Reset 按钮重置整个效果器模块（所有频段）。

侧链选项

正常工作时，Dynamic EQ 检测部分（侧链）接收的信号与模块正在处理的音频相同。也可将其他信号路由到侧链输入，使效果器根据外部 Key 信号特性进行处理。使用 Key Route 按钮路由 Key 输入信号，并启用 Ext Key 按钮以开启 External Key 模式。

要监听全频段侧链输入或某个频段，请使用 Solo 按钮。点按可开关 Solo；按住可选择 Solo 源（Pre-EQ, Band 1, Band 2, Band 3 或 Band 4).

噪声门

要启用 Gate，请将 Range 向下拖至合适电平。 Range 控制噪声门工作的动态范围。

左上角按钮用于选择 RMS 和 Peak 响应模式。

该 Threshld 控件定义噪声门开启并允许音频通过的电平。

注意： 噪声门开启电平高于再次关闭的电平。换言之，噪声门打开后，会保持开启直到信号降至更低的关闭阈值。这称为迟滞，可使乐器衰减更自然。 Threshld 控件指的是开启阈值。

Hold 控制信号电平再次开始下降前的保持时间， Release 控制随后电平下降的速度。 Attack 控制信号超过开启阈值后 Gate 完成“恢复”的时间。1.5 ms（每 40 dB）是良好的 Attack 起点，但打击乐声音可能需要约 0.1 ms。

启用 Duck 按钮可将 Gate 变为 Ducker。这会反转噪声门的侧链阈值逻辑：当侧链 Key 输入超过 Range 时，将输入信号衰减 Threshold，例如在 DJ 讲话时自动压低音乐。

左上方的“交通灯”指示 Gate 状态：红色表示关闭（无音频通过），绿色表示开启（音频以全电平通过），黄色表示正在开启或关闭。

模块右侧的电平表显示输入和输出电平。

Sourcerer™ - 主声源隔离器

Sourcerer 是一款动态处理模块，可显著降低甚至消除开放话筒产生的反馈和舞台串音，使通道主声源更加清晰。简直太神奇了！

Sourcerer 可插入单个 Channel，也可借助 Stem 或 Aux 母线跨多个 Channel 组使用。

调整 THRESHOLD 控件，使环境信号不会令 SIDECHAIN LEVEL 表超过蓝色阴影区域。

初始设置时，将 TIME 模式设为 Slow 或 Medium 或 DETECT ，将 Peak ，并将 LPF 和 HPF 滤波器设为 Off （分别完全顺时针和完全逆时针）。

提高 DEPTH 控件，直到环境噪声充分降低。降噪量显示在 GAIN REDUCTION 表上。

要获得理想的 Sourcerer 响应，需要反复试听调整。如果主声源信号被过早衰减，请选择较慢的 Time 模式；如果从环境声切换到目标声源段落时增益衰减响应不够快，则选择较快模式。

使用 HPF 和 LPF 控件限制进入检测电路的频率范围（不会影响主信号），从而进一步提高侧链对环境噪声和反馈的检测灵敏度。 Sidechain Solo 按钮可用于监听滤波后的侧链信号。

启用 Ext Key 按钮可使用不同于待隔离信号路径的另一信号触发检测电路。此时仅当 Key 输入低于 Threshold 时，Sourcerer 才衰减主信号。例如在一个组（如和声歌手）上插入 Sourcerer，并使用组内某人的专用话筒触发时，这项功能非常有用。

瞬态塑形器

Transient Shaper 使用扩展器提高信号 Attack（起音）的幅度，在保持 Decay（衰减）不变的同时增强打击乐声音开头的起音。

I/P Gain 可以控制信号的检测电平，应设置为只检测需要塑形的瞬态。设置过低时 Shaper 不起作用；设置过高则会检测过多瞬态，导致处理过度、起音显得过长。默认 0 dB 是良好的起点。
注意： I/P Gain 不会直接影响输出信号增益。

Amount 控制已处理信号叠加到原始信号中的比例。此处理可能显著提高信号峰值，请密切观察输出表。

Speed 控制附加起音达到峰值后回落至正常信号电平所需的时间。

按下 Invert 可反转处理后信号，使其从未处理信号中相减。这样会柔化起音，使打击乐声音更具厚度。

Solo 可监听处理后信号，帮助完成参数设置。

注意： 当 Invert 和 Solo 按钮同时按下时，信号不会反相。
