临时信号路径联动

可以在多个信号路径之间实时联动控制参数，同时保持相对偏移量。

要设置 信号路径联动，请进入通道视图，然后点按屏幕右侧的 Path Link。

现在点按屏幕上各信号路径的 Eyeconix 或中央情境窗口区域，或使用推子区块上的 SELECT 按钮，选择要联动的信号路径。这些信号路径随即加入联动。

第一个选中的信号路径将成为控制台的已选中信号路径 （在通道视图中以信号路径周围的白色边框表示），并显示在详细视图（若可见） 和 Focus 推子上（若未锁定到其他信号路径）。

所有信号路径类型均支持 信号路径联动，包括 Matrix 通道和 VCA。

注意：只能联动相同类型的信号路径；Channel 不能与 Stem 联动， 完整路径的 Aux 不能与干路径的 Aux 联动，依此类推。 提示：使用范围选择 可从推子区块快速选择多个信号路径。按住第一个信号路径的 SELECT 按钮，再点按范围内最后一个信号路径的 SELECT 按钮，然后松开两个按钮。也可跨相邻推子区块执行此操作。
范围选择同样适用于通道视图和总览页面：单击第一个信号路径，按住已连接键盘上的 Shift 键， 再单击最后一个信号路径，即可联动两者之间所有相同类型的信号路径。

信号路径加入联动后，调整其中一个信号路径上的参数，其他已联动信号路径将相应更新。 这些是相对变化。

按住 Path Link 按钮， 打开包含一系列多选选项的弹出面板：

  • Select All - 联动所有相同类型的信号路径
  • Active Banks - 联动所有推子区块当前显示的全部 Bank 中， 属于所选信号路径类型的所有信号路径
  • Active Layers - 联动所有推子区块当前启用的全部 Layer 中， 属于所选信号路径类型的所有信号路径

可随时启用这些选项。例如，若联动中已有两个完整输入通道，按下 Select All Channels 后，所有完整输入通道都会加入联动。

点按 Clear Selection 将清除当前联动。 启用 Suspend 会暂时停用联动，关闭该选项即可重新启用。

再次按住 Path Link 可切换弹出面板的显示状态，同时保留联动。

完成对联动信号路径的全部调整后，点按 Path Link 清除联动（若弹出面板已打开，也会将其关闭）。

保留偏移量

默认情况下，当联动组中的信号路径达到最大值或最小值时，信号路径之间的相对偏移量不会保留。 从 Live v6.2 起，Path Link 弹出面板中提供 Preserve Offsets 按钮， 可用于更改此行为。

启用 Preserve Offsets 后，只要任一联动参数值达到极限， 控制台就会停止增大或减小所有联动参数值。此设置为全局设置，同时适用于临时信号路径联动和联动组（见下文）。

若关闭 Preserve Offsets 时仍需保留信号路径之间的相对偏移量， 可使用位置最低的信号路径向下调整参数；同理，可使用位置最高的信号路径向上调整。

快速设置

信号路径联动弹出面板还包含 Fast Setup 按钮。 此按钮会展开弹出面板，显示用于快速设置所选信号路径参数的附加控件。

快速设置应用于单个已选中信号路径，但可与信号路径联动配合使用， 同时更改多个信号路径。

注意：快速设置可大幅调整一个或多个电平。 使用此功能时请务必谨慎！

可以执行以下操作：

  • 将 Input Trim 设为 0 dB
  • 将 Input Trim 设为 X dB*
  • 将 Fader 设为 0 dB 或 -∞ dB（关闭）
  • 将 Fader 设为 X dB*
  • Pan：将信号路径置于 100% 左侧或右侧
  • 将 Pan 设置重置为默认值（中央）
  • 将 EQ/Filters 切换为 In 或 Out
  • 重置 EQ/Filters 部分（Out 且曲线平直）
  • 将 Compressor/Gate 切换为 In 或 Out
  • 重置 Compressor/Gate 部分
  • 将所有 Full/Dry** Stem/Aux/Master 发送切换为 In 或 Out
  • 将所有 Full/Dry** Stem/Aux 发送设为 0 dB 或 -∞ dB（关闭）
  • 将所有 Full/Dry** Stem/Aux 发送设为 X dB*

* To ... 按钮会打开数字键盘，可在其中输入数值； 与信号路径联动配合使用时，可按以下两种方式之一应用：

  • 在 Abs（绝对）模式下：已选中信号路径和联动信号路径 都会被设为输入的绝对值（例如，无论此前位置如何，全部设为 -3 dB）
  • 在 Rel（相对）模式下：已选中信号路径会被设为输入值， 联动信号路径的数值将相对于已选中信号路径发生相对变化。例如，若已选中信号路径当前为 -6 dB，某联动信号路径为 -10 dB（低 4 dB），输入 -3 dB 后， 已选中信号路径将设为 -3 dB，联动信号路径将设为 -7 dB （保留相对的 -4 dB 偏移量）。

** 使用 Full 和 Dry 按钮，在母线发送部分的完整与干发送之间切换。

永久联动信号路径

从 Live v6.2 起，可以采用与信号路径联动相同的方式创建永久信号路径组。 这种组称为联动组，可联动一组信号路径的参数，并通过筛选器指定哪些参数保持独立。

创建组

要创建联动组，请按住 Path Link， 然后展开 Link Group 部分。 按照临时信号路径联动的相同方式，选择应包含在联动组中的信号路径。

联动组显示界面会显示当前正在编辑的组编号。每个组都可设置名称和颜色。 双击名称字段即可编辑；按下颜色会打开下拉菜单，可在其中更改组的颜色。

浏览组

使用弹出面板中的箭头在不同组之间浏览。共提供 48 个组。 也可以双击组编号打开键盘，直接跳转到所需的组。

使用 Select First Blank Group 按钮可自动查找第一个不包含任何信号路径的组。

参数筛选

弹出面板的 Enable Links 部分显示可在组内信号路径之间联动的参数集合。

All 将启用所有可用联动。 None 将取消选择所有集合，不保留任何联动。

可联动的部分如下：

  • In：启用信号路径输入部分之间的联动，包括增益、微调等
  • MgVc：启用信号路径静音组和 VCA 分配之间的联动
  • Fade：启用信号路径推子部分之间的联动
  • Stem：启用信号路径 Stem 发送之间的联动
  • Aux：启用信号路径 Aux 发送之间的联动
  • Mast：启用信号路径 Master 分配之间的联动
  • EqFl：启用信号路径 EQ 和滤波参数之间的联动
  • GCmp：启用信号路径门限和压缩参数之间的联动
  • D/AP：启用信号路径延迟和全通滤波器参数之间的联动
  • Pan：启用信号路径声像参数之间的联动

对组所做的更改无需确认；完成所有所需设置和筛选后，直接关闭弹出面板即可。
