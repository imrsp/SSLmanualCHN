系统选项

这些选项涉及电平表响应特性、时钟源等全系统设置。

系统选项位于 MENU > Setup > Options > SYSTEM 选项卡。

电平表选项

可调整以下时间参数：
  • Peak Hold Duration：最高信号电平指示保持可见的时间（0 至 10 秒、Infinite）；
  • Overload Hold：发生过载后，过载指示灯保持点亮的时间（0 至 60 秒、Infinite）；
  • Attack Ratio：电平升高时电平表的响应速度，以相对于电平变化的时间表示（1 至 100 ms/20 dB）；
  • Release Ratio：峰值过后电平表回落的速度，以相对于电平变化的时间表示（100 至 1000 ms/20 dB）。

如需更改数值，可拖动屏幕旋钮，或双击该数值并使用弹出的标准数字键盘进行编辑。

如需为 Peak Hold 和 Overload Hold 选择 Infinite，请先将参数设为最大数值，再用旋钮向上增加一级。

Clear Peaks 和 Clear Overloads 可用于清除不需要的峰值及过载保持指示。

注意： Attack Ratio 和 Release Ratio 设置不适用于 Overview 屏幕上的电平表，因为这些电平表使用固定响应参数。

该 Onscreen Meter Scale 设置可在 -42 dB 与 -60 dB 范围之间切换屏幕电平表刻度。与电平表响应选项相同，刻度按各前面板分别设置，因此主操作面、远程控制端或 SOLSA 控制实例可以使用不同设置。此设置存储为控制台数据，不随演出文件或场景调用。

电平表信号源

可为不同信号路径类型指定电平表取样点：
Input （Trim 前）、 Post Trim、 Pre Fader 和 Post All 选项可分别用于 Input Channels、 Aux Buses、 Stem/Master Buses 和 Solo Channel。

Post Trim、 Pre Fader 和 Post All 选项可用于 Talkback 通道。

Matrix Channel 的电平表始终设置为 Post All。

定义时钟（同步）源（仅控制台）

如需为控制台选择 Main 和 Backup 时钟源，请在 System Clock 区域的列表中选择相应信号源；如果通过 MADI 或 Blacklight II（BLII）连接取时钟，还需选择端口号。按住 APPLY 按钮以设置时钟源。

只有检测到所请求的信号源时，时钟源才会切换。
注意： 该 Current 字段显示当前正在使用的时钟源。 如果 Main 时钟源丢失，系统将使用 Backup；如果 Backup 也丢失，则使用控制台内部时钟。使用 Reclock 按钮可重新尝试锁定 Main/Backup 时钟源。

有关系统时钟的更多信息，请参阅 系统时钟设置 页面。

时间码源

触摸 Timecode 标签下方的按钮，然后从弹出的子菜单中选择信号源，即可设置控制台的时间码接收源。

如果未检测到时间码源，系统会显示手动设置帧率的按钮，以便正确解析输入的时间码值。请选择以下选项之一：
  • 24 f/s ：每秒 24 帧的时间码源
  • 25 f/s ：每秒 25 帧的时间码源
  • 30 f/s ：每秒 29.97 或 30 帧的时间码源（非丢帧）
  • 30 df ：每秒 29.97 或 30 帧的时间码源（丢帧）
时间码可用于触发 自动化场景。

当前时间码值可显示在 状态栏 中（点击时间区域可在时间与时间码之间切换）。

注意： 如果使用 ipMIDI 传输时间码，应使用不同于以下功能所用端口的 ipMIDI 端口： 自动化 MIDI 输入/输出动作、 DAW Control 和 Events。

兼容模式

在较大型控制台上创建的演出文件可加载到较小型控制台上（例如，将 L500 Plus 的演出加载到 L300）。如果演出文件无法适应可用资源，系统会自动进入 Compatibility Mode 。有关 Compatibility Mode 的更多信息，请参阅 控制台配置 章节。

控制台型号（仅 SOLSA）

SSL Off-Line Setup Application（SOLSA）可配置为以任何当前或旧款 Live 控制台型号运行。这样，SOLSA 可模拟指定控制台，以便编辑演出文件和进行远程访问（参阅 Remote Options）。按住 Model Select 按钮，然后在列表中选择相应控制台按钮，即可切换型号。
重要： 更改型号设置会卸载当前演出文件。

声道折叠

将 4.0 和 5.1 格式的 Channels 与 Stems 路由到 Stereo 和 Mono Stems、Auxes 及 Masters 时会应用 Folddown。系统也提供 Stereo 到 Mono 路由的折叠设置。可选择两套全局用户预设折叠方案，并分别调整 Centre 到 Left/Right、Surround Left/Right 到 Front Left/Right，以及整体 Left/Right 电平微调。
每条信号路径可使用 Use Alt Folddown 按钮选择 Default 或 Alternative 设置；该按钮位于 Config Detail View 页面。 注意： 该 Overall Output Trim 参数仅在将带环绕声道的信号源折叠到只有前置声道的目标时生效。

通道直接输出

可在此自定义 Input Channel 的 默认输出取样点 。新建 Input Channel 将采用该默认输出取样点。

可选项为 Post Trim、 Pre Fader、 Post Fader 和 Post All。

注意： 此选项可通过 Automation 过滤器页面中 Console 行的 Output 列进行自动化。

如需逐通道更改 Direct Output Feed Point，请进入该通道 Detail View 中的 Input/Routing 页面，然后点击右侧的 Output:。
