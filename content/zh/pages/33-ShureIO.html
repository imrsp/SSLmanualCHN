Shure ULX-D 和 Axient Digital 无线麦克风控制与监控

有关控制 Shure ANI22 和 ANI4IN 音频接口的说明，请参阅下面的 ANI 部分。

引言

Shure ULX-D 和 Axient Digital 系列无线麦克风能够从 Live 控制台界面进行远程控制。电池和 RF 状态也会在控制台界面中报告。 请注意，只有支持 Dante 的 ULX-D 和 Axient Digital 产品才能被控制台控制；这排除了 ULXD4 接收器。

这是控制台软件中的一项许可功能，可以在 SSL 工厂或现场启用。请联系您当地的 SSL 办事处了解更多详情。 如果未启用许可证，则 Shure 无线设备可以像任何其他 Dante 设备一样使用控制台界面进行路由，但不能从控制台进行远程控制或监控。

设备 IP 地址

每个 Shure 无线接收器有两个 IP 地址；一个用于 Dante 音频，一个用于控制。Dante IP 地址与其他 Dante 音频设备一样在 Dante Controller 中设置。

控制 IP 地址可以在 Shure 接收器菜单中找到（详细信息请参阅相关的 Shure 用户指南），或使用 Shure Wireless Workbench 应用程序中的 Inventory 视图 (www.shure.com/en-US/products/software/wwb6)。

无线接收器和控制台网络配置

控制台和 Shure 无线接收器的 Dante Primary 网络接口必须配置在同一子网中。 同样，如果使用 Dante Secondary 网络接口，控制台和 Shure 无线接收器的该接口也必须配置在同一子网中。 请参阅 Dante 设置部分和相关 Shure 接收器用户指南以获得帮助：

  • ULX-D : pubs.shure.com/guide/ULXD-DQ/en-US
  • Axient AD4D : pubs.shure.com/guide/AD4D/en-US
  • Axient AD4Q : pubs.shure.com/guide/AD4Q/en-US

注意：确保 Dante 和控制网络接口都连接到与控制台相同的网络。

如果接收器设置为 Switched 模式，则 Shure 控制和 Dante 音频数据将共存于其所有四个网络端口；这些端口中的任何一个（且只有一个）必须连接到控制台的 Dante 网络（连接将不是冗余的）。

如果接收器设置为 Split/Redundant 模式，Shure 控制和 Dante 音频将使用设备背面标记的主要和次要端口。 为了正常运行，必须同时连接控制端口和 Dante 音频端口。确保所有端口都连接到正确的主要/次要网络端口。请参阅相关的 Shure 用户指南以获得指导。

使用 Dante Domain Manager 的网络

请注意，对于包含 Dante Domain Manager (DDM) 服务器的网络，控制台必须登录到与 Shure 接收器相同的域才能建立路由和控制它们。 有关为启用 DDM 的网络配置控制台的详细信息，请参阅网络选项部分。

将 Shure 接收器配置为控制台上的设备

为了从 Live 控制台路由和控制 Shure 无线麦克风，首先必须将其添加到控制台 Dante 配置页面中的“Configured Devices”列表中 (MENU > Setup > I/O > Dante Configuration)， 方法是将连接的设备从“Connected Devices”列表移动过来，或使用“Configure New Device”按钮手动创建新设备。 请参阅配置 Dante 设备部分了解更多详情。

重要提示：如果在 V5.0 Live 控制台软件之前，Shure 无线接收器曾被配置为通用 Dante 设备，则必须执行以下步骤将其重新配置为 Shure 设备：

  • 断开 Shure 无线接收器与控制台的连接
  • 重新启动控制台
  • 加载包含配置为通用 Dante 设备的 Shure 无线接收器的演出文件
  • 记下任何到/来自 Shure 无线接收器的路由
  • 从“Configured Devices”列表中删除 Shure（通用 Dante）设备
  • 重新连接 Shure 无线接收器，一旦被控制台检测为正确的设备类型，将它们从“Connected Devices”列表拖放到“Configured Devices”列表
  • 或者，不重新连接设备，而是手动将它们添加为离线 Shure 设备（如果需要修改多个演出文件，这可能会更容易）
  • 重新建立到/来自设备的路由
  • 保存演出文件

配置完成后，必须在控制台的设置页面中输入接收器的控制地址。这允许控制台与设备通信。 当控制台成功与设备建立通信时，相应的 LED 将从红色变为绿色。

控制 Shure ULX-D/Axient Digital 无线麦克风

配置完成后，可以以标准方式将 Shure 无线麦克风输入路由到控制台通道（更多详情请参阅路由部分）。

当路由到控制台通道时，可以从控制台操作界面远程控制每个麦克风输入的输入静音（按住）和输入增益参数（前提是该功能已正确获得许可）。增益可在 -18 到 +42 dB 之间以 1 dB 步长调节。

电池和 RF 状态监控

当路由到 Shure 无线麦克风输入时，电池和 RF 状态信息会显示在通道的 Input Detail View 中。此信息也会显示在通道条顶部，代替 48V 指示器。

如果接收器没有从麦克风发射器接收到任何数据，电池和 RF 指示灯将显示 X。

电池和 RF 状态也可在 I/O 设备视图中找到 (MENU > Setup > I/O > Dante Configuration)。

从“Configured Devices”列表中选择 Shure 设备，然后按下页面下部的“Inputs”按钮。 输入通道将在此列出，每个通道显示麦克风增益、静音状态、电池电量和 RF 功率电平。

与其他 Dante 设备一样，选择一个输入方块会将其分配到侧边栏以进行更详细的查看。按下“Edit Settings”按钮从此处进行参数更改。

注意：Shure 设备的参数只能在设备在线并连接到控制台时更改。

Shure ANI22 和 ANI4IN 控制

有关控制和监控 Shure ULX-D 和 Axient Digital 无线麦克风的说明，请参阅上面的 Shure 无线麦克风部分。

引言

Shure ANI22 是一个支持 Dante 的 2 x 麦克风输入加 2 x 线路输出接口。Shure ANI4IN 是一个 4 x 麦克风输入版本。两者都通过 PoE 供电，并配有 XLR 或 block 音频连接器。

Shure ANI 输入参数可以从 SSL Live 控制台进行远程控制。这是控制台软件中的一项许可功能，可以在 SSL 工厂或现场启用。 请联系您当地的 SSL 办事处了解更多详情。 如果未启用许可证，则 Shure ANI 设备可以像任何其他 Dante 设备一样使用控制台界面进行路由，但不能由控制台进行远程控制。

设备 IP 地址

每个 Shure ANI 设备有两个 IP 地址；一个用于 Dante 音频，一个用于控制。Dante IP 地址与其他 Dante 音频设备一样在 Dante Controller 中设置。

控制 IP 可以通过使用 Shure Web Device Discovery 应用程序 (www.shure.com/en-US/products/software/shure_web_device_discovery_application) 找到，或者通过 DNS 名称查找。 DNS 名称的结构为 "[DeviceName]-[ConnectorType]-[CtrlMacAddress].local"，其中

  • [DeviceName]: ANI22 或 ANI4IN
  • [ConnectorType]: XLR 或 BLOCK
  • [CtrlMacAddress]: 控制 MAC 地址的后半部分（3 个字节），小写（MAC 地址印在设备底部）

例如，一个带有 XLR 连接器且 MAC 地址为 00:0E:CD:40:D6:8A 的 ANI22 将是“ANI22-XLR-40d68a.local”

舞台接口箱和控制台网络配置

控制台和 ANI22/ANI4IN 接口的 Dante Primary 网络接口必须配置在同一子网中。 请参阅 Dante 设置部分和 Shure ANI 用户指南 (pubs.shure.com/guide/ANI22/en-US 或 pubs.shure.com/guide/ANI4IN/en-US) 以获得帮助。

注意：确保 Shure ANI 设备通过支持 PoE 的交换机端口连接到网络。

使用 Dante Domain Manager 的网络

请注意，对于包含 Dante Domain Manager (DDM) 服务器的网络，控制台必须登录到与 ANI 设备相同的域才能建立路由和控制它们。 有关为启用 DDM 的网络配置控制台的详细信息，请参阅网络选项部分。

将 ANI22/ANI4IN 配置为控制台上的设备

为了从 Live 控制台路由和控制 Shure ANI 设备，首先必须将其添加到控制台 Dante 配置页面中的“Configured Devices”列表中 (MENU > Setup > I/O > Dante Configuration)， 方法是将连接的设备从“Connected Devices”列表移动过来，或使用“Configure New Device”按钮手动创建新设备。 请参阅配置 Dante 设备部分了解更多详情。

重要提示：如果在 V5.0 Live 控制台软件之前，Shure ANI 设备曾被配置为通用 Dante 设备，则必须执行以下步骤将其重新配置为 Shure 设备：

  • 断开 Shure ANI 设备与控制台的连接
  • 重新启动控制台
  • 加载包含配置为通用 Dante 设备的 Shure ANI 设备的演出文件
  • 记下任何到/来自 Shure ANI 设备的路由
  • 从“Configured Devices”列表中删除 Shure（通用 Dante）设备
  • 重新连接 Shure ANI 设备，一旦被控制台检测为正确的设备类型，将它们从“Connected Devices”列表拖放到“Configured Devices”列表
  • 或者，不重新连接设备，而是手动将它们添加为离线 Shure 设备（如果需要修改多个演出文件，这可能会更容易）
  • 重新建立到/来自设备的路由
  • 保存演出文件

配置完成后，必须在控制台的设置页面中输入 ANI22/ANI4IN 的控制地址。这允许控制台与设备通信。 当控制台成功与设备建立通信时，相应的 LED 将从红色变为绿色。

使用 Shure ANI22/ANI4IN

配置完成后，可以以标准方式将 Shure ANI 麦克风输入路由到控制台通道（更多详情请参阅路由部分）。 同样，控制台输出也可以路由到 ANI22 线路输出。

当路由到控制台通道时，可以从控制台操作界面远程控制每个 ANI 麦克风输入的 +48V 幻象电源和输入增益参数（前提是该功能已正确获得许可）。增益以 3 dB 步长调节。ANI22/ANI4IN 的其他参数（PEQ、Dante 数字增益、输出电平微调等）无法从控制台界面控制，但可以通过设备的 Web 界面访问。

注意：Shure 设备的参数只能在设备在线并连接到控制台时更改。
