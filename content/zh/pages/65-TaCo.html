TaCo 平板控制

还可使用 TaCo （平板控制）是适用于 iPad、Mac 和 Android 的应用。 该应用可限制为仅控制供表演者使用的单个 Aux 混音，也可解锁,以快速便捷地控制所有推子电平、发送、静音及信号路径处理参数。 最多可同时连接 10 台平板，为舞台上的每位表演者提供混音控制（其中最多支持 2 台处于 Engineer Mode 的平板）。

请注意： 每个 TaCo 连接至少需要 5 Mbps 无线网络带宽。 例如，连接 10 个 TaCo 实例时，至少需要 50 Mbps 无线网络带宽。
安装和更新 TaCo 应用

安装和更新 TaCo 应用

TaCo 应用可在 iPad 和 Mac 的 Apple App Store 以及 Android 设备的 Google Play Store 中找到，搜索“SSL Live TaCo”即可。

点击 Get 或 Install 以在应用商店页面开始安装。 该应用需要读取和修改设备存储的权限，以便在设备上保存本地设置和首选项， 还需要网络访问权限以连接控制台。出现提示时请允许这些权限。

如果已禁用应用自动更新，则需从 Apple App Store 或 Google Play Store 手动更新 TaCo。

要查看当前安装的 TaCo 版本，请打开 TaCo 并查看右下角的版本号。

请注意： 最新版 TaCo 应用兼容 V4.7 及更高版本的所有控制台软件。 TaCo 无法连接更早的软件版本；如使用低于 V4.7 的版本，请联系 SSL 支持或代理商。
将 TaCo 连接到 SSL Live

将 TaCo 连接到 SSL Live

  • 从 Apple/Android 应用商店将 SSL Live TaCo 应用安装到 iPad、Mac 或 Android 平板。
  • 如果在 iPad 或 Android 平板上使用 TaCo，请将主控制台的 Network 端口连接到无线路由器或接入点。 如果使用 Mac 版 TaCo，则可在控制台与 Mac 之间使用有线或无线连接。
  • 将平板/Mac 连接到该网络。 说明： TaCo 会使用检测到的第一个网络适配器。 启动 TaCo 前请禁用所有其他有线和无线网络连接，确保使用正确的网络适配器。
  • 如果无线接入点不为已连接设备分配 IP 地址，请确保控制台 Connectivity 网络与您的设备处于同一子网。 例如，如果控制台的 Connectivity IP 地址为 192.168.1.1，子网掩码为 255.255.255.0，则您的设备 IP 地址应为 192.168.1.[2-254]，子网掩码同样为 255.255.255.0。 请在 MENU > Setup > Options > NETWORK 选项卡中调整控制台的 Connectivity 网络设置。
  • 在平板/Mac 上打开 TaCo 应用。网络中检测到的所有控制台都会列出。
  • 在控制台上，于 MENU > Setup > Options > REMOTE 选项卡中设置控制台密码。
  • 在 TaCo 的 Password 字段中输入相同密码，然后点击 Connect
  • TaCo 的 Status 区域此时会显示 Awaiting authorisation。 一个闪烁的 图标会出现在控制台状态栏中，表示有待处理的连接。
  • 转到控制台的 Remote 选项卡（MENU > Setup > Options > REMOTE 选项卡）， 在 Tablet Controllers 区域找到正在等待授权的设备。按下 Awaiting authorisation，然后选择显示的选项之一。
    • Accept as Engineer 将允许控制所有推子电平、静音状态、路径处理及母线发送电平。
    • 使用 Accept with 按钮选择特定 Aux。TaCo 将仅能控制所选 Aux 混音（例如交给表演者控制自己的混音）。
    • Recent 按钮会显示最近连接过的 Aux 列表。
    • 使用 Disconnect 按钮拒绝连接请求。
    • 使用 Ban 按钮拒绝请求，并阻止该设备再次请求访问。
  • 接受后，控制台与 TaCo 的连接区域应显示 Connected ，控制台状态栏会显示 图标。

要断开连接，请在控制台进入 MENU > Setup > Options > REMOTE 选项卡， 按下 Connected OK 按钮并选择 Disconnect，或在 TaCo 的 Setup 页面点击 Disconnect。

说明： TaCo 也可连接到运行 SOLSA 的 PC 作为远程控制器 （仅当 SOLSA 本身不是控制台的远程端时）。请按上述步骤操作；确保 PC 与平板/Mac 处于同一子网内。
工程师模式

工程师模式

在 工程师模式 TaCo 可远程控制以下控制台参数：

  • 信号路径推子、Solo 和 Mute
  • 声像
  • 输入控制（例如 Gain、+48V、Digital Trim 等）
  • 信号路径 EQ 和滤波器参数
  • 信号路径压缩器和门限器参数
  • 信号路径延时和全通滤波器参数
  • Stem 分配和发送电平
  • Aux 分配和发送电平
  • Master 分配
  • 静音组分配
  • VCA 分配
  • 已插入的效果器机架模块参数
  • 控制台当前选中的信号路径（可选）

最多支持两个处于工程师模式的 TaCo 实例。

在 工程师模式 下，TaCo 有三个主要页面：推子、控制 和 屏幕 Query，说明如下。

推子

要访问 推子 页面，请点击 MENU > FADERS。

此页面可控制控制台的信号路径推子电平、Solo 和 Mute 参数。 点击推子滑块并上下拖动以调整推子电平。按 M（静音）和 S（独奏）按钮切换其状态。

任何时候最多显示 12 个推子，以对应控制台的推子区块。TaCo 首次连接到控制台时，其推子组和推子层使用 System Layers（即按路径顺序列出）。

在 TaCo 中，推子组显示在右侧。点击推子组即可将其调到屏幕上。

Bank 按钮上方有向上和向下箭头按钮，用于循环切换各层。

点击 Layers 按钮打开导航窗口。Overview 显示 System Layers， 而 Layers 则按控制台上的布局显示 User Layers 中的推子组。 点击 User Layer 中的推子组，会将该 User Layer 的 User Layers 分配到屏幕。

调整 Stem 和 Aux 混音

要远程控制 Stem 和 Aux 的 发送电平，请点击 MIXES。

控制台上的所有 Stem 和 Aux 随即显示在屏幕顶部。 点击 Stem 或 Aux 可调整 发送电平 至该母线。 在屏幕顶部向左或向右滑动，可切换查看 Stem 或 Aux。

在 Mixes 模式下，信号路径有两种显示方式：Compressed Query 或 Expanded Query。 这些模式的行为与控制台上的 Query 模式相同。 要选择 Compressed Query 或 Expanded Query，请转到 MENU > SETUP。 Query 模式列在页面底部。

点击 MIXES 可退出混音视图，返回所有信号路径主推子电平的控制。

控制路径参数

双击彩色信号路径名称区域，即可访问该路径的处理参数。 这些参数与 Controls 页面中的参数完全相同，详情见下文。

点击 Close 按钮返回 Faders 视图。

控制

选择 MENU > CONTROLS 可访问 TaCo 中的信号路径控制，也可在 Fader 视图中双击彩色信号路径名称区域。

左上角显示信号路径名称、颜色、编号和格式。可使用箭头键滚动浏览各路径。

在 TaCo 中，可在竖屏顶部或横屏左侧选择要查看的参数。可用选项如下：

  • INPUT：所有输入参数，包括模拟增益和数字微调
  • DELAY/ALL PASS - 延时控制和全通滤波器
  • EQ - 均衡器和滤波器
  • DYN - 动态处理部分，包括压缩器和门限
  • PAN - 声像
  • STEMS - Stem 分配和发送电平
  • AUX - Aux 分配和发送电平
  • MAST - Master 分配
  • MG/VCA - 静音组和 VCA 分配
  • INS A - 插入 A 的效果器单元
  • INS B - 插入 B 的效果器单元

点击任意按钮即可显示对应部分的参数。

信号路径处理以屏幕中央的图形显示，各参数滑块环绕四周。 来自 Effects Rack 的每个效果器单元都有各自的图形和控制。

可通过点击并拖动图形上的控制点，或点击并拖动四周的滑块来调整参数。

EQ 频段 Q 值和动态拐点可使用捏合手势调整。

要启用或旁通信号路径处理或插入点，请点击相应的 IN 按钮。 竖屏时这些按钮位于顶部页面按钮下方，横屏时位于页面右侧。再次点击 IN 按钮可关闭并旁通处理器。

信号路径 EQ、动态处理器以及部分插入效果也可全屏查看。为此请点击 FULL SCREEN。再次点击 FULL SCREEN 可返回正常视图。

在横屏模式下，Fader 按钮可打开当前所选路径的推子。

在 TaCo 中选择信号路径

在工程师模式下，TaCo 可跟随控制台选中的路径，也可独立运行，使控制台和 TaCo 能选择不同路径。

当启用 Follow Console Path 选项时，在 TaCo 的 Faders 和 Controls 页面中选择路径不会影响控制台选中的路径，也不会受其影响。 当 TaCo 远离控制台操作面使用，或作为独立控制器使用时，此工作流程很实用。

启用 Follow Console Path 选项会将 TaCo 的选中路径链接到控制台的 Focus 推子，后者可跟随控制台选中的路径，也可锁定到单独路径。 这使控制台操作面与 TaCo 紧密集成，可在两者之间顺畅操作 （例如将 TaCo 放在 L100、L200 或 L450 的平板区块上，以扩展控制台操作面）。

当 Follow Console Path 启用且 Focus 推子未锁定时，控制台的 SELECT 键可为 TaCo（以及控制台 Focus 推子）选择信号路径。 同样，在 TaCo 的 Faders 页面选择推子通道条，或使用 Controls 页面左上角的路径选择箭头，也会改变控制台 Focus 推子和选中路径。

要将特定信号路径分配并锁定到 Focus 推子，请参阅 Master Tile 部分。 请确保在控制台的 Surface options 页面中启用 Link Control Tile （MENU > Setup > Options > SURFACE 选项卡）。 此后，TaCo 会持续显示所选路径，直到 Focus 推子 解锁或其分配路径改变。

请注意：
对于装有控制区块的控制台，在启用 Follow Console Path 时，TaCo 与控制区块将始终显示相同的路径。

重置参数

某些参数可从 TaCo 重置，例如 EQ 频段增益、声像和 Stem/Aux 发送电平。 支持此功能的参数，其数值旁会显示箭头。

点击增益值会出现弹出窗口。

点击 Reset 按钮可将该参数重置为默认值。

锁定发送至母线的电平

可锁定母线发送电平，以防意外更改。 为此，请进入 Stem 或 Aux 页面。 启用挂锁可锁定，关闭挂锁可解锁。

屏幕 Query

Screen Query 页面（MENU > QUERY）允许用户从 TaCo 查询信号路径，并在控制台的所有推子区块上显示相关路径。

使用 CHAN、STEM、AUX、MAST、VCA 按钮。 一次最多可显示 36 条信号路径。点击 1-36、37-72、73-108 等按钮查看更多路径。

要了解控制台屏幕 Query 的更多信息，请参阅 Screen Query 部分.

在横屏模式下，Fader 按钮可打开当前所选路径的主推子。

Aux 模式

Aux 模式

TaCo 使用与 Live 控制台相同的 Compressed Query 技术，因此只显示已发送到所选 Aux 的通道。

将 TaCo 连接到控制台时，请在控制台 Remote 页面中点击 Awaiting Authorisation，然后点按 Accept with 并选择要控制的 Aux （见上文 将 TaCo 连接到 SSL Live 部分）。

在平板/Mac 上点击 MENU > FADERS 以访问混音页面。

TaCo 会显示所有送入该 Aux 的源路径，并可控制其发送至所选 Aux 的电平。

TaCo 一次显示 12 条路径。对于较长列表，请使用右侧 Bank 图标循环浏览路径。

Aux 模式始终以 Compressed Query 模式运行。

设置

设置

TaCo 提供显示选项，方便用户自定义信号路径的显示方式。这些设置位于 MENU > SETUP.

页面底部有以下按钮：

  • Compressed Query 和 Expanded Query: 决定工程师模式下调整 Mixes 时所用的显示方式（与控制台本地的 Compressed and Expanded Query 功能相同）。
  • Stems before Channels: 启用后，查看 Mix 源列表时 Stem 排在 Channel 之前（仅适用于 Aux 模式和 Engineer Compressed Query 模式）。
  • Left Justify Controls: 将菜单移到屏幕左侧。
  • Follow Console Path：将 TaCo 的选中路径链接到控制台 Focus Fader（Selected）路径（仅限工程师模式；详见上文工程师模式章节）。

点击任意按钮启用，再次点击关闭。

TaCo 的所有页面均支持竖屏和横屏。 要在竖屏和横屏之间切换，只需转动平板方向。（请确保平板设置中未锁定屏幕方向。）

自动重连

发生断线时，自动重连功能会自动恢复 TaCo 与控制台的连接。

TaCo 设备经控制台授权后，如果连接中断并尝试重新连接 在断开后的 30 分钟内, TaCo 将自动重新连接到控制台。设备会自动恢复断线前的相同授权。 例如，若设备仅获准调整 Aux 7 的发送，断线后重新连接时仍会自动恢复为仅授权 Aux 7。 在此过程中，控制台操作员可继续混音，无需重新授权断线设备。

如果设备因电量耗尽而断线，自动重连同样有效。 设备接通电源并重新打开 TaCo 后，只要仍在 30 分钟自动重连时限内，就会自动恢复与控制台的连接。

要检查设备连接状态，请进入控制台的 MENU > 设置 > Options > REMOTE 选项卡。
