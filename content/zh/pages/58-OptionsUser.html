用户选项

这些设置涉及 Solo 模式等用户专用选项。

用户选项位于 MENU > Setup > Options > USER 选项卡。

Solo 设置

可在本区域调整 Solo 设置和 Solo 模式。

各项 Solo 设置的说明请参阅 Solo 章节。

排练模式（仅控制台）

Live Recorder 和 Rehearse Mode 的设置方法详见 录音与排练模式 章节。
排练模式可全局开启或关闭（按住 Global Rehearse Mode 按钮），也可在 User Options 页面的 Rehearse Mode 区域为各个组分别设置。

FFT 分析仪

该 Enable Analyser 按钮位于 FFT ANALYSER 区域，可用于启用或禁用详细视图中的分析仪（操作详情请参阅 信号路径 EQ 章节）。
该 Analyser Follows Selected Path/FX Slot 选项用于确定分析仪显示在详细视图中时，是跟随当前选择的信号路径或 FX 机架槽位，还是停留在明确启用分析仪的信号路径/FX 槽位上。启用此选项后，详细视图的 Analyser 按钮会亮绿，表示它将跟随选择的信号路径/FX 槽位。

VCA 控制

可使用 Lock V-Aux & V-Stem Control 锁定 V-Aux 和 V-Stem 发送的控制（即使用 VCA 控制母线发送电平）；该控件位于 VCA Control 区域（操作详情请参阅 母线路由 章节）。

用户按键

可分配的 User Key 现在位于 USER KEYS 选项卡中。
