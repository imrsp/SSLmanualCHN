G-Flex 图示均衡器

G-Flex EQ 提供完整的 32 段图示均衡。32 段滤波版本可同时使用全部频段，操作方式与传统模拟 32 段图示均衡器相同。

为节省处理资源，还提供 8、16 和 24 段滤波器版本。这些模块可从 32 个可用频段中同时启用 8、16 或 24 段滤波器。滤波器启用时，其滑块下方的按钮会亮青色。达到上限后（DSP 图表达到 100%，其他滑块被禁用），必须点按已启用滤波器滑块下方的对应按钮将其移除并释放 DSP 资源，之后才能继续应用 EQ。

使用 Mode 按钮选择频段滑块的增益范围：±6 dB, 12 dB 和 18 dB.

该 Const （恒定 Q）与 Prop （比例 Q）按钮用于定义 Q 曲线随增益变化的方式：

在 Constant 模式下，无论频段增益如何变化，Q 曲线形状均保持不变；

在 Proportional 模式下，增益偏离 0 dB 越多，钟形曲线越窄。

通常认为比例 Q 均衡器比恒定 Q 均衡器更具音乐性。

此时可拖动各频段滑块进行调整。

如果启用了 Follow Detail ，EQ 频段中亮起并带蓝色边框的部分会分配给屏幕下方的快捷控制。

拖动模块右上角的大滑块可调整信号的总输出增益。

按下相应滑块下方的数字按钮可重置单个频段。

Contour-EQ

Contour-EQ 是一种新型智能图示均衡器，通过生成更平滑的 EQ 曲线并分配 Node（节点）提供更加快速便捷的控制：

Node 是增益已被锁定的频段；移动某个频段滑块时，其周围频段也会随之移动，以被移动频段两侧的 Node 为支点，使整体 EQ 曲线尽可能平滑。

每次移动频段滑块后，该频段都会被设为 Node；除非明确调整，否则它不会再自动移动。

只需触摸频段下方的编号框，即可在不改变增益的情况下创建 Node 频段；编号框变为绿色表示该频段已成为 Node。

再次触摸编号框可取消频段的 Node 分配；该频段及其周围频段会根据最近 Node 的位置重新移动。

白线精确显示当前配置产生的整体效果。

提示： 将中心频率 Node 设为所需增益，并在中心频率两侧以不同距离创建 Node，可获得非对称 EQ 曲线。靠近中心的 Node 一侧曲线更陡，较远 Node 一侧曲线更缓。

为管理 Contour-EQ 使用的 DSP 资源，左上角设有 DSP 表。当该表接近 100%，且一个或多个 Node 按钮亮黄（表示无法达到所需增益）时，说明 EQ 曲线过于复杂；需简化 EQ 形状，直到 DSP 占用下降。

如果启用了 Follow Detail ，EQ 频段中亮起并带蓝色边框的部分会分配给屏幕下方的快捷控制。

拖动模块右上角的大滑块可调整信号的总输出增益。

参量均衡器

参量均衡器的功能与 通道 EQ 详细视图基本相同，不同之处是该模块可提供四段或八段参量均衡器，另加两个滤波器。

如果启用了 Follow Detail ，所有 EQ 类型的频段均可分配给屏幕下方的 Quick Controls；还可使用 Fader Tile 的 Flip 按钮将其翻转至推子。

启用 Analyser 按钮可在 EQ 图上叠加频谱分析仪。按下 Analyser Settings 按钮可调整分析仪的偏移量，并选择显示 EQ 前或 EQ 后信号。

注意： 频谱分析仪仅适用于单声道和立体声 EQ FX 模块。

更多信息请参阅 信号路径 EQ 处理 章节。
