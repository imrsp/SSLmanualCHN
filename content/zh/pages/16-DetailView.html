详细视图

详细视图用于访问信号路径的详细配置。

详细视图对应当前选中的信号路径（可通过区块上的 Select 按钮选择，也可触摸通道视图屏幕中的信号路径进行选择）。

双击通道条中部的快捷显示“缩略图”，可打开与快捷显示区当前所示功能对应的详细视图页面，例如下图中的 EQ。

也可以双击通道视图底部的 Eyeconix 图像和颜色区域，打开信号路径颜色与名称（信号路径配置）的详细视图页面。

也可以双击 Routing（MG （静音组）、 VCA, Fader, Master, Stem, Aux) 或 Processing（EQ、Dynamics、Pan、Time（Delay 与 All Pass）、Insert A 或 Insert B）分配按钮，或者双击通道视图中的 Input 区域，直接打开对应的详细视图 并 将该功能分配给快捷控制。

现在可以在触摸屏上按压或拖动来调整各元素。

点击某个元素（例如 EQ 节点或推子）会使其获得焦点。详细视图右下角的控制项可用于操作所选参数。例如，在上图中点击 EQ 频段 2 的节点后，会将该 EQ 频段的 Frequency, Q 并 Gain 分配给图形右侧的三个屏幕编码器。双击编码器下方的文本字段会调出键盘，可为最近触摸的参数输入精确值。

注意： 页面中的主要参数可分配给屏幕下方的快速控制。为此需要启用 Follow Detail 按钮（位于主屏幕右下角）。触摸此按钮可在以下模式之间切换：
Follow Channel （按钮未启用）：屏幕下方的快速控制跟随所选功能，其行为与其他推子区块上的快速控制相同。
Follow Detail （按钮已启用）：打开详细视图时，屏幕下方的快速控制会分配给所选通道的不同功能。
在以下模式下更改详细视图页面时： Follow Detail 屏幕下方的快速控制会随之更改；其他推子区块上的快速控制保持不变。
详细视图底部的一排标签指示其正下方快速控制所分配的功能。通常，每个通道的快速控制只分配一项功能。开关功能分配给上方快速按键（可用时会亮起），连续参数则分配给快速编码器。

注意： 在用于将通道分配给静音组、VCA 或母线的页面中， Lock 按钮决定触摸选择器时执行的操作：
锁定时，选择器会分配给快速控制。
未锁定时，选择器会分配给快速控制， and 同时切换启用/停用状态。

以下各节介绍详细视图中的每个页面：

配置/Eyeconix

配置/Eyeconix

除了屏幕通道条顶部显示的信号路径编号和类型外，信号路径名称、Eyeconix 和颜色也可帮助用户直观地区分各条信号路径。

信号路径名称显示在推子区块中 Select 按钮上方的信号路径标签屏幕中。

默认情况下，信号路径仅按其类型和编号命名。

信号路径颜色显示在信号路径标签屏幕上方的色条中，也显示在通道视图中屏幕信号路径的顶部和底部。

信号路径 Eyeconix 是显示在通道视图屏幕信号路径底部的图形，用于表示信号路径来源。

双击信号路径底部的 Eyeconix 区域打开详细视图，即可配置这些信号路径显示元素：

Categories 页面右侧的按钮可按主题筛选所显示的图像和词语 – 触摸该按钮，然后从出现的列表中选择类别。

Preset 按钮（右下角）用于打开 Preset 页面，可加载或保存整条通道条的设置。此预设包括以下设置：

  • 信号路径名称
  • Eyeconix 图像
  • 信号路径颜色
  • 推子电平
  • 滤波器设置
  • EQ 设置
  • 压缩器设置
  • 门限器设置
  • 声像设置
  • Delay
  • 全通滤波器设置

（参阅 预设 了解更多信息）

为通道命名

默认情况下，通道名称由类型（CH 表示输入通道， AUX, STEM 或 MASTER）和编号组成。

要重命名通道，请双击 Channel Name 标题下方的标签区域 – 此时会出现键盘；输入新名称并触摸 OK （也可触摸红色 X 关闭键盘并放弃所有更改）。

Next 并 Prev 按钮（使用 USB 键盘时分别对应 Tab 和 Shift+Tab）可快速滚动浏览连续通道，无需在每个通道之间反复关闭和打开键盘。

要将名称重置为通道编号和类型，请触摸 Reset （位于 Channel Name 标题左侧）；

要彻底清除标签，请触摸 Clear（位于 Channel Name 标题右侧）。

也可从标签区域下方的列表快速插入词语 – 只需触摸某个词，即可将其插入当前标签末尾。页面右侧的 Categories 按钮可按主题筛选所显示的词语 – 触摸该按钮，然后从出现的列表中选择类别。

选择 Eyeconix

详细视图中央显示图像列表。触摸所需图像即可选择 – 该图像会出现在通道 Eyeconix 区域中。

要滚动图像显示，请触摸图像区域并向上或向下滑动。

Categories 页面右侧的按钮可按主题筛选所显示的图像 – 触摸该按钮，然后从出现的列表中选择类别。

要清除当前所选图像并将通道 Eyeconix 区域留空，请触摸 Clear 按钮（位于 Eyeconix title.

选择颜色

要为通道选择其他颜色，请触摸详细视图右侧调色板中的所需颜色 – 屏幕通道顶部和底部的名称区域，以及通道标签屏幕上方的色条都会变为该颜色。

注意： 详细视图此页面顶部的图形用于更改通道处理顺序。详情请参阅 处理顺序 。

控制区块

按下控制区块屏幕右上方的 Channel 按钮，可将基本 Eyeconix 和颜色选择功能分配给控制区块：

所选通道显示在屏幕顶部；其两侧的 Prev 并 Next 箭头按钮可用于滚动浏览通道。

颜色列在右侧，Eyeconix 图像列在左侧；在屏幕上向上或向下滑动可滚动 Eyeconix 列表。

要选择图像或颜色，只需在屏幕上触摸相应项目。

要清除 Eyeconix 图像，请触摸左上角的 Clear 按钮（或按下相邻编码器）。

输入/路由

输入/路由

可在详细视图的 Input/Routing 页面中为每个通道单独设置 Direct Output 馈送点。

每个通道都有两个输入，请确保详细窗口顶部正确的输入按钮（Input A 或 Input B）亮起蓝色。如果没有，请按住所需按钮。

有关输入和输出路由的更多详细信息，请参阅下一节 Routing 。当前路由显示在 Route 按钮左侧。

如果输入信号与预期不符，请启用 Attention! 按钮（仅路由至 SSL 舞台接口箱时可用）。这会点亮该输入所路由到的舞台接口箱插口上的 ATT 指示灯，提示舞台技术人员该输入需要检查。可按下 ATT 按钮，熄灭所有 Attention Clear All。

可使用 Pre Trim Solo Input A 并 Pre Trim Solo Input B 按钮，在切换输入前试听传入信号。 Pre Trim Solo 按钮会独奏当前启用的输入（Input A、Input B；如果处于 Rehearse Mode模式，则也可能是 Live Recorder Record Return）。

其余许多控制项仅在通道路由至本地麦克风输入、SSL Network I/O 或 ML 舞台接口箱话放时适用。非 SSL 舞台接口箱、AES 和模拟线路输入仅提供有限的控制项。本节余下内容假定使用 SSL 话放。

页面主体提供用于开启 +48V 幻象电源及反转输入 Polarity。

使用本地模拟输入或 ML32.32 舞台接口箱时，可插入 20 dB Pad 并启用模拟 HPF （70 Hz、每倍频程 6 dB 的高通滤波器）。 High Gain Sensitivity 按钮（按住以启用）可将话放输入灵敏度提高 +15 dB，适用于电平特别低的声源。

使用 Net I/O 舞台接口箱输入时，可使用 Pad （SB i16 和 SB 8.8 为 20 dB；SB 16.12 和 SB 32.24 为 30 dB）、Mic/Line 开关（启用 Mic 进入麦克风模式，停用则进入线路模式）、 Input Mute 并 Aud （试听，仅 SB 8.8 和 i16）以及 Lim （限幅器，仅 SB 8.8 和 i16）。有关 Net I/O 舞台接口箱的更多信息，请参阅 Net I/O Stageboxes 章节。

注意： 与当前输入类型无关的控制项会显示为灰色。有关如何建立路由，请参阅 Routing。

随后可使用 Gain 控制项调整模拟增益，并使用 Trim （通过输入按钮右侧的滑块）进行数字信号微调。

重要注意事项： 如果同一舞台接口箱/本地模拟输入被路由到多个输入通道和/或对讲通道，则每个通道都可能控制与该模拟输入相关的同一组参数（增益、衰减等），因此这些参数会保持同步。这些参数可在“Input”过滤器部分按通道实现自动化；但如果这些通道没有同时加入或退出 Store 与 Recall 范围，共享模拟输入参数的自动化状态可能会意外产生差异。触发含有此类差异的场景，可能导致模拟输入参数出现非预期行为。

初始化模拟输入到通道的路由时生成的数据属于各个通道，而不是各个模拟输入。因此必须确保为每个通道明确存储这些数据。

Balance 控制项会在立体声通道上启用，用于调整左右输入信号之间的电平平衡。

Mono L, Mono R 并 Mono L+R 按钮可将传入的左右信号重新路由至通道后续处理。例如，如果右输入出现噪声，可将左输入路由至立体声通道的左右两侧。又如，在音乐节中可预先配置立体声键盘通道，但某些演出者可能只携带单声道键盘，此时只需按下 Mono L 按钮，而无需重新路由或更改通道格式。仅对于立体声通道， LR Reverse 按钮会交换左右输入信号。

触摸 Route Output 可对通道输出进行路由。请注意，此发送始终位于信号声像处理之前。

output feed point 可通过触摸 Output: 并选择以下选项之一来调整（Post Trim、Pre Fader、Post Fader 和 Post All）。

default output feed point 可在以下位置为输入通道自定义： Menu > Setup > Options > System 选项卡。新建输入通道会采用默认输出馈送点。

快速控制

可直接通过推子区块快速控制使用话放和微调控制。触摸通道视图屏幕中通道条的顶部区域，或按住推子区块快速控制的 Up 箭头，将 Input 部分分配给这些控制项。

按下任一快速控制编码器，可在以下三个页面之间循环： Analogue （红色）、 Digital Trim （蓝色）和 Routing （青色）。

在模拟和数字页面中，情境显示区会在适用时显示增益、微调、幻象电源、衰减、极性等信息。相应页面启用时，主屏幕顶部会以红色显示模拟增益，或以蓝色显示数字微调。

模拟页面中可用的按键包括 +48V, HPF （话放高通滤波器；仅限本地 I/O 和 MADI 舞台接口箱）以及 Pad。编码器控制模拟 Gain。

数字页面中可使用微调前 Solo 并 Polarity 。编码器控制数字 Trim。

注意： 当模拟输入已路由至通道时，可使用模拟控制。所有 Channel、Stem、Aux、Master 和 Matrix 输出路径均可使用数字微调。

Routing 可在主屏幕中部的情境显示区快速查看现有通道输入和输出路由。按键可在 Input A, Input B 并 Output 路由视图之间切换。请注意，这些按键不会切换 Input A 与 Input B 的实际启用状态。

注意： 所有通道的 Direct Output 馈送点均可由用户定义。默认值在 SYSTEM 选项卡中设置（位于 Options 菜单），并可在详细视图的 Input/Routing 页面中为每个通道单独设置 Direct Output 馈送点。

控制区块

可通过主屏幕右侧的控制区块使用通道信号路径输入控制。

按下 Input 按钮（位于控制台标志下方），可将 Input 功能分配给控制区块屏幕：

屏幕周围的编码器会分配给屏幕上距离其最近区域中的功能。

按下编码器可操作开关/按钮；旋转编码器可更改连续参数。

当编码器分配到任何屏幕参数时，其旁边的 LED 会亮起。如果编码器只分配给开关，LED 颜色也会指示开关状态。

所有屏幕按钮也可通过触摸屏幕进行控制。

控制台标志下方 Input 按钮周围的控制项可访问主要输入元素；无论是否将 Input 功能分配给控制区块屏幕，这些控制项始终有效。

增益共享

MADI 和 Dante 舞台接口箱输入可通过路由至多个控制台进行“共享”。其中一个控制台始终被指定为“Master”，并控制话放设置。其他控制台可以接收音频，但无法控制话放。详情请参阅 增益共享 章节。

声像器

声像器

打开详细视图，然后触摸 Panner 页面。

页面左侧的图形表示可进行声像定位的区域，蓝色方块表示音频在空间中的位置。

在图形中拖动方块，可在空间内移动信号的声像位置。

在图形区域横向捏合可调整信号宽度（不适用于单声道路径）。如果宽度已反转（左右交换），蓝色区域边缘的箭头会显示红色；未反转时显示绿色。

页面右侧也提供所有相关声像参数的滑块 – 可用手指拖动滑块，或使用屏幕下方的快速控制进行调整。

注意：
可用参数取决于通道及目标母线的格式。

Width 受 Pan 参数值限制。将 Pan 滑块向左/向右移动会自动减小 Width（如果未触摸其他控制项，将声像移回中央时会恢复 Width）。

同样，调整 Front/Rear Pan 会自动减小信号的 Depth 。

要将声像重新置中，请按住 Reset Pan （位于详细视图右上方）。

使用 Pan Left 并 Pan Right 快捷按钮可快速将 Pan 设置为最左或最右。使用 Pan Centre 按钮可将 Pan 恢复到中央（保留之前的 Width 值）。

LF Only 按钮启用时，只将 LF 通道路由至 5.1 母线。 LF Only 仅当当前查看的路径不是 5.1（即不包含 LF 通道），且该路径已路由至至少一条 5.1 母线时才可用。

如果 4.0 或 5.1 路径已分配给一条或多条 4.0 或 5.1 母线，还会显示 Front/Rear Depth 控制。可在声像图形上纵向捏合以调整 Front/Rear Depth，也可使用右侧对应滑块。

注意： 声像处理在通道路由至母线的位置应用，因此每个 Aux/Stem 混音均可拥有独立声像设置；详情请参阅 母线路由与 VCA 页面。直接输出和全部处理后电平表使用声像处理前的信号。

辅助母线

可在 Auxiliaries 页面中将信号路径分配给相应母线。详情请参阅 母线路由与 VCA。

插入点 A/插入点 B

插入点 A/插入点 B

每条信号路径都有两个插入点，每个插入点都可路由至外部 I/O、控制台内部效果机架中的模块，以及其他多种位置。如果某个 Insert 已路由至 Insert A 或 B，可使用这些按钮访问并路由相应效果。

注意： 所有信号路径类型（包括 Dry Path）都有两个插入点。

默认情况下，两个 Insert 点都位于推子前。

注意： 可在处理顺序中移动 Insert：进入 Eyeconix 详细视图，将通道静音，解锁详细视图屏幕，然后拖动页面顶部显示区中的通道元素；详情请参阅 处理顺序 。

详细视图控制

模块路由至通道 Insert 后，会出现在该 Insert 的详细视图中。可在页面选择器底部按下 Insert A 或 Insert B 按钮访问该视图。此页面是效果机架模块页面的副本：

注意： 如果没有任何内容路由至 Insert 点，此页面将为空。如果发送和返回路由未全部建立，或路由至没有内部控制功能的外部设备，则会显示“Manual Routing”图形。

触摸右侧路由按钮下方的 Insert IN 按钮以启用 Insert 路由。其余控制项因模块而异，具体说明见这些帮助页面的效果机架章节。

启用 Follow Detail 模式，也可将快速控制分配给详细视图参数。触摸 Follow Detail 按钮（主屏幕右下角），可在 Follow Channel （按钮未启用）与 Follow Detail （按钮已启用）之间切换。屏幕底部的标签指示随后分配给其下方快速控制的功能。通常，每个通道的快速控制只分配一项功能。开关功能分配给 Upper Quick Key（可用时会亮起），连续参数分配给 Encoder。

注意： Insert 也可路由至外部效果器或控制台内的其他母线/通道。在这些情况下，Insert 详细信息页面会显示 Manual Routing 图形，表示路由已建立，但没有可用的详细视图控制。
如果没有任何内容路由至 Insert 或从 Insert 路由出去，详细信息页面将为空。

控制区块

路由至内部效果机架模块后，也可通过控制区块控制该模块。

按下 Insert 按钮（位于控制区块屏幕下方），可将 Insert 控制分配给控制区块。

A IN 并 B IN 按钮（位于屏幕下方的 Insert 区域）可用于在 Insert A 与 B 之间切换。

IN 按钮（位于屏幕下方的 Insert 区域）用于启用或停用当前显示的 Insert。

屏幕周围的编码器会分配给屏幕上距离其最近区域中的功能。

按下编码器可操作开关，旋转编码器可更改连续参数。

当编码器分配到任何屏幕参数时，其旁边的 LED 会亮起。如果编码器只分配给开关，LED 颜色也会指示开关状态。

所有屏幕按钮也可通过触摸屏幕进行控制。

注意： 如果 Insert 未路由至内部效果机架模块，控制区块屏幕会显示“Manual Routing”图形；如果没有任何内容路由至 Insert 或从 Insert 路由出去，屏幕将为空。
