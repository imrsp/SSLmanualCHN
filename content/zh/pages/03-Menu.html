主屏幕菜单

Menu（菜单）按钮位于 Main Screen（主屏幕）最右侧一列，用于进行配置。菜单树分为两类：操作和设置：

展开菜单树：
Channel View（通道视图）显示通过 Fader Tile（推子区块）的 Screen 按钮选中的推子区块。这是最常使用的视图。在任何其他菜单中按 Screen 按钮，也会调出此视图。
Automation（自动化）用于管理演出文件中的场景。
Overview（总览）会同时显示演出文件中的所有信号路径，便于快速了解演出文件的整体配置。
Effects（效果器）显示内置效果器机架。
Matrix（矩阵）用于访问输出矩阵。
Screen Query（屏幕查询）可在 Main Screen 中使用 Query（查询）功能。
Setup（设置）打开控制台设置配置菜单：
Options（选项）用于访问各类控制台选项和用户选项；这些选项按不同类别分页排列。
Console Configuration（控制台配置）用于分配 DSP 信号路径。配置工作从这里开始。
推子层管理器（Layer Manager）用于定义实体 Fader Tile 布局。通常应在完成 Console Configuration 后使用。
I/O 用于定义和管理实体音频连接。
XY Routing（XY 路由）可通过 XY 界面批量设置路由。
File Manager（文件管理器）用于访问演出文件管理、处理预设文件管理，以及输入跳线文件的导入/导出功能。
System / Power*（按住）用于正确关闭控制台，并提供诊断工具和系统级设置。 （System 是此项在 SOLSA 中的名称。）
Tools（工具）打开控制台工具菜单：
Talkback Groups（对讲组）用于配置和启用对讲组。
Signal Switcher（信号切换器）可批量切换信号；请参阅信号切换器章节。
Event Manager（事件管理器）用于设置事件。事件由一个或多个状态驱动源组成；当条件满足时，会触发一个或多个动作。
Help（帮助）用于访问 SSL Live 帮助系统。

注意：进入 Setup 或 Tools 菜单后，可按 Setup / Tools 按钮在两者之间快速切换。
