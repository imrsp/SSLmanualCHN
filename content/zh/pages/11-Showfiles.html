演出文件

SSL Live 控制台设置通过演出文件进行保存和调用。这样不仅无需在每次切换项目时重新创建全部设置，还可在不同控制台之间传输配置。

SSL 提供了多种模板，可作为创建自定义演出文件的起点。由于大部分工作已预先完成，控制台设置通常会非常简洁明了。

首先加载 SSL 模板。点击System Templates，选择一个模板，然后按住Load。

注意：主屏幕最顶部的状态显示区会记录多种控制台活动，包括演出文件管理操作。操作失败时该区域会变红。双击消息区域可打开Status Messages窗口查看详细信息。

演出文件通过Showfiles界面保存和加载，进入路径为MENU > Setup > File Manager > Showfiles选项卡。

将出现以下界面：

演出文件列表导航

要访问 SSL 模板演出文件，请点击System Templates。每个模板的内容详情会显示在说明区域。

要访问用户演出文件，先确认System Templates未启用。随后驱动器会列在演出文件列表顶部；内部驱动器显示为HD，USB驱动器按插入顺序排列。点击驱动器进行选择，必要时用箭头按钮滚动列表。

注意：
USB 驱动器可插入控制台上的任意 USB 端口。控制台操作面右上方有一个（L200、L450 和 L650 位于左上方），控制台前部两端下方各有一个，后部有四个。拔出 USB 驱动器前无需执行弹出操作。
USB 驱动器应仅用于文件传输（复制）。不建议直接从 USB 加载或保存演出文件，因为操作期间驱动器可能被拔出。加载时先复制到内部驱动器再加载；保存时先保存到内部驱动器，再复制到 USB 驱动器。

所选位置的内容显示在驱动器下方的演出文件列表中。可在屏幕上滑动来上下滚动列表。

文件夹显示在列表顶部，并带有文件夹图标；双击可打开，点击Up返回上一级。

点击Name或Date Modified列标题，可按字母或日期排序文件（再次点击同一列标题可切换升序和降序）。

点击演出文件即可选择；选中后会显示青色边框。

该Edit按钮可访问其下方列出的功能，具体说明见下文。

加载和保存演出文件时，Edit必须处于未启用状态。

要在当前位置新建文件夹，请点击New Folder。随后可按下文说明重命名文件夹。

加载、保存、另存为

加载

要加载或保存演出文件，先选择要加载或覆盖的演出文件。

确认Edit按钮处于未启用状态。

要加载所选演出文件，请按住Load。

按住Load with Owner Override可加载演出文件，并覆盖网络上当前的所有所有权状态。

现有设备控制和 Dante 订阅所有权状态将被所选演出文件中的所有权状态覆盖。

注意：
使用Load with Owner Override可能会：
- 移除其他设备到本次加载演出文件所用接收端的活动路由。
- 解除网络中其他控制台对舞台接口箱的控制。
务必谨慎确认此操作正确无误。

保存

要将当前控制台配置保存为新文件，请点击Save As。使用弹出的键盘命名新演出文件，然后点击OK。该演出文件随后会成为当前已加载文件。

注意：
SSL 模板演出文件无法覆盖。请使用Save As功能将其保存为自己的演出文件，以保留任何更改。
名称前带挂锁符号的文件无法覆盖，详见下文。

要用当前控制台配置覆盖现有演出文件，请按住Save。

演出文件备注

可在演出文件列表下方的 Notes 显示区输入相关详情。

要为演出文件输入备注，请导航到该文件并选中，使其带有青色边框。文件名会显示在备注区上方。

双击备注显示区内部，使用弹出的键盘输入备注，然后点击OK。

只要在列表中高亮该演出文件，这些备注就会显示在备注区。

编辑模式

部分文件夹操作仅在演出文件列表处于 Edit 模式时可用。

要启用 Edit 模式，点击屏幕顶部中央的Edit按钮。按钮会亮起，屏幕右侧会出现第二个导航窗格。

要选择需执行编辑操作的演出文件，请在左侧导航窗格找到它，并点击其列表项末端的圆圈；圆圈变红表示该文件已启用编辑。

可选择多个演出文件和文件夹进行编辑。点击列表顶部的All圆圈，还可选择当前位置中的全部演出文件和文件夹。

请记住：Edit 模式功能始终作用于已启用编辑的演出文件，而不是带青色边框的高亮文件。

移动和复制演出文件

要移动或复制文件及文件夹，请在左侧窗格导航到源文件夹，在右侧导航窗格导航到目标文件夹；

在左侧导航窗格中，为要移动或复制的项目启用编辑；

然后点击两个窗格之间的Move或Copy按钮。项目将移动或复制到新位置。

注意：
当前已加载的演出文件无法移动。项目不能复制到其当前位置。

将演出文件设为只读

可将演出文件设为只读，以防通过 Save 功能覆盖。

在左侧导航窗格找到要设为只读的演出文件，并为其启用编辑。

然后点击两个窗格之间的Read Only按钮。文件名前会出现挂锁符号。

要取消文件的只读状态，只需重复此过程。

重命名演出文件和文件夹

要重命名文件或文件夹，请在左侧导航窗格找到它并启用编辑。

然后点击窗格之间的Rename按钮，使用弹出的键盘重命名演出文件，再点击OK。

删除演出文件和文件夹

要删除演出文件或文件夹，请在左侧导航窗格找到它们并启用编辑。

然后按住窗格之间的Delete按钮。

注意：文件夹必须为空才能删除。受 Read Only 保护的演出文件无法删除。

备份演出文件

控制台每分钟保存一次包含当前状态的备份演出文件，因此断电时最多只会丢失一分钟的更改。

控制台重新启动时会自动加载最新的备份演出文件，以便立即使用。

注意：如果最后加载或保存的演出文件位于 USB 驱动器，重启控制台时不会加载备份演出文件。这是一项安全功能，可防止来访工程师的演出文件数据留存在系统中。

状态栏指示

当前加载的演出文件名称显示在屏幕最顶部、状态栏中部偏右位置。状态栏中部偏左会显示以下图标之一：

当前演出文件已保存到用户演出文件，且系统未检测到任何待保存更改；
当前演出文件的一部分自上次用户保存后发生更改，且更改已保存到控制台备份系统；
当前演出文件的一部分自上次用户保存后发生更改，但更改尚未保存到控制台备份系统；
系统当前正在保存或加载演出文件数据，数据来自或写入用户文件或控制台备份系统。

演出文件中存储哪些参数？

演出文件所存参数列表可在参数和设置存储页面中查看。

演出文件兼容性

旧演出文件始终可以在新版软件中加载。但由于旧版软件无法识别新增功能数据，新演出文件可能无法在旧版软件中加载。这也适用于SOLSA版本，因此应使所有 SOLSA 安装版本与控制台软件保持同步更新，确保能够离线编辑演出文件。
