控制台操作面概览

LIVE 控制台操作面由三个 Fader Tiles（推子区块）、一个 Control Tile（控制区块）和一个 Master Tile（主控区块）组成。其中部分区块为选配。

Main Screen（主屏幕）是一块多点触控显示屏，用于访问音频参数和设置功能。

Fader Tiles 是操作音频的主要实体界面。一个 12 推子区块可通过一系列控制层和 Bank（参阅 推子层管理器章节）控制多达 360 个不同推子。 推子可控制单声道、立体声、LCR、4.0 或 5.1 音频信号路径。

每个 Fader Tile 顶部的 Quick Controls（快捷控制）可访问大部分信号路径参数（增益、处理、声像、Aux、母线分配、VCA 分配等）。可以将这些控制器理解为一条完整的模拟通道条，只是通过多层软按键进行了符合人体工程学的紧凑设计。

Control Tile（仅 L300、L350、L500、L500 Plus、L550、L650）提供另一种访问相同信号路径参数的方式。它同样配有多点触控显示屏。你可以按偏好选择触摸屏、Quick Controls、Control Tile，或组合使用。

Tablet Tile（平板区块，仅 L100、L200、L450）用于放置运行 SSL TaCo Tablet Control 应用的平板电脑。 Tablet Tile 配有 iJack 立体声输入（仅 L100、L200）和 USB 充电端口。有关 TaCo 的更多信息，请参阅 TaCo：平板控制。

Master Tile 用于自动化、Master、Solo 和 Talkback 操作。

电源

要开启控制台，请确认控制台背面的一个或两个电源开关已打开，然后按 Channel Control 或 Tablet Tile 顶部的 PSU 按钮 （Remote Expander 的按钮位于操作面背面）。PSU 按钮的颜色表示控制台电源状态：

  • Off（熄灭）- 未连接交流电源，或交流电源未开启。
  • Backlit white（白色背光）- 已接通交流电，可以开机。
  • Cyan（青色）- 控制台已开启。

主电源（A）和备用电源（B）的状态由 PSU 按钮旁的 LED 指示（不适用于 L100）：

  • Green（绿色）- 已开启且工作正常。
  • Green flashing（绿色闪烁）- 已开启，但在过去 20 秒内曾发生供电中断。
  • Red flashing（红色闪烁）- 已关闭或存在故障。

注意：必要时，控制台可以仅使用一个电源运行。两个电源之间的切换不会中断工作。

要关闭电源，请进入 System page（系统页面） （MENU > Setup > System / Power > System 选项卡）。 按住红色的 Power Off Console 按钮。

控制台软件可承受运行中出现的错误并妥善恢复。 但如果需要在音频继续通过的同时仅重启前面板操作面，可使用 PSU 按钮上方小孔内的复位开关。 也可在 System page 中单独重启各进程， 例如 Fader Tiles 无响应但主屏幕仍可操作时。

亮度控制

Control Tile 的电源按钮附近（如配备）有三个亮度编码器，分别控制触摸屏、按钮和顶饰条照明灯：

  • 旋转可改变亮度。
  • 按下可调用已存储的亮度值。
  • 按住可存储当前值，以便之后调用。

这些控制项也可在 Console Options 选项卡（MENU > Setup > Options > CONSOLE）中使用。

这些设置保存在演出文件中，也可以进行自动化。

状态栏

主屏幕顶部设有状态栏，用于显示操作过程中的实用信息。 消息显示在状态栏左侧。 信息消息显示为白色，警告消息显示为橙色（如图），错误消息显示为红色。 双击消息区域可打开消息历史记录窗口。 严重演出消息显示为紫色，并保留在状态栏中，直至点击确认。 状态栏的其余区域说明如下：

用于指示演出文件保存状态、 排练模式 状态、 原位独奏 状态、 兼容模式/Plus Pack 状态、 远程操作面连接状态、 IP 地址冲突警告 和 时钟源警告 的图标。
当前所选自动化场景的名称和编号。* 表示存在未保存的更改。
当前演出文件名称。
当前时间或时间码值（点击可切换）。
当前所选信号路径。
