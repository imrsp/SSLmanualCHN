信号路径处理

完整处理路径（Full Path）包含 EQ、动态、延迟区域以及全通滤波器。


EQ

EQ

所有完整处理路径均配有四段参数均衡器、高通滤波器和低通滤波器。

要访问 EQ，请打开详细视图并选择 EQ 页面。

要启用或旁通 EQ 或滤波器，请触摸页面右上角的 EQ IN 或 Filter IN 按钮；启用后相应按钮会“点亮”。

可用手指或鼠标拖动 EQ 图形中的节点。要更改 Q 值，请使用捏合手势收窄或加宽曲线形状。

EQ 形状

EQ 和滤波器的每个频段都有三种形状选项，不同频段类型的选项也不同。

对于滤波器频段，可选择每倍频程衰减 12 dB、18 dB 或 24 dB 的滤波斜率。

EQ 频段提供以下形状：

Band 1（LF）：Low Shelf（低频搁架式，Q 值可调，可在转折频率处产生过冲/下冲）、ConstantQ（Q 值可调，且增益变化时曲线宽度保持不变）或 Legacy（模拟典型 SSL G-Series EQ 的 Q 值特性）。

Band 2（LMF）和 Band 3（HMF）：ConstantQ、Legacy 或 Notch（陷波，增益衰减量固定为最大值；Q 值和频率仍可按常规方式调整）。

Band 4（HF）：High Shelf（高频搁架式Q 值可调，可在转折频率处产生过冲/下冲）、ConstantQ 或 Legacy。

要更改频段形状：

触摸相应频段的节点；节点会以改变颜色表示已被选择；

触摸 EQ 显示图右侧、Filter IN 按钮下方的按钮，打开形状选项菜单，然后选择所需形状。

FFT 分析仪

详细视图的 EQ 图形可非常精确地显示 EQ 和滤波器的频率响应，既可显示各个频段，也可显示总体效果。

使用 Analyser 按钮查看信号的频谱分析。 此分析仪每倍频程使用固定点数，因此在整个频率范围内均能提供同等精细的分辨率，即使低频也不例外。

可使用 Analyser Settings 弹出面板自定义分析仪显示。

  • Display 的 Slow/Fast 是全局设置，用于确定分析仪的更新速率。
  • Feed Point Pre Filter/Post Filter/Pre EQ/Post EQ 用于设置分析仪的信号源。这是按路径保存的设置。请注意，Filter 和 EQ 模块可在信号路径处理顺序中自由移动。
  • Offset 调整送入分析仪的信号电平。这是按路径保存的设置，不会影响该路径的实际可听信号电平。

Live 支持连接多个操作面来远程控制音频引擎，包括其他控制台操作面和 SOLSA PC 应用程序 （详情请参阅远程控制章节）。 每个“操作面”（包括 SOLSA）都可在所选路径上显示 EQ 分析仪，但分析仪一次只能用于一条路径。 因此，可在主操作面以外的其他操作面上禁用 Analyser 视图，防止其他用户更改分析仪所分配的路径。 可在 User Options 页面（MENU > Setup > Options > USER）中进行设置。 更多详情和选项请参阅用户选项章节。

注意：分析仪仅适用于单声道和立体声路径。

重置 EQ

要重置单个 EQ 或滤波器频段，请按 EQ 图形右侧的 Band Reset 按钮，并从下拉列表中选择所需频段。

要将整个 EQ 或滤波器重置为平直状态，请按住 EQ IN 或 Filter IN 按钮，然后选择 Reset 下拉列表选项。


快捷控制

Follow Detail 模式

屏幕下方的快捷控制也可分配给详细视图参数。 为此，需要启用主屏幕右下角的 Follow Detail 按钮。

启用 Follow Detail 后，屏幕下方的快捷控制 会分配给所选通道的不同功能 （仅在详细视图打开时适用）。其他推子区块上的快捷控制不受影响。

详细视图底部的标签表示分配给其正下方快捷控制的功能。 通常，每个通道的快捷控制一次只分配一种功能。 开关功能分配给上部快捷按键（可用时会点亮），连续参数分配给下方旋钮。

Follow Channel 快捷控制模式

在通道视图或详细视图的 EQ 区域轻触一次，将 Filters/EQ 发送到快捷控制。

按推子区块旋钮（或双击屏幕上的虚拟旋钮），在四个 EQ 频段（青色）与两个滤波器频段（紫色）之间循环切换控制；

在任意四个均衡器频段页面中按 In 按钮，将 EQ 接入信号链。同样，要将滤波器接入信号链，请在滤波器页面按 In 按钮；

对于均衡器频段，使用底部两个快捷控制按钮，在频段频率（左按钮点亮）、增益（右按钮点亮）和 Q（两个按钮均不亮）之间切换按键所控制的参数；

对于滤波器频段，使用底部两个快捷控制按钮，在高通频率和低通频率之间切换按键所控制的参数；

然后拖动旋钮调整所选参数。

图形显示区显示当前 EQ 设置。EQ 键帽上也会显示此图形的简化版本；当图形显示区未显示 EQ 时，该简图仍保持可见。

双击图形显示区可打开通道 EQ 详细视图，在其中编辑更高级的 EQ 参数。

如果某个 EQ 频段的增益控件呈灰色，该频段很可能设置为 Notch（陷波）。

控制区块

要启用或旁通四个 EQ 频段，请按 EQ IN 按钮，该按钮位于屏幕下方的 EQ 区域；要启用或旁通滤波器，请触摸 Filters IN 按钮。

要重置 EQ 频段或滤波器，请按住相应的 IN 按钮，直到其闪烁。

屏幕中央的图形显示当前 EQ 和滤波器设置。

也可通过触摸调整图形。

屏幕两侧的旋钮分别分配给四个 EQ 频段的频率、Q 和增益，具体功能显示在靠近各旋钮的屏幕区域中；要更改某频段的 EQ 形状，请按其 Q 旋钮，在 Q 值条下方出现的设置间循环切换。

滤波器频率在屏幕底部控制；触摸各频率值条上方的按钮选择滤波斜率。

所有屏幕按钮均可直接触摸操作。

动态处理

动态处理

所有完整路径均配有压缩器和门限器/扩展器。

要访问 Dynamics 区域，请打开 Detail View 并选择 Comp Gate 页面。

要启用或旁通压缩器或门限器，请触摸页面右上角的 Comp IN 或 Gate IN 按钮；相应动态处理单元启用后，按钮会“点亮”。

可用手指拖动所显示图形中的节点。

当两个节点重叠时，双击节点可切换当前选择。

要更改压缩器或门限器的 Hold 值，请用双指捏合 Comp Time 或 Gate Time 图形。

要更改压缩器或门限器（仅扩展器模式）的拐点值，请用双指捏合主图形中的 CT 或 GT 节点。

显示选项

左侧图形上方的按钮用于定义所显示的参数，汇总如下：

  • Compressor：页面左侧显示 Threshold 和 Ratio，中间显示增益，右侧显示时间参数和 Compressor Setup；
  • Compressor 和 Comp Filters：左侧显示滤波器频率，中间显示压缩器增益，右侧显示时间参数和 Compressor Setup；
  • Gate：左侧显示 Threshold 和 Range，右侧显示时间参数和 Gate Setup；
  • Gate 和 Gate Filters：左侧显示滤波器频率，右侧显示时间参数和 Gate Setup。
  • Compressor 和 Gate：左侧同时显示两者的 Threshold 和 Ratio，中间显示压缩器增益，右侧显示压缩器和门限器的时间参数。

动态处理显示

门限器启用时，图形顶部会显示“交通灯”式门限表：红色表示信号未通过，绿色表示门已打开，琥珀色表示门限器处于阈值电平。

每个动态处理模块的 Setup 页面会显示该模块的基本路由；触摸所显示的元素即可启用，元素变为绿色表示已启用。

压缩器增益

要手动调整压缩器的补偿增益，请确保页面中的 Compressor 已启用，然后上下拖动页面中央的 COMP 滑块。左侧图形会随所加入的增益发生变化。

也可让控制台自动设置补偿增益：触摸 Auto Gain 按钮（位于 COMP 滑块上方）；按钮点亮表示已启用，同时滑块会停用。

压缩器混合控制

信号路径 Dynamics 区域中的 Comp Mix 控件可便捷地在任意路径上使用并行压缩处理。

可将压缩信号与未压缩信号并行混合。设为 100% 时仅使用压缩信号；设为 0% 时仅通过未经压缩的干信号；50% 时压缩信号与未压缩信号各占相同比例。启用 Follow Detail 后，可使用屏幕推子或对应的 Quick Control Encoder 设置百分比。

启用电子管压缩模拟时无法使用 Comp Mix。启用 Tube 后，混合控制推子帽会消失，混合比例自动设为并显示为 100%。此前的混合百分比会保留，以便停用 Tube 后恢复。

侧链

门限器和压缩器均配有带滤波器及外部输入路由的侧链。

要将高通和低通滤波器插入 Compressor 和/或 Gate 侧链：

触摸 Compressor 或 Gate Setup 显示中的 Filters 按钮；按钮点亮表示已启用；

触摸左侧图形上方的 Comp Filters 或 Gate Filters 按钮；随后会出现带 HPF 和 LPF 节点的频率图形。

拖动节点可更改各滤波器的 -3 dB 点。

要将其他信号路由到 Compressor 或 Gate 侧链，请触摸右侧图形右边的 Comp Key Route 或 Gate Key Route 按钮，并在出现的标准路由界面中找到该信号。（请参阅路由，了解路由界面的操作说明。）

然后触摸 Compressor 或 Gate Setup 显示中的 Ext Key 按钮以启用外部侧链；按钮点亮表示已启用。

要监听 Compressor 或 Gate 侧链，请触摸右侧图形上方的 Comp Key Solo 或 Gate Key Solo 按钮。

注意：Key Listen Solo 信号会路由到通道 Misc Solo 设置中指定的 Solo 母线；详情请参阅 Solo。

电子管模拟

可在压缩电路中加入电子管压缩器模拟：

触摸 Compressor Setup 显示中的 Tube 按钮；按钮点亮表示已启用。

峰值模式

信号路径压缩器提供 RMS 和 Peak 两种检测方式。

RMS 是默认检测模式。要启用 Peak 检测，请按 Compressor Setup 显示中的 Peak Mode 按钮。

扩展器模式

触摸 Gate Setup 显示中的 Exp 按钮，可将 Gate 切换为 Expander 模式。

闪避模式

启用 Duck 按钮，将 Gate 变为 Ducker。 这会反转 Gate 侧链阈值逻辑的工作方式：当侧链（Key）输入信号超过 Threshold 时，输入信号会按 Range 数值衰减， 例如在 DJ 讲话时自动压低音乐。

预设

要为通道加载 Dynamics 预设，请触摸页面右下角的 Preset 按钮，然后在出现的标准 LIVE Preset 页面中选择预设。

重置动态处理数值

要将压缩器或门限器重置为 1:1，请按住 Comp IN 或 Gate IN 按钮，然后选择 Reset 下拉列表选项。


快速控制

Follow Detail 模式

屏幕下方的 Quick Controls 也可分配给 Detail View 参数。 为此，需要启用 Main Screen 右下角的 Follow Detail 按钮。

启用 Follow Detail 后，屏幕下方的 Quick Controls 会分配给所选通道的不同功能 （仅在 Detail View 打开时适用）。其他 Fader Tile 上的 Quick Controls 不受影响。

Detail View 底部的标签表示分配给其正下方 Quick Controls 的功能。 通常，每个通道的 Quick Controls 一次只分配一种功能。 开关功能分配给 Upper Quick Key（可用时会点亮），连续参数分配给 Encoder。

Follow Channel Quick Control 模式

在 Channel View 或 Detail View 的 Dynamics 区域轻触一次，将 Comp/Gate 发送到 Quick Controls。

按 Fader Tile Encoder（或双击 Screen Encoder），在 Compressor（绿色）和 Gate（浅绿色）控制之间循环切换；

按 In 按钮，将 Compressor 或 Gate 接入信号链；

使用 Lower Quick Controls，在 CT 与 CR（Compressor Threshold 和 Ratio）之间，或在 GT 与 GR（Gate Threshold 和 Range）之间切换 Encoder 所控制的参数；

然后拖动/转动 Encoder 调整动态处理。

Context Display 显示当前门限器和压缩器设置。Compressor/Gate 按钮帽上也会显示此图形的简化版本；当 Context Display 未显示动态处理时，该简图仍保持可见。

实际增益衰减情况显示在控制台操作面上、通道推子右侧的 COMP 和 GATE 电平表中。

双击图形区域可打开扩展显示，在其中编辑更高级的动态处理参数。

控制区块

按 DYN 按钮（位于 Control Tile Screen 下方的 Dynamics 区域），可将动态处理控件分配给 Main Screen 右侧的 Control Tile：

要启用或旁通 Compressor 或 Gate，请按 Comp 或 Gate IN 按钮；这些按钮位于屏幕下方的 Dynamics 区域。

要重置 Compressor 或 Gate，请按住相应的 IN 按钮，直到其闪烁。

也可通过触摸调整图形。

所有屏幕按钮均可直接触摸操作。

屏幕周围的编码器分配给其最近屏幕区域中的功能；压缩器控件位于左侧，门限器控件位于右侧。

按下编码器可操作开关，转动编码器可更改连续参数。

延迟

延迟

Live 中每条 Full Path 的信号最多可延迟 2 秒。

打开 Detail View，然后轻触 Delay All Pass 页面。

要启用或旁通 Delay，请触摸页面右上角的 Delay IN 按钮；启用后按钮变为蓝色。

可用手指拖动图形中的 Del 节点。

图形下方有一个滚动条。如果尝试将节点拖出左边界或右边界，显示内容会随之滚动。

可拖动滚动器更改延迟。节点会保持在显示区域中的位置，而其后的图形会滚动。

要放大或缩小，请使用双指合拢或张开。

延迟量显示在图形左上角。图形右侧的 Time 和 Distance 按钮用于定义单位；每个按钮会在相应单位间循环切换（采样/毫秒/秒，以及厘米/米）。

注意：延迟值始终以采样数存储。更改控制台采样率会改变以其他单位表示的延迟时间。

要精确设置延迟量，可使用单位按钮下方的 Delay Time – 和 + 按钮，每次以一个采样为单位增减延迟。也可选择 Del 节点，双击右下角的数值框，然后使用出现的键盘输入。

提示：以下方式可能是操作图形最高效的方法：
- 将图形放大到较近视图，并让 Del 节点保持在中央；
- 使用 Scroller 设置大致延迟值；
- 再移动 Del 节点，对数值进行微调。 注意：信号路径延迟用于延迟整个信号，并非用于制作延迟效果。可将控制台效果引擎中的延迟效果路由到路径插入点。请参阅效果器机架了解更多详情。

全通滤波器

全通滤波器

All Pass Filter 可用于调整特定频率范围的相位。 在解决楔形返听音箱的反馈问题，或同一乐器使用多个麦克风时，全通滤波器尤其有用。 当两个信号的相位关系接近完全相反时，简单的 Ø 反相按钮最为有效，但实际情况往往并非如此。 All Pass Filter 是解决相位相关问题精确得多的工具。

打开 Detail View，然后轻触 Delay All Pass 按钮。

要启用或旁通 All Pass Filter，请触摸页面右上角的 All Pass IN 按钮；启用后按钮变为蓝色。

使用 2nd Order 和 All Pass Ø 按钮定义滤波器类型。

可用手指或鼠标拖动图形中的 AllP 节点。

在二阶模式下，要更改 Q 值，请使用捏合手势。

滤波器的 Q 值和中心频率显示在图形左上角。

要将滤波器重置为默认值，请按住 Detail View 中的 All Pass IN 按钮，然后选择 Reset 下拉列表选项。

快速控制

Follow Detail 模式

屏幕下方的 Quick Controls 也可分配给 Detail View 参数。为此，需要启用 Main Screen 右下角的 Follow Detail 按钮。触摸该按钮可切换状态：

屏幕下方的 Quick Controls 会分配给所选通道的不同功能（仅在 Detail View 打开时适用）。 其他 Fader Tile 上的 Quick Controls 不受影响。

Detail View 底部的标签表示分配给其正下方 Quick Controls 的功能。 通常，每个通道的 Quick Controls 一次只分配一种功能。开关功能分配给 Upper Quick Key（可用时会点亮），连续参数分配给 Encoder。

Follow Channel Quick Control 模式

在 Channel View 或 Detail View 的 Delay 区域轻触一次，将 Delay/All Pass 发送到 Quick Controls。

按 Fader Tile Encoder（或双击 Screen Encoder），在 Delay（洋红色）和 All Pass（粉色）控制之间循环切换；

按 In 按钮，将 Delay 或 All Pass 接入信号链；

转动 Encoder 调整频率。

通道 Quick Controls 会在 Delay 和 All Pass 滤波器控件之间循环切换；按 Fader Tile Encoder（或双击屏幕 Encoder）切换到正确显示。

在 All Pass 显示中，按 In 按钮将滤波器接入信号链。

左下按钮（Freq）定义分配给编码器的参数； 取消选择该按钮后，Encoder 控制滤波器的 Q。

右下按钮控制 All Pass Ø 功能。

控制区块

按 Control Tile Screen 上方的 Time 按钮，可将 All Pass 滤波器控件分配给 Control Tile：

要启用或旁通 All Pass 滤波器，请按 AllPass IN 按钮；该按钮位于屏幕上方的 Time 区域。

要重置 All Pass 滤波器，请按住 IN 按钮，直到其闪烁。

滤波器控件分配在屏幕下半部，Delay 控件分配在屏幕上半部。

位于 All-Pass Filter 图形左侧的编码器可用于更改滤波器频率和 Q，具体功能由屏幕内相邻的数值条指示。

也可左右拖动 AllP 节点来更改中心频率。

左下角的按钮可用于更改滤波器阶数和相位。
