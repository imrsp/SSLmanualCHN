远程推子区块

Remote Tile（远程推子区块）是一个含 12 个推子的独立区块，可作为任何 SSL Live 控制台的远程操作面。 其功能与控制台上的推子区块完全相同。

一台控制台最多可同时连接 2 个远程推子区块。


将远程推子区块连接到控制台

远程推子区块通过 USB 连接控制台，且带有内部电源。 将远程推子区块接通电源，连接到控制台的 USB Expander ports 端口，使用 USB A 转 B 电缆。

请注意： 远程推子区块 必须通过控制台的 USB Expander 端口之一接入。 请勿通过其他 USB 端口连接远程推子区块。

如果这是远程推子区块首次连接控制台，请进入 System 页面（MENU > Setup > System / Power > System 选项卡）。 按住 Rebuild Hardware Configuration，然后重启控制台。 这样即可使控制台支持远程推子区块。硬件配置只需重建一次；下次使用时直接将远程推子区块插入控制台即可。

选择远程推子区块放置在控制台的哪一侧。 远程推子区块背面有一个编码器，可用于分配位置 ID。

如果远程推子区块位于控制台左侧，请选择位置 1。

如果远程推子区块位于控制台右侧，请选择位置 2。

位置 3 和 4 暂不使用。


配置层和推子组

转到 Layer Manager，User Layers 列表中会出现“Remote Fader Tile 1”（左）和“Remote Fader Tile 2”（右）。 配置远程推子区块的推子层和推子组，方法与控制台推子区块相同。 请参阅 推子层管理器 章节了解更多信息。


将 SOLSA 与远程推子区块配合使用

SOLSA 在离线或在线模式下也可配合最多两个远程推子区块使用。

为此，请按照上文将远程推子区块连接到控制台的说明操作。 使用 Rebuild Hardware Configuration 按钮后，关闭并重新打开 SOLSA。 无需重启计算机。

请注意： 请使用 Rebuild Hardware Configuration。 如果在没有远程推子区块实物时要配置包含远程推子区块的演出文件，无需连接远程推子区块，直接照上述步骤操作即可。

将远程推子区块与 SOLSA 配合使用时，“Remote Tile 1”（即位置编码器设为“1”）可与 Follow Detail 配合使用。

请注意： 如果更改了 Remote Tile 的区块 ID（设备背面的定位编码器）， 必须重启控制台或重新启动 SOLSA，新的 ID 才会生效。

要调整区块屏幕和 LED 的亮度，请使用 Surface Tiles 设置，位于 MENU > 设置 > Options > Console 选项卡
