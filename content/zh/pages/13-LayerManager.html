推子层管理器

SSL Live 控制台没有固定的 Layer（推子层） 或 Bank（推子组）。其布局完全灵活，可按您偏好的工作方式排列各种音频信号路径。

强烈建议先在Console Configuration页面中完成所有更改，再安排推子层和推子组，因为Console Configuration决定Layer Manager中可用的信号路径。

定义：System Layer 与 User Layer
推子层分为两种：系统层和用户层。

System Layer（系统层）按类型顺序包含控制台中存在的每一条音频信号路径。可将系统层视为构建用户层所用的路径库。系统层不可编辑，也不能直接调用到推子区块。

User Layer（用户层）允许以任意配置排列系统层中的路径；实际调用到操作面的是用户层。每条路径可在任意数量的用户层中重复出现。

定义：Layer / Bank
每个推子区块都有自己的用户层。每个区块最多可有六个推子层，且每个推子层可包含五个由 12 根推子组成的推子组。使用 Solo 按钮右侧的箭头滚动到相应推子层，再按推子右侧对应推子组的 Call 按钮，即可将推子组调用到操作面。

用户层在 MENU > Setup > Layer Manager 中排列。

推子层配置采用拖放方式，将路径从系统层或用户层中的当前位置复制到新的用户层位置。

Layer Manager 界面分为上下两部分：上半部分显示来源推子层或库 (称为来源 Source)，下半部显示正在构建的推子层或库 (称为目标 Destination)。

来源推子层可按整个推子层、整个推子组，或推子组内选定的单条路径进行复制。

设置用户层

选择目标

点击屏幕右下角（Fader Tile区域内）的 Destination 按钮，然后在弹出的子菜单中选择包含待配置目标推子组的区块。

点击 Layer 区域中的 Destination 按钮。按钮会变为青色，其 Bank 会显示在下方。

点击所需的目标 Bank 按钮。其当前布局会显示在屏幕底部。

选择来源

点击屏幕右上角的System Layers或User Layers按钮，指定上半部显示哪一种推子层。

如果来源为 System Layer，请通过Channels按钮选择路径类型。如果来源为 User Layer（用于在区块间复制），请通过出现的Fader Tile按钮选择来源区块。

复制整个 Layer 和 Bank

要复制整个推子层（包括其中所有推子组和路径），请将 Source 区域中的 Layer 图标拖到 Destination 区域中所需的 Layer 位置。

要复制推子层中的单个推子组，请点击 Source 区域中相应的 Layer 图标，其 Bank 会显示在下方。将 Source Bank 图标拖到 Destination 区域中所需的 Bank 位置。

复制单条路径

要从推子组复制一条或多条路径，请点击相应的 Source Bank 图标。其路径会显示在下方。点击要复制的路径使其高亮。

注意：所选路径不必连续，但复制后会占用连续槽位，并覆盖这些位置原有的分配。

将所选路径拖入屏幕底部的 12 个槽位之一。如果可用空间足以容纳插入的路径数量，目标区域中的槽位会亮起青色。

将拖动图标置于第一个目标槽位上方。目标位置可用时，图标轮廓会由红变绿，第一个槽位会变蓝。

删除推子层、推子组和路径

要删除整个推子层（及其中所有推子组）或单个推子组，只需将相应 Layer 或 Bank 图标拖到屏幕右侧的回收站图标上。

要删除单条路径，点击要删除的路径并拖到图标上。

注意：修改 Layer 和 Bank 后无法撤销。

重新排列目标推子组

可使用与创建时相同的拖放方式，重新排列目标区域中的路径。

在目标区域点击要移动的一条或多条路径，并拖到 Bank 内其他槽位。可用槽位会亮起青色；拖动经过时，当前选定槽位会高亮为蓝色。松开拖拽即可将路径放到新位置。

在区块之间复制

配置好一个区块后，可将其推子层和推子组复制到其他区块。在 Source 区域中选择 User Layers，再选择来源区块以查看可用的 Layer 和 Bank。

在屏幕下半部选择目标区块。

按照上述从 System Layer 复制的方法，将整个推子层（包括其中推子组）拖入目标区块的 Layer，或将单个推子组拖入目标 Bank 图标。

同样，也可按照从 System Layer 复制路径的步骤，在区块之间复制单条路径。

命名推子组和推子层并设置颜色

每个用户层和推子组都可自定义颜色和名称。其颜色和名称会显示在操作面上的 Layer 或 Bank 选择按钮内部及上方。

推子层和推子组的名称与颜色选择器位于推子层管理器中 Layer 和 Bank 图标的右侧，并作用于其左侧当前高亮的推子层和推子组。

名称

System Layer 和 Bank 根据其内容命名。User Layer 和 Bank 可按以下方式指定自定义名称：

按上述说明，在 Destination 区域选择区块和 Layer（或以及相关 Bank）。

双击所选 Layer 或 Bank 右侧相应的 Name 文本框。随后会出现用于输入名称的键盘。

如有需要，使用红色箭头按钮删除现有文字。

输入新名称，然后点击OK。

注意：直接复制自系统层/组的用户层/组，初始名称和颜色与对应系统层/组相同。

颜色

要更改推子层颜色：

按上述说明，在 Destination 区域选择区块和 Layer（或以及相关 Bank）。

点击相应的颜色选择器，然后从出现的下拉列表中选择颜色。
