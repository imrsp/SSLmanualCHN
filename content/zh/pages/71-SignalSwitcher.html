信号切换器

系统提供四组多通道 A/B 输入切换器，可在两组输入之间快速切换。可通过 Tools 菜单中的专用页面访问 （Menu > Tools > Signal Switcher）。 信号切换器的输入和输出路由通过 XY Routing 页面 完成（在 Path > Signal Switcher 中选择，并将其用于 Source 或 Destination 区域）。每个信号切换器支持 1 至 64 个音频通道。 此功能非常适合在主用与备用播放设备之间切换；由于切换器可以自由路由，也可用于许多其他应用场景。

可从 GUI 手动触发切换，也可通过 Event Manager 触发（例如由 GPI、信号路径静音、DAW Control 功能等触发），或将其分配给 User Keys。 Signal Switcher 界面中的开关图形会指示当前启用的输入组。

此外，FX Rack 中的 Signal Detector 可在参考信号低于指定电平时执行自动切换。 如果未启用 Bidirectional Auto-Switching 按钮，自动检测只会在信号低于所设阈值时切换至 B。 触发切换至 B 输入后，如有需要，用户必须手动切回输入 A。 这样可避免控制信号持续越过所设阈值时（例如连接时断时续）切换器在两个输入之间反复切换。

如果启用 Bidirectional Auto-Switching 按钮，当控制信号超过 Signal Detector 的 Threshold 点时，Signal Switcher 会切回 A 输入。 如果播放停止时控制信号也会停止，请使用此模式，以防切换器切换至输入 B 后一直停留在该输入。 启用 Bidirectional Auto-Switching 后，Signal Detector 具有瞬断检测功能，可在检测到间歇性控制信号时防止 Signal Switcher 在 A、B 输入之间来回切换。

建议从主音频源提供专用控制信号来管理切换，不要依赖节目音轨始终含有连续音频内容。

注意：Signal Switcher 不适用于需要由输入通道进行控制的 Mic 输入。控制数据不会通过切换器传递。
