母线与 VCA

本页介绍如何将信号路径发送到母线或分配给 VCA。有关不同母线类型的更多信息，请参阅 控制台配置部分。

  • Channel 可发送到 Stem、Aux 和 Master。
  • Stem 可发送到其他 Stem、Aux 和 Master。
  • Aux 可发送到 Master，但不能发送到 Stem 或其他 Aux。
  • Master 不能发送到 Stem、Aux 或其他 Master。

发送到 Stem 和 Aux 时，既有 In/Out 控制，也有电平发送量。

发送到 Master 或分配给 VCA 时，只有 In/Out 控制，没有电平发送量。

可通过以下 4 种方式调整发送：

  • 使用 Query
  • 从详细视图调整
  • 从通道视图和快捷控制调整
  • 从控制区块调整

注意： Query 是将信号路径发送到母线或分配给 VCA 的最快捷方式，而且无需使用触摸屏。
使用 Query 进行母线路由/VCA 分配

使用 Query 进行母线路由/VCA 分配

使用 SSL Live 便捷的查找功能 Query，可快速轻松地配置发送。首先使用 Query Assign 路由到母线或分配给 VCA，然后使用 Compressed（精简）或 Expanded（扩展）Query 模式调整发送电平。

Query Assign

Query Assign 可用于将信号路径快速分配给被查询的信号路径。

按住某个信号路径的 Q 按钮进入 Query Assign。然后使用其他信号路径的 SELECT 按钮，开启或关闭其至被查询信号路径的分配。

处于 Query Assign 时，被查询信号路径的 Q 按钮会呈紫色闪烁。Control Mode 将与该信号路径类型的设置一致。

再次按下被查询信号路径的 Q 按钮即可退出 Query Assign。

提示：也可以按住第一个信号路径的 SELECT 按钮， 再按下最后一个信号路径的 SELECT 按钮，以选择一段信号路径。推子区块上两者之间的所有信号路径都会被选中。范围选择也可跨多个推子区块进行。

使用 Query 调整发送

按一下某个信号路径的 Q 按钮。系统会显示该信号路径所发送到的全部母线、向该信号路径馈送信号的信号路径，以及该信号路径所分配到的所有 VCA。

请注意：根据 Query Mode 的不同，相关贡献信号路径可能作为其他区块上仅有的信号路径显示（精简模式）， 也可能通过 Q 按钮的灯光颜色显示：向被查询信号路径馈送的信号路径为绿色，从被查询信号路径接收馈送的信号路径为红色（扩展模式）。

如果 Query Control Mode 设为 Quick Controls，则可使用显示贡献信号路径的区块上的快捷控制旋钮， 提高或降低发送电平。

如果 Query Control Mode 设为 Faders，按下 Q 按钮时推子会自动翻转， 此时可使用推子提高或降低发送电平。

要进一步了解 Query 和 Query Mode，请参阅 Query 页面。

Detail View

从详细视图路由到母线/分配给 VCA

打开 Detail View，然后点按 Stems、Auxiliaries、Masters 或 MG + VCA 页面。

该类型所有母线的按钮都会列在页面左半部分。 如果母线数量超出单页可显示范围，请使用页面右边缘的数字按钮选择要显示的母线页。

页面右上角的 Lock 按钮决定触摸母线时执行的操作：

  • 如果 Lock 为开启，母线/VCA 将分配给快捷控制
  • 如果 Lock 为关闭，母线/VCA 将分配给快捷控制，同时发送会切换为 In 或 Out

选中母线后，所有可用参数的滑块会显示在屏幕右半部分。 可用手指在触摸屏上拖动这些滑块，也可使用屏幕底部相关标签下方所分配的快捷控制进行调整。 只会显示与所选母线相关的控件。例如，如果母线已设为 Follow Pan（见下文），则不会显示 Pan 控件； 如果从单声道通道路由到立体声母线且已禁用 Follow Pan，则只会显示 Left/Right Pan 控件。

按下 IN 按钮（或上方快捷控制按钮），将发送切换为 In 或 Out。

按下 0 dB 按钮，将发送电平设为 0 dB。

使用 Pan Left、Pan Centre 和 Pan Right 按钮快速设置发送的声像。

使用 Copy Main Path Pan to [Bus] Send Pan，将信号路径的声像设置复制到所选母线发送的声像部分 （仅当所选母线未设为 Follow Pan 时应用；否则使用主信号路径的 Pan）。 可使用 Path Link，将多个信号路径的 Pan 设置复制到所选母线。 另请参阅下文的 Copy Send Pans from Main Path Pans 功能，以整条母线为单位进行复制。

注意：IN 是配置发送到 Master 母线时对话框右侧唯一显示的控件。

可点按 Feed: 按钮（位于 Lock 按钮下方）， 然后从出现的菜单中选择信号源，以指定此发送的馈送点。选项如下：

  • 发送到 Master 母线：Default、Post Fader（位于放置在推子之后的任何处理单元之前）和 Post All（位于放置在推子之后的所有处理单元之后）；
  • 发送到 Aux：Default、Pre Fader 和 Post All；
  • 发送到 Stem：Default、Pre Fader、Post Trim、Post Insert A、Post Insert B、Post Fader 或 Post All
注意：当发送信号取自 Default 点时，将遵循 Console Configuration 页面中设定的默认设置。 详情请参阅 控制台配置。

从通道推子之前馈送的母线发送（Post Trim、Pre Fader，或当插入点在通道处理顺序中位于推子之前时的 Post Insert A/B）会在发送电平 左侧显示指示标记：

从通道推子之后馈送的母线发送（Post Fader、Post All，或当插入点在通道处理顺序中位于推子之后时的 Post Insert A/B）会在发送电平 右侧显示指示标记：

母线选择器的状态表示如下：

Master/VCA：

Stem 和 Aux：
母线未选中且处于 Out； 母线未选中且处于 Out；
母线已分配给 Quick Controls 且处于 Out； 母线已分配给 Quick Controls 且处于 Out；
母线已分配给 Quick Controls 且处于 In； 母线已分配给 Quick Controls 且处于 In。选择器下半部分显示其电平，上半部分在适用时显示声像信息。
母线未选中且处于 In。

静音组和 VCA 的详细视图页面外观略有不同，但功能相同；点按相应按钮即可将信号路径分配给静音组和 VCA。

Channel View 与 Quick Controls

从通道视图和快捷控制路由到母线/分配给 VCA

要将母线发送映射到快捷控制，请进入 Channel View，点按母线分配按钮 （Master、Stem 或 Aux），然后点按相应母线：

查看推子区块右上角的快捷控制标签显示，即可确认当前映射到快捷控制的母线。


该类型所有母线的选择器都会列在通道电平表下方。如果母线数量超出单页可显示范围，请使用屏幕右边缘的数字按钮（如右图所示）选择要显示的选择器页。

页面右上角的 Lock 按钮决定触摸母线时执行的操作：

  • 如果 Lock 为开启，母线将分配给快捷控制
  • 如果 Lock 为关闭，母线将分配给快捷控制，同时发送会切换为 In 或 Out

各发送类型可用的控件如下：

Master

上方按键用于将发送切换为 In 或 Out。由于 Master 没有电平发送量，因此没有其他控件。

Stem 与 Aux

  • 上方按键用于将发送切换为 In 或 Out。
  • 按下旋钮，可在声像、声场宽度、扩散度、前/后、深度和低频增益之间循环切换右下方按钮的功能。
  • 在 Gain/Pan 模式下，左、右按键分别将编码器切换到增益和声像。
注意：仅当该母线的 Follow Pan 模式关闭，并且通道和母线的格式确实需要这些控件时，Pan、Width 和 Divergence 才会启用。

快捷显示区（标记为 Assigned）列出从该通道到此类型所有母线的路由。

要选择其他母线，只需在主屏幕的 Channel View 中点按另一条母线。

也可以使用 ▲/▼ 按钮（位于 Enter 按钮上方）在该母线信号路径类型内滚动；使用前请先启用推子区块上的 Enter 按钮。 短按箭头可逐项浏览列表；按住箭头会向上或向下跳过八条母线，以便快速浏览。关闭 Enter 后，将退出信号路径浏览并返回其他分配功能。

当通道未向该类型的任何母线馈送信号时，母线按钮中的指示灯熄灭；当至少向一条此类母线馈送信号时，指示灯呈绿色（VCA 为橙色）。

Control Tile

从控制区块路由到母线

可通过控制区块控制通道发送到 Stem 和 Aux，以及通道路由到 Master。

按下控制区块屏幕上方的 Aux、Stem、Master 或 VCA MG 按钮：

屏幕中间的页面按钮和参数按钮决定分配给两侧编码器的内容。

可用按钮以蓝色标签显示；不相关的按钮以灰色标签显示；当前启用的按钮亮蓝灯。

使用右侧的页面按钮，选择当前启用的十二路发送页面；

使用左侧的 Parameter 按钮，选择分配给旋钮的参数。

可用参数取决于相应通道和母线的格式。

随后可使用屏幕两侧的旋钮，更改屏幕邻近区域所示母线的选定参数。按下相应旋钮可开启或关闭发送。

所有屏幕按钮也都可通过触摸屏进行控制。

跟随声像

可使用 Aux 或 Stem 的 Detail 对话框中 Eyeconix 页面里的 Follow Pan 按钮，决定发送到 Stem 或 Aux 的信号是否跟随通道的声像。

关闭 Follow Pan 后，该 Aux/Stem 的发送通道 Detail 对话框会在发送控件中加入声像控件。只会提供与源通道格式及 Aux/Stem 格式组合相关的声像控件。

注意：开启或关闭 Follow Pan 可能会改变从源信号路径应用到母线的声像设置。

从主信号路径声像复制发送声像

禁用 Follow Pan 后，每个信号路径发送到所选母线的声像设置将独立于该信号路径的主声像控件。 各发送独立的声像数默认为不做声像定位。不过，可以按住 Copy Send Pans from Main Path Pans 按钮， 将每个信号路径发送到所选母线的声像设置从主信号路径声像设置复制过来。 其效果相当于对发送到该母线的每一个源信号路径，分别按下 Copy Main Path Pan to [Bus] Send Pan（见上文详细视图部分）。

跟随静音

默认情况下，无论 Stem 和 Aux 发送位于推子前（因此也位于静音前）还是推子后，都会由通道的 Mute 按钮静音。若要在通道 Mute 按钮开启时保持推子前发送不静音，请在 Aux/Stem 的 Detail 对话框中取消选择 Follow Mute。

VCA 控制母线发送电平（V-Aux 和 V-Stem 控件）

V-Aux 和 V-Stem 提供了一种从 VCA 高效调整通道组混音的方法。 从 VCA 信号路径调整 V-Aux 或 V-Stem 发送，会微调该 VCA 所有成员信号路径发送到相应 Aux/Stem 的电平。

例如，一个名为“Drums”的 VCA 可能包含 10 个通道（底鼓、军鼓、踩镲等）。 这些通道通过 Aux 1 路由到监听混音，并已设置初始发送电平以获得良好平衡。 将鼓组 VCA 的 Aux 1 发送设为 -3 dB，会相应衰减全部 10 个成员通道对 Aux 1 的电平发送量，同时保持它们之间的相对电平。

通道视图和详细视图中的 Aux 与 Stem 母线指示器，会同时显示信号路径的独立电平和应用 VCA 偏移后的电平。 洋红色/深蓝色条表示源信号路径发送到母线的独立（“本地”）Aux/Stem 发送电平。

如果发送电平中的 VCA 部分为正值，本地 Aux/Stem 电平条末端会增加一段浅青色条。组合条的末端表示总电平。

如果发送电平中的 VCA 部分为负值，本地 Aux/Stem 电平条会减去一段深青色条。较浅的本地 Aux/Stem 电平条末端表示总电平。

注意： 一个信号路径可由多个 VCA 控制，例如“鼓组”和“乐队”VCA；每个 VCA 都会对各自的成员信号路径应用自己的 V-Aux/V-Stem 偏移。 每个信号路径发送所显示的浅青色/深青色条，表示来自所有控制 VCA 的 V-Aux/V-Stem 总偏移量。

可在 自动化筛选器页面中，使用 VCA 信号路径的 Stem 和 Aux 列，对 VCA 对混音的贡献进行自动化。

使用 VCA Query 调整混音

通常，查询某个混音或信号路径后，控制台操作面上的 VCA 推子会控制 VCA 的主电平。 但是，为相关母线类型启用 VCA Control 选项后（该选项位于 操作面选项 的 Menu > Setup > Options > Surface 选项卡中）， 即可使用 VCA 推子（母线设为 Query on Faders 模式时）或快捷控制旋钮（设为 Query on Quick Controls 模式时）控制母线发送。

注意： 来自 VCA 的 V-Aux 和 V-Stem 发送没有馈送点选项；它们是“虚拟”控件，只是在各信号路径所定义的馈送点上， 为分配给该 VCA 的音频信号路径发送电平提供偏移。V-Aux 和 V-Stem 始终处于 IN。

锁定 V-Aux 和 V-Stem 贡献

可以锁定 V-Aux 和 V-Stem 贡献，使其无法更改，例如需要始终保持在 0 dB 时。 此选项可在 用户选项页面中访问，并会存储在演出文件中。

注意：此选项默认启用，即使加载由旧版软件创建的演出也是如此。
