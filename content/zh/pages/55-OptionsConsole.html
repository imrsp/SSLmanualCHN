控制台选项

这些选项涉及屏幕亮度、外接屏幕选项等控制台专用设置。

控制台选项位于 MENU > Setup > Options > CONSOLE 选项卡。


鼠标与触控

屏幕上 press & hold （长按）类按钮的触发时长可在 Options 页面中的 Mouse & Touch 区域调整。

如需更改，可拖动屏幕旋钮，或双击文本字段。建议以 400 ms 作为初始值。

时间和日期（仅控制台）

控制台内部时间和日期显示在 Console Options 页面的 Time 区域。
如需设置时间或日期，请触摸当前时间或日期显示后面的 Set 按钮，然后使用弹出的键盘输入新值。

如需在主屏幕上显示时钟，请触摸时钟时间显示旁边的 Open 按钮。无论主屏幕当前打开哪个页面，该时钟窗口都会保持置顶。
如需关闭时钟显示，请触摸时钟窗口右上角的红色 X 。

外接概览屏幕显示

控制台背面的概览屏幕 VGA 输出可提供额外屏幕，用于显示概览、自动化或效果器机架页面。可在 Console Options 页面的 Overview、 Automation 或 Effects Rack 按钮分别进行设置，这些按钮位于 External Screen 区域。
选择 None 可阻止任一页面显示在外接屏幕上。请注意，如果已将某个页面设为在外接屏幕显示，随后又从主屏幕访问该页面，外接概览屏幕将恢复为 None ，因为这些页面同一时间只能在一个屏幕上查看。

外接键盘区域设置（仅控制台）

选择正确的区域设置可确保控制台正确解析所连接 USB 键盘发送的按键。只需在 External Keyboard 区域选择所需选项。

操作面锁定（仅控制台）

可锁定操作面以防止改变控制台状态。如需锁定操作面，请按住 Console Options 页面 Lock 按钮，该按钮位于 Surface Lock 区域。

屏幕将显示挂锁图标，表示操作面已锁定。硬件区块不会响应任何输入，但仍会显示当前可见通道的状态（电平表、静音/独奏状态、推子位置等）。

如需解锁操作面，请按下位于屏幕左下、右下和顶部中央的三个按键符号。必须在 5 秒内按完这三个按键区域。

控制台亮度

以下各项的亮度均可从选项页面的 Console Screens、Control Surface Tiles 和 Console Lights （屏幕上方的 LED 灯带）进行调整，相关控件位于 Brightness 区域。
控制台每个区域上方的文本字段显示当前亮度值，可使用旁边的屏幕旋钮进行调整。
每个区域中的第二个字段是该控制台区域的预设亮度。按下各区域的 Preset 按钮可调用该值。如需将当前亮度存为预设值，请按住 Preset 按钮并选择 Store 选项。
双击任意字段会弹出屏幕键盘，可用于输入指定的亮度值。

注意： 这些控件与 Control Tile 硬件上的 Brightness 控件（如已安装）功能相同。按下硬件旋钮会调用预设值；按住旋钮会存储新的预设值。
这些亮度设置也可在控制台的 Automation 系统中存储和调用。
