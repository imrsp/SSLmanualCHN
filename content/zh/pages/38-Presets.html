预设

详细信息对话框中的所有通道设置都可使用预设进行保存和调用；预设既可涵盖整个通道，也可只涵盖其中较小的部分。

要打开控制台任一区域的预设列表，请在详细信息对话框中打开相应页面，然后触摸右下角附近的 Preset 按钮。预设列表将在详细信息对话框内打开：

从处理页面（例如 EQ/Filters、Panner 等）进入预设对话框时，只能从预设文件覆盖信号路径的该处理部分。从路径的 Eyeconix/Config 页面进入的预设对话框时，则可使用一个预设文件覆盖多个处理部分。

使用 Store 部分列选择要保存到预设文件的范围（启用蓝点即可允许存储）。要覆盖现有预设，请选择范围，然后按住 Save 按钮。要保存新的预设文件，请按 Save As 按钮并输入文件名。

注意： 将处理部分保存到现有预设文件是一种叠加操作。例如，若现有文件包含 Gate 设置，而预设存储范围设为 Compressor，则保存 Compressor 设置时，之前保存的 Gate 设置仍会保留在文件中。

使用 Recall 部分列选择从所选预设文件加载的范围（启用紫点即可允许调用）。Recall 列中的青色方框表示所选文件内已存储且可调用的部分。只有文件中存在且已启用调用的部分（即青色方框和紫点均启用）才会加载。按住 Load 按钮，从预设文件加载所选部分。

提示： 在路径 Eyeconix/Config 的预设页面中，可使用 Store 和 Recall 列标题切换所有存储/调用范围部分的选择状态。 注意： 文件夹创建后，只能在下述主预设列表中重命名。 下文.

预设备注

可在预设列表右侧的 Notes 显示区输入与预设有关的详细说明。

要输入备注，请找到该预设并将其选中，使其显示青色边框。

双击备注显示区，使用弹出的键盘输入备注，然后触摸 OK.

每当该预设在预设列表中高亮显示时，这些备注都会出现在备注显示区。

再次按 Preset 即可关闭预设列表。

旧版预设文件： 请注意，V4.10 及更早版本软件保存的预设采用与 V4.11 预设不同的格式。在控制台或 SOLSA 软件中打开其所在文件夹时，旧版预设文件会自动转换为新格式。

主预设列表

也可打开主预设列表，查看和管理控制台中的全部预设。访问路径为 Setup 菜单： MENU > Setup > File Manager > 预设 选项卡

此显示界面的操作方式与演出文件界面非常相似，主要区别是这里没有 SSL Templates。文件会显示完整扩展名。为预设指定的“用户”名称位于预设文件名的第一部分；下一部分是保存该预设时所用的通道类型；第三部分是预设类型（EQ 频段、压缩器、完整通道路径等）。

编辑模式

与演出文件一样，部分文件夹操作仅在预设列表处于 Edit 模式时可用。

要启用 Edit 模式，请触摸主预设列表屏幕顶部中央的 Edit 按钮。该按钮会点亮，屏幕右侧将出现第二个导航窗格。

要选择供两个窗格之间任一编辑操作处理的预设，请在左侧导航窗格中找到它，并触摸其列表项末端的圆圈；圆圈变红表示该预设已启用编辑。

可选择多个预设和多个文件夹。也可触摸列表顶部的 All 圆圈，选择当前位置中的全部预设和文件夹。

请记住： Edit 模式功能始终作用于已启用编辑的预设，而不是仅被高亮显示（带青色边框）的预设。

移动和复制预设

要移动或复制文件和文件夹，请在左侧窗格中进入源文件夹，并在右侧导航窗格中进入目标文件夹；

在左侧导航窗格中，对要移动或复制的项目启用编辑；

然后按 Move or Copy 按钮。项目将被移动或复制到新位置。

注意： 不能将项目复制到其当前位置。

将预设设为只读

可将预设设为只读，以防止通过 Save 功能覆盖它们。

要选择需设为只读的预设，请在左侧导航窗格中找到它们并启用编辑。

然后按 Read Only 按钮。文件名前将出现挂锁符号。

要取消文件的只读状态，只需重复此过程。

重命名预设和文件夹

要重命名文件或文件夹，请在左侧导航窗格中找到它并启用编辑。

然后按 Rename 按钮，使用弹出的键盘重命名预设，然后触摸 OK.

注意： 重命名功能只允许更改文件名中的“用户”部分。

删除预设和文件夹

要删除预设或文件夹，请在左侧导航窗格中找到它们并启用编辑。

然后按住 Delete 按钮。

注意： 文件夹必须为空才能删除。受 Read Only 保护的预设无法删除。
