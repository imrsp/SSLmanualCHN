系统设置

要访问系统级功能，请进入 MENU > Setup > System / Power 页面（在 SOLSA 中标为 System ）。这些设置保存在控制台中（不保存在用户演出文件中）。大多数设置位于此页面的 System 页签中。有关许可功能，请参阅 下方的 Licensing 章节。

工作电平

双击该数值可在 +12 至 +24 dBu 之间调整模拟线路工作电平。默认设置为 +24 dBu。

此设置决定控制台中与 0 dBFS 对应的 dBu 电平（控制台的所有电平表均以 0 dBFS 为基准）。

例如，当 Operating Level 设为 24 dBu 时，20 dBu 的输入信号将在电平表上显示为 -4 dBFS。同样，输出 -6 dBFS 信号时，模拟输出端将得到 +18 dBu 的信号。

注意： MADI 舞台接口箱和本地 I/O 输出可按设备相对于工作电平进行微调。请在 I/O Setup 页面选择相应舞台接口箱或 Local I/O 卡，然后使用 Output Trim 滑块。更多信息请参阅 本地与 MADI I/O 配置 章节。
SSL Network I/O Dante 舞台接口箱的工作电平也可按设备配置。更多信息请参阅 网络 I/O 设备 章节。

采样率

系统可在 96 kHz（推荐）或 48 kHz 采样率下工作。长按相应按钮可更改控制台采样率。

重要提示： 更改控制台采样率会导致 I/O 配置发生变化，因此现有路由可能丢失。所以应在配置控制台的其他部分 之前 设置采样率。

每个舞台接口箱都必须配置为与控制台相同的采样率。更多信息请参阅 时钟设置 章节。

重要：
更改采样率后，控制台 必须 彻底断电后重新上电。

控制台名称

该 控制台名称 可在此编辑。通过远程操作面（SOLSA、TaCo 或处于远程模式的另一台控制台）连接本控制台时，将引用此名称。

该 Control ID （可在 MENU > Setup > Options > NETWORK 页签中查看）用于在取得 Net I/O 设备参数的 Ownership 时确定控制台名称。不过， Control ID 也取自 控制台名称。

双击名称字段以编辑控制台名称，然后长按 Apply。

请注意：
更改控制台名称会使控制台关机，并丢失对所有 Net I/O 设备参数的 Ownership。这表示必须手动重新取得之前拥有的输入（或其他设置），才能恢复控制。有关取得输入和舞台接口箱设置 Ownership 的说明，请参阅 控制 Net I/O 舞台接口箱与 Ownership。

硬件配置

该 Rebuild Hardware Configuration 按钮用于配置控制台，以便与 Remote Tile 配合使用。

有关设置 Remote Tile 的更多信息，请参阅 设置：Remote Tile。

操作面

控制台的这些组件可在不影响音频的情况下重新启动。仅当组件无响应或 SSL 明确要求时使用。

请注意，刷新 Front Panel 大约需要两分钟，在此期间用户无法进行控制。

推子校准

如果推子位置与显示值不一致，请长按 Calibrate Faders （位于 操作面 中）重新校准推子。该过程约需 30 秒；音频将继续通过，但推子在此期间会与音频控制断开。

数据备份

如果遇到技术问题，SSL 可能会要求提供控制台日志文件以供分析。收到此要求后，请将 FAT32 或 NTFS 格式的 USB 驱动器插入控制台，然后按下 Data Backup 按钮。日志文件和备份演出文件将保存到 USB 驱动器上 `SSL\Live\DataBackup` 子文件夹内以时间戳命名的文件夹中。

管理员访问

此功能用于 SSL 实时远程支持。SSL 要求时，请长按 Admin Access 按钮以获取响应代码。

注意： 强烈建议：在没有 SSL 明确指示的情况下，不要操作 System 页面中的任何其他按钮。

许可功能

控制台的部分功能需要许可证。详情请联系 SSL。 Licensing 页签用于管理这些许可证。

SSL 要求时，请插入 FAT32 或 NTFS 格式的 USB 存储设备，并使用 Export Console ID 按钮将控制台的唯一 ID 复制到 USB 存储设备。

将 Console ID 文件发送给 SSL。

将 SSL 提供的许可证文件复制到 FAT32 或 NTFS 格式的 USB 存储设备，然后使用 Import Licences 按钮将许可证从 USB 存储设备复制到控制台。

导入的许可证会显示在列表中。与 Console ID 不匹配的许可证不会被导入。

如果控制台的系统日期和时间不在许可证的生效日期与到期日期范围内，许可证将显示为未激活，相应功能不可用。

如果控制台硬件发生变化，Console ID 也会改变，许可证将显示为无效（即许可功能不可用）。遇到这种情况请联系 SSL Support。
