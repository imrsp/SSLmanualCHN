独奏

设置独奏系统

设置独奏系统

SSL Live 具有 两条独奏母线 A 和 B，它们可馈送至 三个独奏输出通道（1、2、3）。

按下推子区块上的 SOLO 按钮时，该路径的信号会馈送至独奏母线。

可使用主控区块 Solo 部分的编码器调节独奏母线电平。按下编码器可将独奏母线静音或取消静音。

独奏母线输出必须路由到 独奏输出通道，信号才能继续传输。主控区块上有一个小型 独奏矩阵 用于控制此路由。

该 A & B 按钮用于选择要路由的独奏母线。请选择所需独奏母线。

1、2 和 3 按钮用于选择该独奏母线应路由到哪些 独奏输出通道。请选择要由该独奏母线馈送的独奏输出通道。

也可通过主屏幕右侧的 独奏 电平按钮调节独奏母线的输出路由和电平。

在通道视图中找到独奏输出通道（默认显示为紫色，位于所有推子区块的“Misc”库中）。在有信号的路径上按 独奏 ，并确保所用独奏母线的编码器电平已调高。推高推子后，应能在电平表中看到信号（独奏通道电平表默认为 Post All）。

信号到达独奏通道后，还必须路由其输出。与其他路径一样，请在独奏通道详细视图的 Input/Routing 部分中完成。

要将独奏通道输出路由到推子区块下方的耳机插孔，请将左侧耳机插孔路由到本地模拟输出“HP1 L”和“HP1 R”，右侧耳机插孔路由到“HP2 L”和“HP2 R”。

注意： 对于 L300、L350、L450、L500、L500 Plus、L550 和 L650 控制台，本地模拟输出 A13-A16 还在内部与控制台的两个耳机插孔并联硬接线。因此，如果正在使用对应的耳机插孔，应避免同时使用控制台背面的 XLR 输出，否则会降低音频性能。
对于 L100 和 L200 控制台，耳机插孔未与任何本地模拟输出硬接线，可独立工作。

独奏通道具有与普通通道相同的处理功能，因此可按需对独奏信号进行 EQ、压缩等处理。

提示：
  • 使用独奏通道的 Delay 可使 FOH 位置的耳机/近场监听与主扩声系统实现时间对齐。延迟可用米为单位输入，便于配置。
  • 可将 FX Rack 中的信号分析器（例如 Spectra-lyser 和 Phase-Scope）插入独奏通道的 Insert 点，以便轻松监测独奏路径。

发送到独奏母线的馈送点

每条路径均可使用以下馈送点选项发送到独奏母线 A 和/或 B：

  • Bus Default：使用默认馈送点设置馈送到独奏母线
  • None：不馈送到独奏母线
  • Pre Fader：在推子前馈送到独奏母线
  • Post All：在信号链最末端馈送到独奏母线（不仅在推子后，也在路径所有其他处理单元之后）

此项在详细视图的 Fader 页面中设置。

该 Bus Default 馈送点在以下位置设置： Menu > Setup > Options > User 选项卡。

提示：
  • 可使用 Default Feed Point 选项在推子前监听和推子后（Post All）监听之间切换。
  • 每条独奏母线的默认馈送点均可通过 User Key切换。

新建路径时，两条独奏母线馈送点均设为 Bus Default。

注意： 推子在通道处理顺序中的位置并不固定，可完全由用户定义。 Pre Fader 馈送点是指信号链中紧邻推子之前的位置，无论推子位于何处。路径处理顺序在详细视图的 Config 页面中设置（详见 处理顺序 ）。

独奏发送的声像

该 Solo Follow Pan 选项控制馈送至独奏母线的信号是否跟随路径声像。

注意：
SSL Live 控制台路径的声像可针对每个发送单独配置。因此，声像器在信号链中不属于推子的一部分，发送至母线的 Post Fader 也不等同于 Post Pan。

要调整 Follow Pan 设置，请前往 MENU > Setup > Options > USER 选项卡。

在左侧 SOLO SETUP 部分有两个按钮： Follow Pan: Pre Fader Sends 和 Follow Pan: Post Fader Sends。

Follow Pan: Pre Fader Sends 影响所有发送到独奏母线的 Pre Fader 发送。启用后，发送至独奏母线时会使用路径的主声像。

停用 Follow Pan: Pre Fader Sends 后，发送至独奏母线的 Pre Fader 发送会忽略路径主声像。信号将使用默认 Folddown 设置发送到独奏母线。可在以下位置调整： MENU > Setup > Options > SYSTEM 选项卡。

例如，通道 1 为单声道，以 Pre Fader 方式馈送到独奏母线 A。独奏母线 A 又发送到立体声独奏通道 1。通道 1 的声像置于最左。启用 Follow Pan: Pre Fader Sends 后，信号会发送到独奏通道 1 的最左侧。停用 Follow Pan: Pre Fader Sends 后，信号会发送到独奏通道 1 的中央。

Follow Pan: Post Fader Sends 的工作方式与对应的 Pre Fader 选项相同，但作用于 Post Fader 发送。

独奏模式

独奏模式

SSL Live 控制台包含以下独奏模式：

  • 互相取消（默认）
  • 瞬时
  • 独奏母线 A 叠加
  • 独奏母线 B 叠加
  • 原位独奏
  • 输入独奏优先
  • 广播独奏模式

要更改独奏模式，请前往 MENU > Setup > Options > User 选项卡。在 Solo Setup 区域中触摸相应的 Solo Mode 按钮。各模式详述如下。

叠加/互相取消独奏模式

默认情况下， SOLO 按钮采用互相取消方式（也称自动取消或单独独奏模式）；按下一个按钮会取消所有其他 SOLO 按钮。

触摸 Solo Bus A 或 Solo Bus B 按钮（位于 Additive Solo Mode 选项卡的 User 部分）可为相应独奏母线开启叠加独奏模式。

在叠加独奏模式下，按 SOLO 按钮会锁定其开关状态，因此可同时启用多个 SOLO 按钮。

注意：
即使在互相取消独奏模式下，也可同时独奏两个或更多通道；只需同时按下它们的 SOLO 按钮。

互相取消独奏模式会覆盖叠加独奏模式。因此，如果通道设置为同时发送到两条独奏母线，但只有一条母线设为叠加独奏模式，则独奏操作仍会采用互相取消方式。
提示： 可使用 User Keys切换。

临时瞬时模式

要临时使用 Momentary Solo Mode，请按住推子区块上的任一 SOLO 按钮，信号即会发送到独奏母线。

松开 SOLO 按钮后，该按钮会停用，信号也会停止发送到独奏母线。


原位独奏

独奏系统还可在 Solo In Place （SIP）模式下工作。在此模式下，通道 SOLO 按钮不再将该通道发送至独奏母线（标准操作），而是通过将主混音输出中的其他所有通道静音，原位监听该通道。

在 Solo In Place 中，SOLO 按钮采用 按住 方式。

这是一项安全措施，因为 Solo In Place 属于 破坏性 操作。

Solo In Place 始终采用 叠加 模式。

处于 Solo In Place 模式时，状态栏中会出现闪烁的 图标。

提示： 可通过 User Key切换。

SIP 隔离

各路径具有 SIP Isolate 按钮，可防止其他通道在 SIP 模式下独奏时该路径被静音。它适用于隔离要与当前独奏通道一同监听的效果通道或子组。

要开启 SIP Isolate，请打开该路径的详细视图并进入 Fader 部分，然后启用 SIP Isolate。

也可在通道视图中选择 Fader 分配按钮，从快速控制访问 SIP Isolate。


输入独奏优先

Input Solo Priority 允许通道独奏暂时覆盖 Stem 和 Aux 的独奏。

用户先独奏 Stem 或 Aux，再独奏一个或多个通道时，母线独奏会被临时覆盖。清除通道独奏后，该母线会再次独奏。

要开启输入独奏优先，请前往 MENU > Setup > Options > USER 选项卡。 在 SOLO SETUP 部分中触摸 Input Solo Priority 以启用。

现在按 Stem 或 Aux 上的 SOLO 按钮。Stem/Aux 信号将发送到独奏母线。母线的 SOLO 按钮会变为黄色以指示此状态。

现在按 SOLO。此时只有通道信号会发送到独奏母线。通道的 SOLO 按钮会变黄。Stem/Aux 的 SOLO 按钮会变为橙色，表示它是最近独奏的母线。

再次按通道的 SOLO 按钮取消该通道的独奏，Stem/Aux 随即再次独奏。Stem/Aux 信号会再次发送到 SOLO 母线，Stem/Aux 的 SOLO 按钮会变回黄色。

再次按 Stem/Aux 的 SOLO 按钮可清除独奏。

主控区块上的独奏 CLEAR 按钮可用于清除 Input Solo Priority 模式中的独奏。当 Input Solo Priority 启用时，CLEAR 按钮会清除最近一次独奏。因此，如果先独奏母线再独奏通道，则需要按两次 CLEAR 按钮才能清除两次独奏。

注意： 要同时使用 Input Solo Priority 和 Solo In Place，请先使用推子区块上的 SOLO 按钮独奏母线，然后使用通道视图或详细视图 Fader 部分主屏幕上的 SOLO 按钮独奏通道。

要关闭 Input Solo Priority，请返回 MENU > Setup > Options > USER 选项卡，再次触摸 Input Solo Priority 以停用。


广播独奏模式

启用后，独奏通道的功能会重新配置，使输出可根据独奏通道格式用作传统监听输出，馈送单声道、立体声或 5.1 扬声器系统。

独奏通道输出电平直接由独奏通道推子控制，该推子成为主监听电平控件。主控区块上的 Solo A 和 Solo B 控件在独奏母线已路由到独奏通道且路径独奏有效时，仍会微调各自独奏母线的电平。但是，Solo A 和 B 控件不再影响分配给独奏通道的 Default/External 监听源电平（见下文），要监听 Default 或 External 输入也不再必须将独奏母线路由到独奏通道。

新增了独奏通道 DIM 功能，由独奏通道的 SOLO 按钮控制（主控区块主推子上标为 Solo/Dim ）。可在独奏通道 Fader 部分，通过触摸屏或控制区块的 Fader TB 按钮，将 Dim 电平设为 0 至 -40 dB。

也可使用控制台 GPI 端口远程控制 Dim。GPI 1、3、5 分别控制独奏通道 1、2、3 的 Dim；GPI 2、4、6 则分别切换独奏通道 1、2、3 的 Mute。

如果使用单独的独奏扬声器，可使用 Auto Dim 功能在路径独奏有效时衰减主监听馈送。该功能也可在独奏通道 Fader 部分启用。

默认配置为：所有路径将 Post All 处理后的信号路由到独奏母线 A，将 Pre Fader 信号路由到独奏母线 B。默认情况下，独奏母线 A 和 B 分别馈送独奏输出 1 和 2。例如，可将独奏通道 1 路由到一组立体声监听，将独奏通道 2 路由到专用“PFL”扬声器。独奏通道 1 的 Default Input 可设为主混音输出；独奏某条路径时会覆盖该输入。

在独奏通道 1 的 Fader 详细信息页面启用 Auto Dim 按钮。每当路径独奏到独奏通道 2 的“PFL”输出（由独奏母线 B 馈送）时，这会自动衰减独奏通道 1 到主监听输出的电平。


独奏输出通道的默认/外部输入

默认独奏源

三个独奏输出通道都可配置为接收默认输入源。该信号源可以是控制台内的任意信号，例如主混音输出、通信系统馈送或某一路监听混音。每个独奏输出通道的默认独奏源相互独立。

进入独奏通道的详细视图，然后进入 Input/Routing。触摸右侧的 Route Default Input ，路由视图随即打开。

路由 Default Input 后，在 Input/Routing 页面的 Solo Source Select 区域中选择 Default Input。

分配给独奏通道的独奏母线将控制该通道中默认独奏源的电平，因此可通过主控区块上的独奏母线控件和主屏幕 Solo 弹出面板控制电平与静音。从独奏通道取消分配两条独奏母线会将外部输入静音。

注意： 如果两条独奏母线都分配给同一独奏通道，则两条母线将 并行 控制该通道默认独奏源的电平。

外部输入

与上述默认独奏源类似，还可向独奏通道注入另外六个输入，用于监听混音、节目输出等。

可使用独奏通道 Input/Routing 详细信息页面右侧的 Route External Input 按钮路由各外部输入。

在同一页面左侧的 Solo Source Select 部分选择 Ext Input，然后使用上方的 Ext 1 - Ext 6 按钮选择要监听的外部输入。如果该输入已路由到信号源，LED 条会亮绿色。

在同一页面左侧的 Sum Both 可同时监听外部输入和默认信号源。

控制区块小屏幕中，独奏通道的 Input 部分也提供外部信号源选择器。

分配给独奏通道的独奏母线将控制该通道中外部输入的电平，因此可通过主控区块上的独奏母线控件和主屏幕 Solo 弹出面板控制电平与静音。从独奏通道取消分配两条独奏母线会将外部输入静音。

默认/外部输入启用状态

Default/External 输入可配置为始终启用，或仅在没有其他信号源独奏时启用。使用独奏通道 Input/Routing 页面中的 Ext/Default Input Always Active 按钮，决定当其他信号源通过路径/效果 SOLO 按钮路由到独奏通道时，该信号源是保持启用（按钮按下）还是静音。

Default/External 信号源的典型示例包括：FOH 混音时的主 Master 母线（此时应关闭 Ext/Default Input Always Active 选项，以便单独独奏各路径）；或监听混音时向一组 IEM 注入技术/喊话通信系统（此时应启用Ext/Default Input Always Active ，使通信信号源与其他独奏信号相加）。

其他独奏选项

Effects Rack 独奏配置

Effects Rack 中的部分模块也具有独奏功能。这些独奏信号所用的独奏母线按机架分别定义，请使用位于 Effects 界面右侧的 Misc Solo: 按钮。

触摸 Misc Solo: 按钮，打开母线选项子菜单；

然后选择 Solo A&B、Solo A、Solo B 或 None。
