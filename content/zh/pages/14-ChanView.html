通道视图

通道视图为当前选中的区块库提供可视反馈。 注意：按下 Screen（位于推子区块上）也会在主屏幕中调出通道视图。

屏幕会显示所有类型的信号路径。

主屏幕右侧的 Layers 按钮也可用于将信号路径调到屏幕上，即使这些路径当前并不位于区块上（有关 Layers 与 Banks 的详细信息，请参阅上一节）。

触摸屏幕上的信号路径即可将其选中 – 所选路径会以青色边框突出显示。同时也会启用该区块的 Select 按钮。

通道条的各个组成部分如右侧及下方所示。

双击通道条中部的快速显示“缩略图”，或通道条底部带颜色的 Eyeconix 区域，即可打开 详细视图页面并进入相应视图。

也可以双击通道条顶部的 Input 区域、Routing 分配按钮或 Processing 分配按钮，以打开 详细视图并进入相应部分，and将该功能分配给 快速控制.

通道电平表

通道电平表显示在屏幕通道条顶部，以及实体推子的右侧。

默认情况下，输入通道采用微调后计量，输出通道采用全部处理后计量（包括推子）。

要查看屏幕电平表的展开版本，请双击电平表。电平表会展开并覆盖 Selector 区域。

注意：小型电平表是展开电平表的紧凑版本；两种显示方式的电平范围相同。

动态处理的增益衰减量显示在推子区块上。

可在以下位置调整电平表设置： 系统选项

展开

Spill 功能可将立体声、LCR、4.0 和 5.1 信号路径拆分为独立声道，每个声道都会“展开”到自己的通道条中。这样便可单独配置各分量信号的某些部分。

要展开通道，请在通道视图中将其选中（使其出现青色边框），然后触摸主屏幕右侧的 Spill 按钮。各分量通道会展开至“父”通道右侧的通道条，并覆盖原先位于这些通道条中的通道；青色边框会包围父通道和所有分量通道。分量通道保留父通道名称，并在名称后附加 L、C、R、LF、Ls 或 Rs。

注意：如果展开的通道无法全部容纳在“父”通道右侧的通道条中，其余通道会延续到屏幕左侧的通道条。

要将分量通道重新“收拢”回父通道，请再次触摸 Spill。此时即可重新控制先前被分量通道覆盖的通道。

展开后，可按以下方式分别调整各项内容：

推子

分量通道相当于 VCA 的成员，而“父”通道相当于 VCA 主控。

换言之，分量通道的输出电平同时受父通道推子及其自身推子的影响（通道未展开时，其自身推子会隐藏）。

展开通道时，分量通道推子默认位于 0 dB（单位增益）；移动推子会产生增益偏移，并在分量通道“收拢”回父通道后继续保留。

静音

此时可使用分量通道的 Mute 按钮。如果启用父通道的 Mute 按钮，分量通道的 Mute 按钮会变为橙色，表示它们作为 VCA 成员工作，而父通道充当 VCA 主控。

详情请参阅 静音与静音组。

独奏

此时可使用分量通道的 Solo 按钮，以便监听单个分量。启用或停用“父”通道的 Solo 功能时，系统会保留分量通道的 Solo 设置（使用 Additive Solo 模式时）。

信号处理

大多数分量通道处理设置均可更改。

每个参数都会相对于“父”通道设置形成偏移。换言之，如果更改某个分量的 EQ 频段频率，之后在“父”通道 EQ 显示中调整该频段频率时，这一差值仍会保持不变。

注意：
请注意，频率偏移按对数方式计算！
某些设置（例如模拟输入设置、EQ 形状、Compressor/Gate 的 Ext Key 与 Peak Mode、Pan 和 All Pass Order）不能在单独的分量通道上更改。

发送

可分别调整发送至 Stem 和 Aux 母线的各分量增益；重新“收拢”后，各分量之间的差值会作为偏移保留。但不能在单个分量上独立启用或停用发送。必须先提高“父”通道的发送电平，才能调整其分量通道电平。

注意：父通道参数更改时，可在分量通道间形成偏移的参数（EQ、滤波器、母线发送等）会在所有分量通道上同步变化。不过，在调整父通道的同一控制项时，如果触摸某个分量通道的编码器或推子，该分量参数会保持当前值；即父通道所作的更改会被该分量通道忽略。

用户按键/DAW 控制弹出面板

通道视图侧边栏包含一个弹出面板，可显示 DAW 控制 功能，或用户自定义的 用户按键 功能。

按住侧边栏按钮（根据上次选择的选项，其标签为 DAW or User Keys），即可选择弹出面板中显示的内容。
